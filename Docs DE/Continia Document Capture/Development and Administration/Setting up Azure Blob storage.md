# Azure Blob Storage einrichten

{{include "setting-up-azure-blob-storage" "Document Capture"}}

> [!NOTE]
> Document Capture unterstützt Azure Blob Storage nur für neuere Versionen von Microsoft Dynamics NAV/Business Central (NAV 2013 oder höher).
>
> Damit Azure Blob Storage funktioniert, muss die Dynamics NAV/Business Central-Dienstebene für die Verwendung des TLS 1.2-Sicherheitsprotokolls konfiguriert sein.
