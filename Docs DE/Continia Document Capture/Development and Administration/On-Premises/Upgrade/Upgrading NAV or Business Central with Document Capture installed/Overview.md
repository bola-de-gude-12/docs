# Upgrade von NAV/Business Central mit einer Installation von Document Capture

Wenn Sie Continia Document Capture in einer alten Version von Microsoft Dynamics NAV/Business Central verwenden und NAV/Business Central auf eine neuere Version aktualisieren möchten, müssen Sie je nach verwendeter Version auch Document Capture aktualisieren und/oder zugehörige Aufgaben ausführen.

> [!NOTE]
> Die letzte FOB-Version von Document Capture ist 2024 R2 (25.00). On-Premises-Benutzer können die App-Version von Document Capture 2025 R1 (26.00) installieren.

Die folgenden Artikel enthalten Richtlinien zu den Schritten, die in solchen Szenarien erforderlich sind. Um auf die für Sie relevanten Richtlinien zuzugreifen, wählen Sie die Version von Business Central aus, auf die Sie aktualisieren möchten:

- [Business Central 2025 Release Wave 1 (v26)](@DC-457)
- [Business Central 2024 Release Wave 2 (v25)](@DC-235)
- [Business Central 2024 Release Wave 1 (v24)](@DC-203)

Um vorherige Richtlinien zu sehen, navigieren Sie über das Inhaltsverzeichnis auf der linken Seite direkt zum entsprechenden Abschnitt. Wenn Sie ein Mobilgerät verwenden, tippen Sie oben links auf der Seite auf die Schaltfläche **Inhalt**.

> [!NOTE]
> Wenn die Version, auf die Sie aktualisieren möchten, in den Anleitungen oben nicht genannt ist, verwenden Sie die in den Anleitungen bereitgestellten Informationen (einschließlich aller Links), um zu den für Sie relevanten Informationen zu gelangen. Informationen zum Upgrade auf Microsoft Dynamics NAV 2018 finden Sie beispielsweise unter [Business Central April 2019 (v14, FOB-basiert)](Business Central April 2019, v14.md). Obwohl dieses spezielle Upgrade in dieser Anleitung nicht direkt beschrieben wird, enthält die Anleitung alle Informationen, die Sie zur Durchführung des Upgrades benötigen.
