# Upgrade auf Business Central April 2019 (v14, FOB-basiert)

Die folgende Tabelle bietet einen Überblick darüber, welche Schritte erforderlich sind, wenn Sie Continia Document Capture (DC) installiert haben und Ihre alte Version von Microsoft Dynamics NAV/Business Central (BC) auf Business Central April 2019 (v14, FOB-basiert) aktualisieren möchten. Wenn Sie die unter **Aufgaben** unten angegebenen Richtlinien befolgen, bleibt Document Capture voll funktionsfähig und mit Ihrer aktualisierten Version von Business Central kompatibel.

<br>

| Szenario                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    | Aufgaben                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Vollständiges Upgrade von einer der folgenden Versionen: <ul><li>Microsoft Navision NAV 3.70</li><li>Microsoft Dynamics NAV 4.0</li><li>Microsoft Dynamics NAV 4.0 SP1</li><li>Microsoft Dynamics NAV 4.0 SP2</li><li>Microsoft Dynamics NAV 4.0 SP3</li><li>Microsoft Dynamics NAV 5.0</li><li>Microsoft Dynamics NAV 5.0 SP1</li><li>Microsoft Dynamics NAV 2009</li><li>Microsoft Dynamics NAV 2009 SP1</li><li>Microsoft Dynamics NAV 2009 R2</li></ul> | Um ein Upgrade von diesen Versionen durchzuführen, müssen Sie wie folgt vorgehen: <ol><li>Aktualisieren Sie DC auf die neueste Version. Auf <a href="/continia-document-capture/development-and-administration/on-premises/upgrade/supported-upgrade-paths">dieser Seite</a> finden Sie Informationen, wie Sie von Ihrer jeweiligen DC-Version aktualisieren.</li><li>Führen Sie ein Upgrade auf NAV 2013 oder NAV 2015 mit <a href="https://docs.microsoft.com/en-us/dynamics365/business-central/dev-itpro/upgrade/upgrading-to-business-central-on-premises">dieser Microsoft-Anleitung</a> durch (siehe dritte Reihe in der Tabelle). Wir empfehlen ein Upgrade auf NAV 2015.</li><li>Aktualisieren Sie von NAV 2013 oder NAV 2015 auf BC v14 (FOB-basiert) mit der Vorgehensweise, die weiter unten in der Tabelle unter „Microsoft Dynamics NAV 2013, Microsoft Dynamics NAV 2013 R2, Microsoft Dynamics NAV 2015“ beschrieben ist.</li></ol>                                                                                                                                                                                                                                                                                             |
| Vollständiges Upgrade von einer der folgenden Versionen: <ul><li>Microsoft Dynamics NAV 2013</li><li>Microsoft Dynamics NAV 2013 R2</li><li>Microsoft Dynamics NAV 2015</li></ul>                                                                                                                                                                                                                                                                                                                                                                                           | Um ein Upgrade von diesen Versionen durchzuführen, müssen Sie wie folgt vorgehen: <ol><li>Aktualisieren Sie DC auf die neueste Version (es sei denn, Sie haben dies bereits von einer früheren BC-Version getan). Auf <a href="/continia-document-capture/development-and-administration/on-premises/upgrade/supported-upgrade-paths">dieser Seite</a> finden Sie Informationen dazu, wie Sie ein Upgrade von Ihrer jeweiligen Version von DC durchführen.</li><li>Führen Sie die in <a href="https://continia.zendesk.com/hc/en-us/articles/360000872980-Upgrading-NAV-2013-2013-R2-or-2015-to-NAV-2016-and-above-with-Document-Capture-5-00">dieser Anleitung</a> beschriebenen Aufgaben aus, um Ihre Version von NAV ohne Workflows (2013, 2013 R2 oder 2015) auf eine Version mit Workflows (NAV 2016 und später) zu aktualisieren. Diese Aufgaben müssen nach dem technischen Upgrade von NAV, aber vor der Datenkonvertierung ausgeführt werden.</li><li>Führen Sie ein Upgrade auf BC v14 (FOB-basiert) mithilfe <a href="https://docs.microsoft.com/en-us/dynamics365/business-central/dev-itpro/upgrade/upgrading-to-business-central-on-premises">dieser Microsoft-Anleitung</a> aus.</li></ol> |

Auf <a href="/continia-document-capture/development-and-administration/on-premises/upgrade/supported-upgrade-paths">dieser Seite</a> finden Sie Informationen, wie Sie von Ihrer jeweiligen DC-Version aktualisieren.</li><li>Führen Sie ein Upgrade auf BC v14 (FOB-basiert) mithilfe <a href="https://docs.microsoft.com/en-us/dynamics365/business-central/dev-itpro/upgrade/upgrading-to-business-central-on-premises">dieser Microsoft-Anleitung</a> aus.</li></ol>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
