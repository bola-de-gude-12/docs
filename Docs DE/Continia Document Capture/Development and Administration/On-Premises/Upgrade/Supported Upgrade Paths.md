# Unterstützte Upgradepfade für Continia Document Capture (FOB)

In diesem Artikel wird beschrieben, wie Sie Continia Document Capture für FOB-Versionen von Microsoft Dynamics 365 Business Central aktualisieren. Informationen zum Upgrade von Document Capture für App-Versionen von Business Central finden Sie unter [Upgrade von Continia Document Capture für App-Versionen von Business Central](@DC-6).

> [!TIP]
> Sie können die Document Capture-Installationspakete und Upgrade-Dateien für FOB-Versionen in der [Continia PartnerZone](https://partnerzone.continia.com/downloads/) herunterladen (nur für Partner verfügbar).

Document Capture (DC) ist in vielen verschiedenen Versionen verfügbar. Wenn Sie eine ältere Version haben, können Sie auf die neueste Version aktualisieren. Beachten Sie jedoch, dass ein direktes Upgrade unter Umständen nicht möglich ist und Sie eventuell auch Ihre Version von Microsoft Dynamics NAV/Business Central aktualisieren müssen.

Ein Upgrade wird in den folgenden Fällen empfohlen:

- Sie verwenden eine Version von Document Capture, die nicht mehr unterstützt wird, und es tritt ein Fehler auf, der in dieser Version nicht behoben wurde.
- Sie möchten Funktionen nutzen, die nur in neueren Versionen als der von Ihnen verwendeten Version verfügbar sind.

In allen anderen Szenarien wird unbedingt empfohlen, Service Packs zu installieren. Außerdem sollten Sie immer das neueste Service Pack installieren, um sicherzustellen, dass Ihre Version auf dem neuesten Stand ist und alle Verbesserungen und Bug Fixes implementiert sind.

> [!NOTE]
> Es gibt einen möglichen Add-In-Konflikt, wenn Sie Document Capture und Continia Document Output (und damit auch Continia Expense Management) verwenden und nur eins dieser Produkte aktualisieren. Weitere Informationen finden Sie unter [Upgrade notes concerning a possible add-in conflict](https://continia.zendesk.com/hc/da/articles/4414689784210) (nur für Continia-Partner verfügbar).

Wählen Sie in der folgenden Liste die Version von Document Capture aus, auf die Sie aktualisieren, um weitere Details und eine ausführliche Upgrade-Anleitung zu erhalten:

- [Upgrade auf Continia Document Capture 2024 R2 (25.00) für FOB-Versionen](@DC-231)

  > [!IMPORTANT]
  > Die letzte FOB-Version von Document Capture ist 2024 R2 (25.00). On-Premises-Benutzer können die App-Version von Document Capture 2025 R1 (26.00) installieren.

- [Upgrade auf Continia Document Capture 2024 R1 (24.00) für FOB-Versionen](@DC-190)

- [Upgrade auf Continia Document Capture 2023 R2 (12.00) für FOB-Versionen](@DC-157)

- [Upgrade auf Continia Document Capture 2023 R1 (11.00) für FOB-Versionen](@DC-122)

- [Upgrade auf Continia Document Capture 2022 R2 (10.00) für FOB-Versionen](@DC-92)

- [Upgrade auf Continia Document Capture 2022 R1 (9.00) für FOB-Versionen](@DC-43)

- [Upgrade auf Continia Document Capture 2021 R2 (8.00) für FOB-Versionen](@DC-10)

- [Upgrade auf Continia Document Capture 2021 R1 (7.00) für FOB-Versionen](@DC-9)

> [!NOTE]
> Wenn Sie mehrere Continia-Lösungen in einer FOB-Version von Business Central installiert haben und eine oder mehrere dieser Lösungen aktualisieren möchten, müssen Sie beim Importieren von Objekten bestimmte Schritte ausführen. Weitere Informationen finden Sie unter [Upgrade mit mehreren installierten Continia-Lösungen (FOB)](@DC-54).
