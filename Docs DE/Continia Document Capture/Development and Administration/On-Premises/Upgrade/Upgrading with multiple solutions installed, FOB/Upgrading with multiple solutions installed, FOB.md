# Upgrade mit mehreren installierten Continia-Lösungen (FOB)

Wenn Sie mehrere Continia-Lösungen installiert haben und eine oder mehrere davon aktualisieren möchten, müssen Sie ein paar wichtige Dinge in Bezug auf Continia Core und das Continia Delivery Network beachten. Diese wichtigen Details werden in den folgenden Artikeln beschrieben:

- [Upgrade von Continia Core (FOB)](Upgrading Continia Core, FOB.md)
- [Upgrade des Continia Delivery Networks (FOB)](Upgrading the Continia Delivery Network, FOB.md)

Continia Core ist für die ordnungsgemäße Funktion aller Continia-Lösungen unerlässlich, während das Continia Delivery Network derzeit nur für Continia Document Capture und Continia Document Output relevant ist.
