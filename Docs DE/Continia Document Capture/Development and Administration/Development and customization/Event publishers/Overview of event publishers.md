# Überblick über Event Publisher

Sie können die Funktionalität von Continia Document Capture mithilfe der vielen verschiedenen Event Publisher (Ereignisherausgeber) anpassen, die Continia bereitstellt.

> [!IMPORTANT]
> Ab dem Release von Document Capture 2025 R1 (26.00) wird die Liste der Event Publisher nicht mehr aktualisiert. Es ist jedoch möglich, den in Microsoft Dynamics 365 Business Central integrierten Event Recorder (Ereignis-Aufzeichnung) zu verwenden, um eine Liste der Ereignisse zu erhalten, die als Ergebnis Ihrer Aktionen auf der Plattform ausgelöst wurden. Weitere Informationen finden Sie unter [Discoverability of events](https://learn.microsoft.com/en-us/dynamics365/business-central/dev-itpro/developer/devenv-events-discoverability) (Microsoft-Artikel).

Unten finden Sie eine Liste der Document Capture-Versionen, für die eine Liste von Event Publishern vorhanden ist:

- [Event Publisher für Document Capture 2024 R2 (25.00)](@DC-233)
- [Event Publisher für Document Capture 2024 R1 (24.00)](@DC-189)
- [Event Publisher für Document Capture 2023 R2 (12.00)](@DC-152)
- [Event Publisher für Document Capture 2023 R1 (11.00)](@DC-130)
- [Event Publisher für Document Capture 2022 R2 (10.00)](@DC-103)
- [Event Publisher für Document Capture 2022 R1 (9.00)](@DC-47)
- [Event Publisher für Document Capture 2021 R2 (8.00)](@DC-28)
