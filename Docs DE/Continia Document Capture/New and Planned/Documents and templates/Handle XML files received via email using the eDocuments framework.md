# Verarbeiten von XML-Dateien, die über E-Mail empfangen werden, mit dem eDocuments-Framework

| Funktion                                                                                    | Öffentliche Vorschau | Allgemeine Verfügbarkeit Online |
| ------------------------------------------------------------------------------------------- | :------------------: | :-----------------------------: |
| Verarbeiten von XML-Dateien, die über E-Mail empfangen werden, mit dem eDocuments-Framework |           -          |    Okt. 2025    |

> [!NOTE]
> Aufgrund der Komplexität, die mit der ordnungsgemäßen Implementierung dieser Funktion verbunden ist, wurde die Veröffentlichung auf Oktober 2025 verschoben.

## Vorteile

Dieses Update vereinheitlicht, wie XML-Dateien in Document Capture verarbeitet werden. Unabhängig von der Herkunft der XML-Datei (d. h. E-Mail oder Continia Delivery Network) wird das XML über die eDocuments-Funktion verarbeitet.

## Details zur Funktion

Derzeit werden per E-Mail empfangene XML-Dateien mithilfe der grundlegenden XML-Importfunktion verarbeitet. Mit diesem Update können Sie XML-Dateien mit der neuen eDocuments-Funktion importieren und verarbeiten und dabei von der erweiterten Funktionalität profitieren.
