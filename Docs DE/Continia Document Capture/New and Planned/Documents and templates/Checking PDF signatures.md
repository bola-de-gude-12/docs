# Überprüfung von PDF-Signaturen

| Funktion                       | Öffentliche Vorschau | Allgemeine Verfügbarkeit Online |
| ------------------------------ | :------------------: | :-----------------------------: |
| Überprüfung von PDF-Signaturen |           -          |               N/A               |

> [!NOTE]
> Da es bei Continia bislang keine rechtlichen oder geschäftlichen Anforderungen hinsichtlich der Überprüfung von Signaturen auf eingehenden PDF-Dateien gibt, wurde diese Funktion auf eine spätere Version von Document Capture verschoben.

## Vorteile

Indem Sie in Continia Document Capture die Überprüfung der Signaturen eingehender PDF-Dokumente wie Rechnungen und Gutschriften aktivieren, trägt diese Funktion zur Vermeidung von Betrug bei. Darüber hinaus können Sie damit auch die gesetzlichen Anforderungen in Frankreich erfüllen, wo Unternehmen gesetzlich dazu verpflichtet sind, PDF-Rechnungen und -Gutschriften zu signieren. Da mit der Einführung derartiger Anforderungen in naher Zukunft auch in anderen Ländern zu rechnen ist, wird die Funktion so implementiert, dass sie universell anwendbar ist und nicht nur auf Frankreich beschränkt bleibt.

## Details zur Funktion

Es wird eine neue Option für Kreditorenvorlagen eingeführt, mit der Sie Document Capture so einrichten können, dass überprüft wird, ob eingehende PDF-Dokumente korrekt signiert wurden. Die Einrichtung umfasst außerdem eine Option, mit der Sie die Signatur von PDF-Dateien obligatorisch machen können.
