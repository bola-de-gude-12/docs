# Unterstützung für Verkaufslieferungen

| Funktion                              | Öffentliche Vorschau | Allgemeine Verfügbarkeit Online |
| ------------------------------------- | :------------------: | :-----------------------------: |
| Unterstützung für Verkaufslieferungen |           -          |    Okt. 2025    |

> [!NOTE]
> Diese Funktion wird in Document Capture 2025 R2 veröffentlicht.

## Vorteile

Mit dieser Funktion können Verkäufer ihre Käufer elektronisch darüber informieren, dass ihre bestellten Artikel versendet wurden. Für Käufer bedeutet dies, dass ihnen alle versandbezogenen Daten in elektronischer Form zur Verfügung stehen und nicht als angehängte PDF-Dateien, die per OCR verarbeitet werden müssten, damit sie auf die Daten zugreifen können.

## Details zur Funktion

Wenn Sie eine Bestellung buchen, werden zwei Dokumente generiert: eine gebuchte Rechnung und eine gebuchte Verkaufslieferung. Mit dieser Funktion wird der Import und Export solcher Verkaufslieferungen in Continia eDokumente vollständig unterstützt. Wenn Sie Verkäufer sind, können Sie hierdurch ganz einfach mit Ihren Käufern über den Versand bestellter Artikel kommunizieren. Und für Sie als Käufer bedeutet es, dass Sie problemlos elektronisch auf die Daten der Verkaufslieferungen zugreifen können, und zwar direkt im System, sodass deren Richtigkeit sehr wahrscheinlich ist.
