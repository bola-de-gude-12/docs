# Eine neue Seite, auf der Benutzer mehr Informationen über eDocuments und Formate finden

| Funktion                                                                                | Öffentliche Vorschau | Allgemeine Verfügbarkeit Online |
| --------------------------------------------------------------------------------------- | :------------------: | :-----------------------------: |
| Eine neue Seite, auf der Benutzer mehr Informationen über eDocuments und Formate finden |           -          |    Okt. 2025    |

> [!NOTE]
> Diese Funktion wird in Document Capture 2025 R2 veröffentlicht.

## Vorteile

Diese Funktion stellt mehr Daten zu den Feldern bereit, die auf verschiedenen Continia eDocuments-Seiten verfügbar sind. So können Sie das elektronische Format, mit dem Sie arbeiten, besser verstehen. Dies ermöglicht Ihnen auch, besser mit Validierungsfehlern umzugehen.

## Details zur Funktion

Es wird eine neue Seite eingeführt, die Benutzer über die Felder der verschiedenen unterstützten elektronischen Formate informiert. Auf der Seite werden die einzelnen Formate beschrieben und alle möglichen Felder zusammen mit beschreibenden Details wie Beispielwerten und Hinweisen aufgelistet, zum Beispiel, ob die Felder Pflichtfelder sind, ob sie Validierungsregeln haben usw.
