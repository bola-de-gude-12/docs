# Option zum Einbetten von PDF-Dateien

| Funktion                             |           Öffentliche Vorschau          | Allgemeine Verfügbarkeit Online |
| ------------------------------------ | :-------------------------------------: | :-----------------------------: |
| Option zum Einbetten von PDF-Dateien | {{checkmark}} Sep. 2025 |    Okt. 2025    |

## Vorteile

Unabhängig davon, ob Sie es als Ausfallsicherung oder als zusätzliche Komfortebene verwenden, können Sie mit dieser Funktion Dateien wie PDFs an ein ausgehendes eDokument anhängen.

## Details zur Funktion

Sie haben die Möglichkeit, das vom Standard-Business Central generierte PDF (oder andere Anhänge) in ein eDokument einzubetten, bevor Sie es über das Continia Delivery Network senden.
