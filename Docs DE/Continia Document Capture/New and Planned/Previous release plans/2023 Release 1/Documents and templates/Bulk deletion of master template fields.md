# Massenlöschung von Mastervorlagenfeldern

| Funktion                                 |   Öffentliche Vorschau  |     Allgemeine Verfügbarkeit Online     |
| ---------------------------------------- | :---------------------: | :-------------------------------------: |
| Massenlöschung von Mastervorlagenfeldern | {{checkmark}} März 2023 | {{checkmark}} Apr. 2023 |

## Vorteile

Mit der Einführung dieser Funktion können Sie Mastervorlagenfelder aus mehreren zugehörigen Vorlagen in einem einzigen Vorgang löschen. Das bedeutet, dass Sie nicht mehr den zeitaufwändigen Prozess durchlaufen müssen, irrelevante Felder manuell in einer Vorlage nach der anderen zu löschen. Continia Document Capture erledigt dies in einem Durchgang für Sie und spart Ihnen so Zeit und Ärger.

## Details zur Funktion

Wenn Sie ein Mastervorlagenfeld löschen, werden Sie gefragt, ob Sie das Feld auch aus allen zugehörigen Vorlagen löschen möchten. Wenn Sie **Ja** auswählen, wird das Feld anschließend automatisch aus allen zugehörigen Vorlagen gelöscht, sofern es nicht bereits in einer dieser Vorlagen zum Erfassen von Werten verwendet wurde.
