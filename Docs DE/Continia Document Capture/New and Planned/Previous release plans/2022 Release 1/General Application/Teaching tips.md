# Unterrichtstipps

| Funktion         | Allgemeine Verfügbarkeit On-Premises |     Allgemeine Verfügbarkeit Online     | Öffentliche Vorschau |
| ---------------- | :----------------------------------: | :-------------------------------------: | :------------------: |
| Unterrichtstipps |        {{checkmark}} März 2022       | {{checkmark}} Apr. 2022 |           -          |

## Unterstützte Versionen

Die Funktion wird in Microsoft Dynamics 365 Business Central 2021 Release Wave 1 (BC v18) und später verfügbar sein.

## Vorteile

Unterrichtstipps erleichtern neuen Benutzern den Einstieg in die Verwendung von Continia Document Capture.

## Details zur Funktion

In Microsoft Dynamics 365 Business Central 2021 Release Wave 1 (BC v18) hat Microsoft das Konzept von [Unterrichtstipps](https://docs.microsoft.com/en-us/dynamics365/business-central/dev-itpro/administration/onboarding-teaching-tips-tours) (Teaching Tips) eingeführt. So hat Continia dem Belegjournal in Document Capture 2021 R2 (8.00) ebenfalls eine Reihe nützlicher Unterrichtstipps hinzugefügt, und in der kommenden Version 9.00 werden weitere hinzugefügt.

Die neuen Unterrichtstipps finden Sie auf den folgenden Seiten:

- **Belegkategorie**
- **Vorlagenkarte**
- **Übersetzungen**
- **Standard Betragsverteilungscode Karte**
- **Dokumentenkarte**
- **Bestellabgleich**
- **Belege aufteilen und zusammenführen**
- **Einkaufszuordnung**
