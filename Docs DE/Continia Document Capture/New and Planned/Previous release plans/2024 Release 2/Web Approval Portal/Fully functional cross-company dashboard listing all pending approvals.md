# Mandantenübergreifende Übersicht aller ausstehenden Genehmigungen einschließlich verfügbarer Aktionen

| Funktion                                                                                              |           Öffentliche Vorschau          |     Allgemeine Verfügbarkeit Online     |
| ----------------------------------------------------------------------------------------------------- | :-------------------------------------: | :-------------------------------------: |
| Mandantenübergreifende Übersicht aller ausstehenden Genehmigungen einschließlich verfügbarer Aktionen | {{checkmark}} Sep. 2024 | {{checkmark}} Okt. 2024 |

## Vorteile

Alle ausstehenden Genehmigungen werden in einer einzigen, mandantenübergreifenden Ansicht mit vollem Funktionsumfang im Continia Web Approval Portal zusammengefasst, sodass kein Wechsel zwischen Mandanten mehr erforderlich ist. Dadurch wird Ihr Arbeitsablauf erheblich rationalisiert und beschleunigt.

## Details zur Funktion

Als Genehmiger haben Sie im Web Approval Portal eine mandantenübergreifende Übersicht über alle Ihre ausstehenden Genehmigungen, einschließlich Einkaufsverträge. Sie sehen bei jedem Genehmigungseintrag, um welchen Mandanten es sich handelt und können sämtliche Genehmigungsaktionen direkt von der Übersicht aus durchführen. Das bedeutet beispielsweise, dass Sie in jedem Ihrer Mandanten jedes zur Genehmigung vorgelegte Dokument problemlos genehmigen, ablehnen oder auf Abwarten setzen können, ohne manuell zwischen Mandanten wechseln zu müssen.
