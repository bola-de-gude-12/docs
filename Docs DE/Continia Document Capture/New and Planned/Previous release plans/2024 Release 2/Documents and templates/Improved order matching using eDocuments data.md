# Verbesserter Bestellabgleich mithilfe von eDokumente-Daten

| Funktion                                                   |           Öffentliche Vorschau          |     Allgemeine Verfügbarkeit Online     |
| ---------------------------------------------------------- | :-------------------------------------: | :-------------------------------------: |
| Verbesserter Bestellabgleich mithilfe von eDokumente-Daten | {{checkmark}} Sep. 2024 | {{checkmark}} Okt. 2024 |

## Vorteile

Wenn Continia eDokumente-Daten im Abgleichprozess von Dokumenten einbezogen werden, können Bestellungen einfacher mit Einkaufsrechnungen abgeglichen werden und es wird ein höherer Automatisierungsgrad erreicht. Die eDokumente-Daten werden auch verwendet, um die Richtigkeit der Einkaufsrechnungen zu prüfen, da jede Einkaufsrechnungszeile aufgrund der 1:1-Beziehung zwischen beiden vollständig mit der entsprechenden Bestellzeile übereinstimmen muss.

## Details zur Funktion

Zusätzliche Daten aus Continia eDokumente werden beim Abgleichen von Bestellungen mit Einkaufsrechnungen verwendet, was den Prozess deutlich verbessern wird.
