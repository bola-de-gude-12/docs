# Automatisches Ausfüllen von Vorlagenfeldern mit Standardwerten

| Funktion                                                       |           Öffentliche Vorschau          |     Allgemeine Verfügbarkeit Online     |
| -------------------------------------------------------------- | :-------------------------------------: | :-------------------------------------: |
| Automatisches Ausfüllen von Vorlagenfeldern mit Standardwerten | {{checkmark}} Sep. 2024 | {{checkmark}} Okt. 2024 |

## Vorteile

Es ist möglich, das in einem importierten Dokument ein Feldwert im Belegjournal fehlt, der aber an anderer Stelle verfügbar ist – beispielsweise in der **Kreditorenkarte**. Mit dieser Funktion wird dies vermieden, da die Funktion automatisch Standardwerte in Vorlagenfelder einfügt, die andernfalls leer gewesen wären.

## Details zur Funktion

Wenn derzeit für ein importiertes Dokument kein dokumentspezifischer Wert in einem Vorlagenfeld erfasst wird, wird für dieses Feld im Belegjournal kein Wert angezeigt. Mit der Implementierung dieser Funktion füllt Continia Document Capture das entsprechende Vorlagenfeld automatisch mit dem Standardwert der entsprechenden Karte, anstatt das Feld leer zu lassen.
