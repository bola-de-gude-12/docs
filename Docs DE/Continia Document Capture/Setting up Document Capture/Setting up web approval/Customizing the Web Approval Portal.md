# Das Web Approval Portal anpassen

Die Vorgehensweise zum Anpassen des Continia Web Approval Portals ist für Online- und On-Premises-Installationen von Microsoft Dynamics 365 Business Central gleich.

So passen Sie das Portal an:

1. Suchen Sie {{search}} nach **Document Capture Einrichtung** und wählen Sie dann den entsprechenden Link aus.
2. Wählen Sie in der Aktionsleiste **Einrichtung** > **Web-Genehmigungen** aus, um die Seite **Document Capture Einrichtung / Continia Web Genehmigungen** zu öffnen.
3. Wählen Sie im Inforegister **Allgemein** unter **Anzuzeigende Felder** aus, welche Felder ggf. zusätzlich zu den standardmäßigen Pflichtfeldern bei der Genehmigung im Web Approval Portal angezeigt werden sollen.
4. Geben Sie unter **Einstellungen weiterleiten** an, ob Benutzer Belege an andere Genehmiger weiterleiten und gleichzeitig genehmigen können, ob sie Belege zur Genehmigung delegieren können (d. h.
5. Geben Sie unter **Archiv** an, ob Benutzer des Web Approval Portals alle Belege im Archiv oder nur eigene Belege einsehen können.
6. **Optional**: Weitere Anpassungsoptionen (einschließlich der Möglichkeit, die Standardeinstellungen für die Sprache, die Zeitzone und die Formatierung der Einheiten zu bearbeiten) sind verfügbar, wenn Sie in der oberen rechten Ecke **Mehr anzeigen** auswählen.
7. Wählen Sie im Inforegister **Dimensionen** aus, welche Kopf- und/oder Zeilendimensionen in Einkaufsbelegen im Web Approval Portal angezeigt werden sollen.
8. Wählen Sie **OK**, wenn Sie fertig sind, um Ihre Änderungen zu speichern und die Seite zu schließen.
