# Registrierung von Belegen als Fibu. Buch.-Blattzeilen einrichten

In diesem Artikel wird beschrieben, wie Sie eingehende Belege als Zeilen in einem Fibu Buch.-Blatt in Document Capture registrieren.

So richten Sie Belege so ein, dass sie direkt als Fibu. Buch-Blattzeilen registriert werden:

1. Wählen Sie das Symbol {{search}}, geben Sie **Belegkategorien** ein, und wählen Sie dann den zugehörigen Link.
2. Wählen Sie die Zeile für Einkauf aus (nicht den Code **EINKAUF**), und wählen Sie dann **Bearbeiten** in der Aktionsleiste aus, um die Seite **Belegkategorie** zu öffnen.
3. Nehmen Sie im Inforegister **Allgemein** unter **Buch.-Blattvorlage** die folgenden Änderungen vor:
   1. Wählen Sie unter **Buch.-Blattvorlagenname** die Buch.-Blattvorlage aus, die verwendet werden soll, wenn Belege als Fibu. Buch-Blattzeilen registriert werden sollen.
   2. Wählen Sie unter **Buch.-Blattname** den Namen des Buch.-Blattes aus, der verwendet werden soll, wenn Belege als Fibu. Buch-Blattzeilen registriert werden sollen.
4. Wählen Sie im Inforegister **Vorlagen** die entsprechende Vorlage in der Liste aus, und wählen Sie dann **Bearbeiten** aus, um die **Vorlagenkarte** zu öffnen.
5. Legen Sie im Inforegister **Einkaufsbelege** die Option **Rechnung Reg. Schritt 1** und/oder **Gutschrift Reg. Schritt 1**, auf **Buchblattzeilen erfassen**, je nachdem, welche Belege (Rechnungen und/oder Gutschriften) Sie als Fibu. Buch-Blattzeilen registrieren möchten
6. **Optional**: Wenn Sie möchten, dass Fibu-Buch.-Blattzeilen bei der Registrierung gebucht werden, können Sie **Rechnung Reg. Schritt 2** und/oder **Gutschrift Reg. Schritt 2** auf **Buchen** setzen, je nach dem Belegtyp, den Sie registrieren möchten

Mit diesen Einstellungen ändern Sie, wie Belege registriert werden, sodass anstelle von Belegen in Microsoft Dynamics 365 Business Central jetzt Fibu. Buch-Blattzeilen erstellt werden. Weitere Informationen finden Sie unter [Belege als Fibu. Buch-Blattzeilen registrieren](@DC-67#belege-als-fibu-buch-blattzeilen-registrieren).
