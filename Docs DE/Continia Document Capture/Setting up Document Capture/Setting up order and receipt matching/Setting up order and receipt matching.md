# Überblick über die Einrichtung von Bestell- und Lieferabgleich

Mit Continia Document Capture können Sie alle eingehenden Rechnungen und Gutschriften mit den entsprechenden Belegen abgleichen (z. B. können Rechnungen mit entsprechenden Bestellungen und Lieferungen abgeglichen werden), um sicherzustellen, dass alles überprüft wird und beispielsweise Rechnungen bezahlt werden können. Dies können Sie entweder selbst manuell oder automatisch mit Document Capture durchführen.

Das manuelle Verfahren erfordert keinerlei Einrichtung, während der automatische Abgleich entsprechend den Anforderungen Ihres Unternehmens aktiviert und konfiguriert werden muss. Hierfür finden Sie in der folgenden Tabelle eine Liste nützlicher Links:

| Thema                                                                                                                                        | Siehe                                                        |
| -------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------ |
| Aktivieren und Konfigurieren des automatischen Abgleichs, einschließlich Abweichungen                                                        | [Automatischen Abgleich konfigurieren](@DC-68)               |
| Aktivieren und Konfigurieren des Zeilenabgleichs, einschließlich Abweichungen                                                                | [Zeilenabgleich einrichten](@DC-69)                          |
| Einrichten einer Aufgabenwarteschlange, um Belege automatisch in großen Mengen abzugleichen                                                  | [Belege mit einer Aufgabenwarteschlange abgleichen](@DC-479) |
| Aktivieren der automatischen Genehmigung von Einkaufsbelegen, die mit zugehörigen Belegen innerhalb vorgegebener Abweichungen übereinstimmen | [Automatische Genehmigung aktivieren](@DC-461)               |
| Einrichten von Document Capture, um zu prüfen, ob Belege vor der Registrierung abgeglichen wurden                                            | [Abgleichprüfungen vor der Registrierung einrichten](@DC-84) |
