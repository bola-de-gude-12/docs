# Grundlegendes zum Continia Delivery Network

Das Continia Delivery Network ist ein Service, der in das Peppol eDelivery Network und das NemHandel-Netzwerk integriert ist und Ihnen so den Empfang elektronischer Einkaufsrechnungen und Gutschriften direkt in Microsoft Dynamics 365 Business Central ermöglicht.

Sobald Ihr Unternehmen im Continia Delivery Network registriert ist, werden alle eingehenden elektronischen Belege genauso angezeigt wie PDF-Dateien, die Ihnen per E-Mail zugesandt werden. Eine zusätzliche Einrichtung oder andere Software ist nicht erforderlich.

Sie können sich beim Continia Delivery Network direkt über Business Central registrieren und es ist kein weiterer Service von Drittanbietern erforderlich, um sich mit dem Peppol- und dem NemHandel-Netzwerk zu verbinden. Informationen zur Registrierung beim Continia Delivery Network finden Sie unter [Das Continia Delivery Network einrichten](/Continia Document Capture/Setting up Document Capture/Setting up general business functionality/Setting up the Continia Delivery Network.md).

Continia Document Capture unterstützt eine Vielzahl elektronischer Dokumentenformate, die mit dem Peppol- und dem NemHandel-Netzwerk kompatibel sind. Eine vollständige Liste der unterstützten Formate finden Sie unter [Unterstützte elektronische Dokumentenformate](/Continia Document Capture/Getting Started/Importing electronic documents/Supported Formats.md).
