# Überblick über den Import elektronischer Dokumente

Der Import elektronischer Dokumente in Continia Document Capture ist der einfache Weg zur Optimierung Ihres Unternehmens. Immer mehr Behörden auf der ganzen Welt schreiben den elektronischen Austausch von Dokumenten vor, und dieser erfreut sich auch bei Business-to-Business-Organisationen immer größerer Beliebtheit, weshalb er ein wesentlicher Bestandteil von Document Capture ist.

Ihre Optionen zum Importieren elektronischer Dokumente in Document Capture hängen davon ab, welche elektronischen Dokumente und Dateien Sie importieren möchten. Sie können Dateien importieren, indem Sie sie entweder an eine von Document Capture überwachte E-Mail-Adresse senden oder eine Verbindung zum Continia Delivery Network herstellen.

Das Continia Delivery Network ist ein Service, der in das Peppol eDelivery Network eDelivery Network integriert ist und Ihnen so den Empfang elektronischer Einkaufsrechnungen und Gutschriften direkt in Microsoft Dynamics 365 Business Central ermöglicht. Continia ist ein zertifizierter Peppol Access Point, der die Integration Ihres Unternehmens in das Peppol eDelivery Network durch Drittanbieter überflüssig macht. Wenn Sie sich beim Continia Delivery Network registrieren, werden Sie automatisch als Empfänger im globalen Peppol eDelivery Network eingerichtet.
