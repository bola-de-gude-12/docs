# Elektronische Dokumente in Deutschland importieren
Als Kunde in Deutschland können Sie elektronische Dokumente entweder per E-Mail (als angehängte Dateien) oder über das PEPPOL eDelivery Network in Continia Document Capture importieren.

Document Capture verbindet Sie direkt mit dem PEPPOL eDelivery Network, sodass Sie sich nicht mehr bei Drittanbietern für die Integration registrieren müssen. Sobald Sie verbunden sind, können Sie elektronische Dokumente von Kreditoren in Deutschland und anderen Ländern empfangen. Weitere Informationen zum Herstellen einer Verbindung zum PEPPOL eDelivery Network finden Sie unter [Das Continia Delivery Network einrichten](/Continia Document Capture/Setting up Document Capture/Setting up general business functionality/Setting up the Continia Delivery Network.md).

Um die allgemeine Einführung von PEPPOL zu fördern, verpflichtet der Gesetzgeber die öffentliche Verwaltung in Deutschland, elektronische Dokumente über das PEPPOL eDelivery Network empfangen zu können.

Das XRechnung-Format wurde in Deutschland weitgehend übernommen, sowohl in Unternehmen als auch bei Behörden. Document Capture unterstützt dieses Format und Dokumente können über das PEPPOL eDelivery Network empfangen werden. XRechnung-Versionen vor 1.2 werden derzeit nicht unterstützt.

ZUGFeRD ist ein weiteres Format, das in Deutschland verwendet wird. Es soll jedoch langfristig durch PEPPOL und XRechnung ersetzt werden. Document Capture unterstützt derzeit das Format bis einschließlich Version 2.1.1.

Sie können auch elektronische Dokumente von Kreditoren aus anderen Ländern empfangen und importieren. Die gängigsten Formate werden unterstützt, jedoch nicht alle. Eine Liste der unterstützten Formate finden Sie unter [Unterstützte Formate](Supported Formats.md).
