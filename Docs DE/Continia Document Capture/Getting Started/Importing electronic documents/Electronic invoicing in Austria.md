# Elektronische Dokumente in Österreich verwalten

Als Continia Document Capture-Kunde in Österreich können Sie elektronische Dokumente über das Peppol eDelivery Network austauschen. Darüber hinaus können Sie elektronische Dokumente per E-Mail-Anhang direkt in Document Capture empfangen.
