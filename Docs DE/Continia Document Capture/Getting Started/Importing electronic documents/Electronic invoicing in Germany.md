# Elektronische Dokumente in Deutschland verwalten

Als Continia Document Capture-Kunde in Deutschland können Sie elektronische Dokumente über das Peppol eDelivery Network austauschen. Darüber hinaus können Sie elektronische Dokumente per E-Mail-Anhang direkt in Document Capture empfangen.
