# Das Einkaufsvertragsarchiv anzeigen

Alle Statusänderungen von Einkaufsverträgen werden im **Einkaufsvertragsarchiv** protokolliert. Beispielsweise wird automatisch ein Protokolleintrag erstellt, wenn ein Vertrag zur Genehmigung gesendet, genehmigt, zwangsgenehmigt, storniert, abgelehnt oder weitergeleitet wird oder wenn sich der Jahresbetrag eines Vertrags ändert.

> [!NOTE]
>
> Das **Einkaufsvertragsarchiv** wurde aktualisiert, um die im Rahmen von Continia Document Capture 25.00 veröffentlichte [Genehmigungsfunktion](@DC-87) zu integrieren. Es enthält aber weiterhin die Werte des Überprüfungsprozesses.

So greifen Sie auf das Archiv zu:

1. Suchen Sie ({{search}}) nach **Einkaufsverträge** und wählen Sie dann den entsprechenden Link aus.
2. Wählen Sie in der Vertragsliste den Vertrag aus, dessen Historie Sie anzeigen möchten.
3. Kllicken Sie in der Aktionsleiste auf **Genehmigung anfordern** > **Archivierte Versionen**.

Dadurch wird die Seite **Einkaufsvertragsarchiv** geöffnet, die Ihnen einen vollständigen Überblick über die Historie des ausgewählten Vertrags bietet. Jede Statusänderung des Vertrages wird in der Übersichtstabelle durch eine Versionszeile repräsentiert und jede dieser Zeilen ist mit einer Versionsnummer versehen.

Wenn Sie weitere Details zu einer bestimmten Version des Vertrags anzeigen möchten, wählen Sie die Zeile dieser Version in der Tabelle aus und klicken Sie dann auf die Nummer in der Spalte **Nr.**, um eine neue Seite mit allen Details der ausgewählten Version zu öffnen.
