# Die Kategorie „Geb. Einkaufslieferungen“

Mit der Kategorie „Geb. Einkaufslieferungen“ – **EK-LIEF** – in Continia Document Capture können Sie Belege für den Kauf von Waren oder Dienstleistungen (wie Lieferscheine oder Frachtbriefe) per OCR verarbeiten und so durchsuchbar machen.

Beim Importieren einer Einkaufslieferung prüft diese Belegkategorie alle erkannten Werte und vergleicht sie mit den Identifikationsfeldern **Kreditor**, **Kreditoren-Bestellnr.** und **Kred.-Lieferungsnr.**. [Jedes dieser Felder hat eine Bewertung](@DC-81#calculation-confidence-scores), die auf der Seite **Identifikationsfelder** der Kategorie „Geb. Einkaufslieferungen“ bearbeitet werden kann.

Das resultierende Dokument wird dem ursprünglich gebuchten Einkaufsbeleg hinzugefügt.

## So arbeiten Sie mit der Kategorie „Geb. Einkaufslieferungen“

So verarbeiten Sie Belege, die in die Kategorie „Geb. Einkaufslieferungen“ importiert wurden:

1. Suchen Sie {{search}} nach **Belegkategorien** und wählen Sie dann den entsprechenden Link aus.

2. Klicken Sie auf der Seite **Belegkategorien** auf den Code **EK-LIEF**, um die Kategorie „Gebuchte Einkaufslieferungen“ zu öffnen.

3. Wählen Sie im Belegjournal den gewünschten Beleg aus und stellen Sie sicher, dass das Feld **Einkaufslieferkopf** den richtigen Wert übernommen hat.

4. Klicken Sie in der Aktionsleiste auf **Start > Registrieren**, um den Beleg zu registrieren.

Den registrierte Beleg können Sie folgendermaßen anzeigen:

- Suchen Sie ({{search}}) nach **Posten suchen** und wählen Sie den entsprechenden Link aus.
- Fügen Sie die Infobox **Dokumente** zur Seite **Geb. Einkaufslieferung** hinzu. Diese ist standardmäßig ausgeblendet, kann aber angezeigt werden, indem Sie oben rechts auf der Seite auf **Einstellungen** ({{settings}}) > **Personalisieren** klicken.
