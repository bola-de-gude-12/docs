# Genehmigungsposten

Auf der Seite **Genehmigungsposten** können Sie alle für die Einkaufsgenehmigung erforderlichen Aktionen durchführen, sofern Sie [als Genehmiger eingerichtet](@DC-23#so-richten-sie-benutzer-als-genehmiger-ein) wurden.

Sie können die Zeilen der einzelnen Belege bearbeiten (sofern Sie die entsprechenden Berechtigungen haben) und falls nötig Genehmigungsbemerkungen hinzufügen. Außerdem ist die Dokumentenvorschau in die Seite integriert, sodass Sie jeden einzelnen gescannten Beleg auf der rechten Seite des Bildschirms anzeigen können. Sie können die Genehmigung sogar bei Abwesenheit an andere Benutzer weiterleiten.

Die folgenden Abschnitte beschreiben die Funktionalität auf der Seite genauer.

## Die Aktionsleiste

Die Aktionsleiste enthält eine Reihe häufig verwendeter Aktionen für die Einkaufsgenehmigung, die übersichtlich an einem Ort angeordnet sind. Die meisten Aktionen kommen von Continia Document Capture, während einige Standardaktionen von Microsoft Dynamics 365 Business Central hinzugefügt wurden, damit Sie leichten Zugriff darauf haben. ## Zusätzliche Funktionalität

Im Inforegister **Einkaufszeilen** können Sie die Zeilen des Belegs anzeigen, wie sie von Document Capture erfasst oder von anderen Benutzern bearbeitet/eingegeben wurden. Bei Bedarf können Sie die Zeilen auch selbst bearbeiten, um etwaige Fehler zu korrigieren oder um neue Zeilen hinzuzufügen. Dies ist möglich, sofern für Sie als Genehmiger die Option **Darf Buchungszeilen editieren** aktiviert wurde. Weitere Informationen finden Sie unter [So richten Sie Benutzer als Genehmiger ein](@DC-23#so-richten-sie-benutzer-als-genehmiger-ein).

Im Inforegister **Genehmigungsbemerkungen** sind alle Bemerkungen zu dem ausgewählten Genehmigungsposten in chronologischer Reihenfolge aufgeführt, zusammen mit Datum und Uhrzeit sowie den IDs der Benutzer, die die Bemerkungen hinzugefügt haben. Um eine neue Bemerkung hinzuzufügen, gehen Sie folgendermaßen vor:

1. Klicken Sie in der Aktionsleiste auf **Bemerkungen**, um die Seite **Genehmigungsbemerkungen** zu öffnen.
2. Klicken Sie in der Aktionsleiste auf **Neu**.
3. Geben Sie Ihre Bemerkung ein und schließen Sie die Seite, um zur Seite **Genehmigungsposten (Document Capture)** zurückzukehren.

Ihre Bemerkung wird jetzt als letzte Bemerkung in der Liste der Bemerkungen im Inforegister **Genehmigungsbemerkungen** angezeigt.

Rechts auf der Seite zeigt die Dokumentenvorschau eine OCR-verarbeitete Version des zu genehmigenden Belegs an. Die Dokumentenvorschau auf der Seite **Genehmigungsposten (Document Capture)** ist nicht interaktiv wie im Belegjournal oder auf der Dokumentenkarte. Abgesehen davon ist sie jedoch identisch mit der Dokumentenvorschau auf diesen Seiten: Sie hat das gleiche Gesamterscheinungsbild, Sie können sie durch Ziehen des vertikalen Griffs auf der linken Seite in der Größe ändern, und sie verfügt über ein Dropdown-Menü mit Anhängen in der oberen linken Ecke. Weitere Informationen finden Sie unter [Dokumentenvorschau](@DC-71#dokumentenvorschau).
