# Zeilen beim Abgleich filtern

Wenn Sie [Belege manuell abgleichen](@DC-62) und [Abweichungen beim Abgleich bearbeiten](@DC-63), kann es hilfreich sein, die Liste der potenziellen Übereinstimmungen einzugrenzen, um deren Identifizierung einfacher zu machen. Dies gilt insbesondere, wenn in Ihrem Unternehmen eine große Anzahl von Belegen eingeht. Dies ist mit vordefinierten Filtern möglich.

## So filtern Sie die Liste der Zeilen beim Abgleich

Die Filter können entweder auf der Seite **Bestellabgleich** oder der Seite **Gutschrift Abgleich** angewendet werden, je nachdem, welchen Belegtyp Sie im Belegjournal abgleichen möchten.

Um eine dieser Seiten zu öffnen und Filter auf die Listen der übereinstimmenden Belegzeilen anzuwenden, gehen Sie folgendermaßen vor:

1. Suchen Sie {{search}} nach **Belegkategorien** und wählen Sie dann den entsprechenden Link aus.

2. Klicken Sie auf den Code **EINKAUF**, um das Belegjournal zu öffnen.

3. Wählen Sie in der Belegliste den Beleg aus, den Sie mit anderen Belegen abgleichen möchten.

4. Klicken Sie in der Aktionsleiste auf **Start** > **Bestellabgleich**.

5. Unter **Abgleich Übersicht** werden in zwei Zeilenabschnitten relevante Zeilen aus zugehörigen Belegen aufgelistet, abhängig davon, welchen Belegtyp Sie oben in Schritt 3 ausgewählt haben. Um diese Listen zu filtern, klicken Sie auf **Start** in der Aktionsleiste und wählen Sie dann die Filter aus, die Sie anwenden möchten:

   <br>

   | Filter                                | Beschreibung                                                                                                                                                                                                                                                                                                                                                                                         |
   | ------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
   | **Bestellnummer filtern**             | Zeigt nur Zeilen mit der Bestellnummer an, die in der Rechnung oder Gutschrift angegeben ist und die von Document Capture erkannt wurde.                                                                                                                                                                                                                                             |
   | **Nicht abgeglichene Zeilen filtern** | Zeigt nur Zeilen an, die noch nicht abgeglichen wurden.                                                                                                                                                                                                                                                                                                                              |
   | **Auf Differenzen filtern**           | Zeigt nur Zeilen an, in denen sich die direkten Einstandspreise unterscheiden. Wenn Sie beispielsweise in Schritt 3 oben eine Rechnung ausgewählt haben, werden bei Anwendung dieses Filters in den Listen Zeilen mit unterschiedlichen Werten in den Feldern **EK-Preis (Bestellung)** und **EK-Preis (Rechnung)** angezeigt. |

   <br>

6. Um einen ausgewählten Filter zu löschen, klicken Sie auf **Start** in der Aktionsleiste aus und wählen Sie den relevanten Filter dann erneut aus.
