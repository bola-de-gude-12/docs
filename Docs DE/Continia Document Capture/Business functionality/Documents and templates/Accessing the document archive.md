# Auf das Belegarchiv zugreifen

Über das Belegarchiv haben Sie jederzeit Zugriff auf alle bisher importierten und verarbeiteten Belege. Dort werden alle im Belegjournal registrierten Belege automatisch gespeichert.

Sie können auf das vollständige Archiv entweder in Microsoft Dynamics 365 Business Central oder über das Continia Web Approval Portal zugreifen. Die am häufigsten verwendeten Zugriffsmethoden werden unten beschrieben. Außerdem finden Sie alternative Möglichkeiten, um nach archivierten Dateien zu suchen und sie zu öffnen.

## So greifen Sie in Business Central auf das Archiv zu

So greifen Sie in Business Central auf das Belegarchiv zu:

1. Suchen Sie {{search}} nach **Belegkategorien** und wählen Sie dann den entsprechenden Link aus.
2. Klicken Sie auf den Code der entsprechenden Belegkategorie, zum Beispiel **EINKAUF**, um das Belegjournal zu öffnen.
3. Gehen Sie in der oberen rechten Ecke über der Aktionsleiste zu **Statusfilter**, um die Filteroptionen anzuzeigen. Klicken Sie dann auf **Registriert**.

Die Belegliste wird anschließend gefiltert, um alle registrierten (d. h. archivierten) Belege anzuzeigen.

## So greifen Sie im Web Approval Portal auf das Archiv zu

So greifen Sie über das Web Approval Portal auf das Belegarchiv zu:

1. Öffnen Sie das Web Approval Portal.
2. Klicken Sie in der Menüleiste oben auf **Einkauf** > **Archiv**, um das Belegarchiv zu öffnen.
3. Um das Belegarchiv zu durchsuchen, geben Sie Ihre Suchkriterien in die dafür vorgesehenen Felder ein und klicken Sie auf **Suchen** in der oberen linken Ecke.

Es werden alle archivierten Belege aufgelistet, die Ihren Kriterien entsprechen.

## So öffnen Sie archivierte Belege aus Sachposten

So öffnen Sie einen archivierten Beleg von einem Sachposten aus:

1. Suchen Sie ({{search}}) nach **Sachposten**.
2. Wählen Sie auf der Seite **Sachposten** in der Liste den Posten aus, für den Sie die Original-PDF-Datei anzeigen möchten.
   > [!NOTE]
   > Sie können nur Posten für registrierte Einkaufsbelege auswählen, die in Document Capture verarbeitet wurden. Andernfalls ist die in Schritt 3 unten erwähnte Option **PDF-Datei** nicht sichtbar.
3. Klicken Sie in der Aktionsleiste auf **Posten** > **PDF-Datei**, um die Original-PDF-Datei zu öffnen.

## So öffnen Sie archivierte Belege aus Kreditorenposten

So öffnen Sie einen archivierten Beleg von einem Kreditorenposten aus:

1. Suchen Sie ({{search}}) nach **Kreditorenposten**.
2. Wählen Sie auf der Seite **Kreditorenposten** in der Liste den Posten aus, für den Sie die Original-PDF-Datei anzeigen möchten.
   > [!NOTE]
   > Sie können nur Posten für registrierte Einkaufsbelege auswählen, die in Document Capture verarbeitet wurden. Andernfalls ist die in Schritt 3 unten erwähnte Option **PDF-Datei** nicht sichtbar.
3. Klicken Sie in der Aktionsleiste auf **Posten** > **PDF-Datei**, um die Original-PDF-Datei zu öffnen.
