# Verbesserter Benutzerimport in der Continia Benutzereinrichtung

| Funktion                                  |         Allgemeine Verfügbarkeit        |           Öffentliche Vorschau          |
| ----------------------------------------- | :-------------------------------------: | :-------------------------------------: |
| Mehrere Benutzer gleichzeitig importieren | {{checkmark}} Okt. 2025 | {{checkmark}} Sep. 2025 |

## Vorteile

Diese Funktion vereinfacht den Import einer großen Anzahl von Benutzern in die Continia Benutzereinrichtung. Bisher wurde bei der Erstellung für jeden Benutzer ein Dialogfeld zur Bestätigung angezeigt. Dieses Dialogfeld wurde entfernt, wodurch der Import einer großen Anzahl von Benutzern schneller und effizienter durchgeführt werden kann. Das Ergebnis ist eine enorme Zeitersparnis für Administratoren und eine Reduzierung der manuellen Schritte während der Einrichtung.

Der verbesserte Workflow minimiert Unterbrechungen, verbessert die Skalierbarkeit der Benutzerverwaltung und sorgt für ein einfacheres Onboarding in Unternehmen, die eine große Anzahl von Benutzern verwalten.

## Details zur Funktion

Die Funktion zum Import von Benutzern in der Continia Benutzereinrichtung wurde optimiert, um die Effizienz und Benutzerfreundlichkeit zu verbessern. Die Funktion bietet Ihnen folgende Vorteile:

- Einen vereinfachten Importvorgang – Das Bestätigungsdialogfeld, das beim Import für jeden Benutzer angezeigt wurde, wurde entfernt, sodass das System lange Benutzerlisten effizient verarbeiten kann, ohne dass einzelne manuelle Bestätigungen erforderlich sind.
- Erhöhte Effizienz – Da keine wiederholte Benutzerinteraktion mehr erforderlich ist, läuft der Importvorgang deutlich schneller ab und ist daher ideal für Unternehmen, die eine große Anzahl von Benutzern gleichzeitig importieren müssen.
