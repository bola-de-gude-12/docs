# Anhänge können auf der Seite „Verteilungen“ in der Mobile Expense App angezeigt werden

| Funktion                                                                               |            Öffentliche Vorschau            |     Allgemeine Verfügbarkeit Online     |
| -------------------------------------------------------------------------------------- | :----------------------------------------: | :-------------------------------------: |
| Anhänge können auf der Seite „Verteilungen“ in der Mobile Expense App angezeigt werden | {{checkmark}} 19. Sep 2024 | {{checkmark}} Okt. 2024 |

## Vorteile

Diese Funktion bietet einen schnelleren Zugriff auf Anhänge und macht es für Expense-Benutzer einfach, Details auf Belegen schnell zu überprüfen, anstatt durch mehrere Seiten zurück navigieren zu müssen.

## Details zur Funktion

Derzeit werden Anhänge nur auf der Belegseite angezeigt. Mit dieser neuen Funktion werden Spesenanhänge jedoch auch auf der Seite „Verteilungen“ angezeigt.

Weitere Informationen finden Sie im Artikel [Kosten in Belegen verteilen](@EM-276).
