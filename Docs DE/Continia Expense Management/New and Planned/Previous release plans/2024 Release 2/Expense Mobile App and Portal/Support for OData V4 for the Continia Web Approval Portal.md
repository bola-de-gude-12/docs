# Unterstützung für OData V4 für das Continia Web Approval Portal

| Funktion                                                        | Öffentliche Vorschau | Allgemeine Verfügbarkeit Online |
| --------------------------------------------------------------- | :------------------: | :-----------------------------: |
| Unterstützung für OData V4 für das Continia Web Approval Portal |           -          |                -                |

## Vorteile

Durch die Umstellung vom SOAP-Webdienst-Zugriffsprotokoll auf OData V4 bleibt das Continia Web Approval Portal auch dann voll funktionsfähig, wenn Microsoft den Support für das SOAP-Protokoll schließlich einstellt.

## Details zur Funktion

Das Web Approval Portal basiert derzeit in vielerlei Hinsicht auf dem SOAP-Protokoll. Mit dieser Funktion endet diese Abhängigkeit und es beginnt die vollständige Umstelung auf OData V4.
