# Hervorheben der relevanten Zeilen für die erweiterte Genehmigung

| Funktion                                                         | Öffentliche Vorschau | Allgemeine Verfügbarkeit Online |
| ---------------------------------------------------------------- | :------------------: | :-----------------------------: |
| Hervorheben der relevanten Zeilen für die erweiterte Genehmigung |           -          |    Okt. 2024    |

## Vorteile

Indem im Continia Web Approval Portal nur die relevanten Zeilen für die Genehmigung hervorgehoben werden, können Sie mit dieser Funktion nicht relevanten Inhalt herausfiltern und sich ausschließlich auf das konzentrieren, was für Sie wichtig ist. Dadurch wird Ihr Arbeitsablauf erheblich optimiert.

## Details zur Funktion

Im Web Approval Portal werden die für Sie relevanten Zeilen zur Genehmigung hervorgehoben, damit Sie diese leichter erkennen können. Dies gilt nur für die erweiterte Genehmigung.
