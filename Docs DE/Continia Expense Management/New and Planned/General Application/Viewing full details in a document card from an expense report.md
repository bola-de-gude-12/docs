# Anzeige der vollständigen Details in einer Dokumentenkarte aus einer Abrechnung

| Funktion                                         |   Allgemeine Verfügbarkeit On-Premises  |     Allgemeine Verfügbarkeit Online     |   Öffentliche Vorschau  |
| ------------------------------------------------ | :-------------------------------------: | :-------------------------------------: | :---------------------: |
| Anzeigen aller Details auf einer Dokumentenkarte | {{checkmark}} Apr. 2025 | {{checkmark}} Apr. 2025 | {{checkmark}} März 2025 |

## Vorteile

Die Möglichkeit, aus einer Abrechnung direkt zu einem Expense-Dokument zu navigieren, erhöht die Transparenz und Effizienz bei der Verwaltung von Ausgaben.

## Details zur Funktion

- Benutzer können jetzt eine Ausgabe in der Abrechnung öffnen, um die vollständige Expense-Dokumentenkarte anzuzeigen.
- Sie bietet eine umfassende Anzeige aller Details zu jeder Ausgabe, die über die Listenansicht hinausgeht.

Weitere Informationen finden Sie unter [Abrechnungen einreichen und verwalten](@EM-294) und [Abrechnungen einrichten](@EM-306).
