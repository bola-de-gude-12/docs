# Konfiguration der Buchungsbeschreibung für Banktransaktionen

| Funktion                                                     |         Allgemeine Verfügbarkeit        |           Öffentliche Vorschau          |
| ------------------------------------------------------------ | :-------------------------------------: | :-------------------------------------: |
| Konfiguration der Buchungsbeschreibung für Banktransaktionen | {{checkmark}} Okt. 2025 | {{checkmark}} Sep. 2025 |

## Vorteile

Sie können jetzt anpassen, wie Banktransaktionsbuchungen im Hauptbuch angezeigt werden. Anstelle allgemeiner Beschreibungen können Sie eine Vorlage definieren, die wichtige Transaktionsdetails enthält. Dies führt zu einer schnelleren Abstimmung und verbesserter Klarheit bei Prüfungen.

## Details zur Funktion

Sie können eine benutzerdefinierte Vorlage für Buchungsbeschreibungen direkt auf der Seite Expense Management Einrichtung festlegen, ähnlich wie bei anderen Buchungsbeschreibungen für Dokumente (wie Belege oder Fahrstrecken). Die Vorlage verwendet Platzhalter, die zum Zeitpunkt der Buchung dynamisch durch tatsächliche Transaktionsdaten ersetzt werden. Beispielsweise Daten wie Mitarbeitername, Betrag oder Firmenname.
