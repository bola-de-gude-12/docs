# Häufig gestellte Fragen zu Kreditkartentransaktionen

In diesem Abschnitt finden Sie Antworten auf einige der am häufigsten gestellten Fragen zu Kreditkartentransaktionen.

## Kann ich einer Kreditkarte mehrere Expense-Benutzer hinzufügen?

Ja, in Expense Management können Sie mehreren Benutzern eine Kreditkarte zuweisen, einschließlich Reisekreditkarten und virtueller Karten.

Hierfüg gibt es zwei Möglichkeiten:

**Ohne Einrichtung**

Die Banktransaktion wird im Posteingang gestoppt und Sie müssen den Benutzer auswählen, zu dem eine bestimmte Transaktion gehört. Die Fehlermeldung sieht folgendermaßen aus:

_Kreditkartennr.: xxxx xxxx wird für mehrere Benutzer verwendet. Sie müssen daher manuell Continia Benutzer-ID auswählen_

Wählen Sie den Benutzer im Dropdown-Menü in **Continia-Benutzer-ID** aus. Nehmen Sie das für jede Transaktion vor.

**Bei Einrichtung für mehrere Benutzer und Kreditkarten-Benutzerzuordnung**

Weitere Informationen finden Sie unter [Eine Kreditkarte mehreren Expense-Benutzern zuweisen](@EM-249).
