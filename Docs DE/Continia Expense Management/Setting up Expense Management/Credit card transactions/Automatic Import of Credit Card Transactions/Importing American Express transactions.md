# American Express-Transaktionen importieren

Continia unterstützt den automatischen Import von American Express-Transaktionen direkt in Continia Expense Management. Transaktionen können direkt von American Express für American Express-Firmenkarten importiert werden.

Im folgenden Abschnitt erfahren Sie mehr darüber, was für die Einrichtung des automatischen Imports erforderlich ist.

## Transaktionen direkt von American Express importieren

Damit die Transaktionen Ihrer American Express-Firmenkarte direkt in Expense Management importiert werden können, muss zunächst ein Transaktions-Feed zu Expense Management eingerichtet werden. Dazu müssen Sie die folgenden Formulare ausfüllen:

- [American Express GL Data Feed Implementation Form](https://partnerzone.blob.core.windows.net/documents/web/Is-your-creditcard-on-the-list/AMEX/1.AMEX-Feed-setup-form.pdf)

> [!IMPORTANT]
> Die vorausgefüllten Teile des Formulars müssen unverändert bleiben, da American Express diese Informationen benötigt, um den Feed korrekt einzurichten.

- <a href="https://partnerzone.blob.core.windows.net/downloads/info/cem/2._AMEX_-_Global_Data_Transfer_Form(Continia).pdf" target="_blank">Global Data Transfer Form (GDTF)</a>
- [Anweisungen zum Ausfüllen des GDTF-Formulars](https://partnerzone.blob.core.windows.net/downloads/info/cem/2.a_AMEX_-_Guide_for_Global_Data_Transfer_Form.pdf)

Wenn Sie die beiden Formulare ausgefüllt haben, senden Sie sie an das Data Files Enablement Team von American Express: <b>electronic.transmissions.team@aexp.com</b>
