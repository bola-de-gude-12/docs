# Manueller Dateiimport

Wenn es nicht möglich ist, Ihre Kreditkartentransaktionen automatisch von Ihrem Kreditkartenanbieter oder Ihrer Bank in Continia Expense Management zu importieren, ist ab Version 2021 R1 von Expense Management der manuelle Import möglich. Weitere Informationen hierzu finden Sie im Artikel [Transaktionsvorlagen einrichten](@EM-42).
