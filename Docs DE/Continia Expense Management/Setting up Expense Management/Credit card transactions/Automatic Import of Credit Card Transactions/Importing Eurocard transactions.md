# Eurocard-Transaktionen importieren

Continia unterstützt den automatischen Import von Eurocard-Transaktionen direkt in Continia Expense Management. Transaktionen können direkt von Eurocard für Eurocard-Firmenkarten importiert werden.

Im folgenden Abschnitt werden die erforderlichen Anforderungen beschrieben, damit Eurocard Ihre Transaktionsdateien automatisch an Continia Expense Management sendet.

Wenn Sie bei Eurocard einen Ansprechpartner haben, teilen Sie diesem mit, dass Ihre Transaktionen an Continia Expense Management gesendet werden sollen. Wenn Sie keinen Ansprechpartner bei Eurocard haben, schreiben Sie je nach Standort Ihres Unternehmens an eine der folgenden E-Mail-Adressen: cosu@eurocard.dk (Dänemark), corporatesupport@eurocard.se (Schweden), corporatesupport@eurocard.no (Norwegen) oder corporatesupport@eurocard.fi (Finnland).

Eurocard stellt Ihnen dann ein PDF-Formular zur Verfügung, das Sie ausfüllen und an eine der oben genannten E-Mail-Adressen zurücksenden müssen.
