# Einer Kreditkarte mehrere Expense-Benutzer hinzufügen

In Expense Management können Sie eine Kreditkarte mehreren Benutzern zuweisen, einschließlich Reisekreditkarten und virtueller Karten.

## So fügen Sie einer Kreditkarte mehrere Expense-Benutzer hinzu

Hierzu muss die Bank oder das Reisebüro eine Kennung senden können, die den Benutzer mit den Transaktionsdaten verknüpft, beispielsweise den Namen oder die Nummer eines Mitarbeiters.

Kreditkarten-Benutzerzuordnung wird zur Identifizierung der Benutzer verwendet. Der erste Schritt in diesem Prozess besteht darin, mehrere Benutzer für die Kreditkarte Ihrer Wahl zu aktivieren. Und so geht’s:

1. Gehen Sie zu **Zahlungsarten**.
2. Wählen Sie die Zahlungsart aus, die mehreren Benutzern hinzugefügt werden soll.
3. Aktivieren Sie im Inforegister **Banktransaktionen/Kreditkarte** unter **Kreditkarte** die Option **Aktivieren Sie mehrere Benutzer pro Karte**.

Anschließend ist es möglich, einer Kreditkarte mehrere Benutzer zuzuordnen. Im nächste Schritt können Sie damit beginnen, Benutzer zuzuordnen.

Diese Funktion ist für Reisekarten vorgesehen, bei denen der Herausgeber in einem bestimmten Feld die Namen der Mitarbeiter senden kann, um so leichter zu identifizieren, zu wem die Ausgabe gehört.

1. Gehen Sie zu **Continia Benutzereinrichtung**.
2. Wählen Sie den Benutzer aus, dem Sie die Kreditkarte zuweisen möchten.
3. Wählen Sie in der Aktionsleiste **Kreditkarten** aus.
4. Geben Sie unter **Kartennr.** die Kartennummer der Kreditkarte ein.
5. Wählen Sie unter **Zahlungsart** eine Zahlungsart mit der Lookup-Funktion aus.
6. Wählen Sie unter **Anzahl Zuordnungen** die Zahl im Feld aus, um Details zur Anzahl der Feldzuordnungen zu öffnen.
7. Wählen Sie **Neu** aus.
8. Geben Sie unter **Feldnr.** die Nummer für das Feld ein, das die Informationen zur Identifizierung des Mitarbeiters enthält, der die Karte verwendet hat. Zwei Beispiele für die Eingabe in dieses Feld sind **Mitarbeiternr.** und **Name des Reisenden**.
9. Geben Sie unter **Wert** den Namen des Mitarbeiters ein. Achten Sie darauf, dass Ihre Angaben eine eindeutige Identifizierung des Mitarbeiters ermöglichen, d. h. geben Sie ggf. zusätzlich zum Vornamen auch den Nachnamen ein.
