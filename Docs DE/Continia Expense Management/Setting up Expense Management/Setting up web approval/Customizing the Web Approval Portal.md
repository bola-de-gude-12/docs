# Das Web Approval Portal anpassen

Die Vorgehensweise zum Anpassen des Continia Web Approval Portals ist für Online- und On-Premises-Installationen von Microsoft Dynamics 365 Business Central gleich.

So passen Sie das Portal an:

1. Suchen {{search}} Sie nach **Expense Management Einrichtung** und wählen Sie dann den zugehörigen Link.
2. Wählen Sie in der Aktionsleiste **Einrichtung** > **Web Genehmigung** aus, um die Seite **Expense Management Einrichtung / Continia Web-Genehmigung** zu öffnen.
3. Wählen Sie unter **Allgemein** > **Anzuzeigende Felder** aus, welche Felder ggf. zusätzlich zu den standardmäßigen Pflichtfeldern bei der Genehmigung im Web Approval Portal angezeigt werden sollen.
4. Unter **Benutzerdefinierte Genehmigungsfelder** können Sie Ihre eigenen benutzerdefinierten Felder aktivieren, die dann im Web Approval Portal angezeigt werden. Wenn Sie zusätzliche Felder benötigen, wählen Sie oben rechts **Mehr anzeigen** aus.
5. Wählen Sie **Schließen**, wenn Sie fertig sind, um Ihre Änderungen zu speichern und die Seite zu schließen.
