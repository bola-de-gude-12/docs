# Steuerpflichtige Sätze für Pauschalen festlegen

Ein Unternehmen kann seinen Mitarbeitern Beträge erstatten, die über den staatlich festgelegten steuerfreien Sätzen liegen, indem es steuerpflichtige Sätze aktiviert. Diese Funktion ist nicht verfügbar, wenn steuerpflichtige Sätze deaktiviert sind.

So konfigurieren Sie steuerpflichtige Sätze:

1. Wählen Sie das Symbol {{search}}, geben Sie **Expense Management Einrichtung** ein und wählen Sie dann den zugehörigen Link.
2. Navigieren Sie zum Inforegister **Pauschale** und aktivieren Sie **Steuerpflichtige Sätze aktivieren**.
3. Wählen Sie das Symbol {{search}}, geben Sie **Pauschalsätze** ein und wählen Sie dann den zugehörigen Link.
4. Doppelklicken Sie auf einen Satz, um die Details zu öffnen.
5. Navigieren Sie zum Inforegister **Übernachtung** und legen Sie den **Steuerpflichtiger Betrag** fest. Stellen Sie sicher, dass der Betrag den steuerfreien Pauschalsatz nicht überschreitet. Dasselbe können Sie für Mahlzeiten im Inforegister **Mahlzeiten** einrichten.
