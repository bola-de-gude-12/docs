# Pauschalgruppen einrichten

Pauschalgruppen werden verwendet, um die Einrichtung verschiedener Benutzerkategorien zu ermöglichen. Beispielsweise gelten in manchen Unternehmen für unterschiedliche Mitarbeitergruppen unterschiedliche Pauschalsätze.

Eine Pauschalgruppe ist eine Zusammenfassung aller verfügbaren Sätze für verschiedene Länder und Regionen. Eine Pauschalgruppe kann auch Pauschalencodes enthalten, die angeben, welche Konten für welchen Satz verwendet werden.

## Eine neue Pauschalgruppe einrichten

So richten Sie eine Pauschalgruppe ein:

1. Wählen Sie das Symbol {{search}}, geben Sie **Pauschalgruppen** ein, und wählen Sie den zugehörigen Link.
2. Wählen Sie in der Aktionsleiste **Neu** aus.
3. Geben Sie im Inforegister **Allgemein** einen Code unter **Code** ein.
4. Optional: Geben Sie im Feld **Beschreibung** eine Beschreibung ein.
5. Optional: Wenn diese Pauschalgruppe die Standardpauschalgruppe sein soll, aktivieren Sie **Standard**.
6. Optional: Geben Sie einen Wert in **Zielland/-Region** ein. Dies ist nur relevant, wenn **Mehrere Reiseziele aktivieren** auf der Seite **Expense Management Einrichtung** aktiviert wurde.

## Einer Pauschalgruppe Pauschalen hinzufügen

Sie können Ihren Pauschalgruppen verschiedene Arten von Pauschalen hinzufügen, müssen diese jedoch zuvor auf der Seite **Expense Management Einrichtung** aktivieren. Beachten Sie, dass die Pauschalen **Übernachtung** und **Mahlzeiten** standardmäßig aktiviert sind.

So fügen Sie einer Pauschalgruppe eine Übernachtungspauschale hinzu:

1. Wählen Sie das Symbol {{search}}, geben Sie **Pauschalgruppen** ein, und wählen Sie den zugehörigen Link.
2. Wählen Sie in der Aktionsleiste die Pauschalgruppe aus, der Sie Pauschalen hinzufügen möchten.
3. Wählen Sie unter **Übernachtungspauschale Code** das Dropdown-Menü und dann **Neu**.
4. Geben Sie unter **Code** einen Namen für Ihren Pauschalencode ein.
5. Geben Sie unter **Beschreibung** eine Beschreibung ein. Diese wird den Benutzern angezeigt.
6. Wählen Sie unter **Art** eine Pauschalenart aus.
7. Wählen Sie die drei vertikalen Punkte und dann **Einrichtung**.
8. Wählen Sie unter **Buchungskontoart** die Option **Sachkonto** aus.
9. Wählen Sie unter **Buchungskonto Nr.** das Konto aus, auf das gebucht werden soll.

Der oben beschriebene allgemeine Vorgang kann auch für andere Pauschalarten wie Mahlzeiten, Fahrtkosten und Getränke verwendet werden, sofern diese auf der Seite **Expense Management Einrichtung** aktiviert wurden.
