# Pauschalsätze einrichten

Die korrekte Einrichtung von Pauschalsätzen ist von entscheidender Bedeutung, um genaue Erstattungsbeträge für alle beteiligten Parteien zu gewährleisten. In diesem Artikel wird beschrieben, wie Sie Pauschalsätze und -untersätze richtig konfigurieren.

Zuerst müssen Sie die erforderlichen Pauschalenfunktionen auf der Seite **Expense Management Einrichtung** aktivieren. Wenn diese Funktionen nicht aktiviert sind, werden sie auf den entsprechenden Seiten nicht angezeigt.

## So richten Sie Pauschalsätze ein

So definieren Sie einen Pauschalsatz für eine Pauschalgruppe:

1. Wählen Sie das Symbol {{search}}, geben Sie **Pauschalgruppen** ein, und wählen Sie den zugehörigen Link.
2. Wählen Sie eine vorhandene Pauschalgruppe aus oder wählen Sie **Neu** in der Aktionsleiste aus, um eine neue Pauschalgruppe mit dieser [Anleitung](@EM-85) zu erstellen.
3. Fügen Sie im Inforegister **Allgemein** unter **Buchungsgruppen** eine neue Zeile ein.
4. Füllen Sie bei Bedarf **Zielland/-Region** und **Zielname** aus.
5. Füllen Sie den **Übernachtungspauschale Code** und den **Verpflegungspauschale Code** aus.
6. Wählen Sie **Pauschalsätze** aus.
7. Wählen Sie im Dialogfeld **Ja** aus, um einen neuen Satz zu erstellen.
8. Füllen Sie die Felder in den Inforegistern nach Bedarf aus.

Auf derselben Seite haben Sie im Inforegister **Mahlzeiten** die Möglichkeit, bei Bedarf Untersätze anzugeben. Dies wird im Folgenden beschrieben.

## So richten Sie Untersätze ein

Mit Hilfe von Untersätzen können Sie detaillierte Pauschalen für bestimmte Komponenten oder Szenarien festlegen.

So richten Sie Untersätze für einen Pauschalsatz ein:

1. Wählen Sie im Inforegister **Mahlzeiten** neben **Berechnungsmethode** die Option **Unter-Sätze** aus.
2. Füllen Sie in der Tabelle **Pauschalsatz Details** die Felder nach Bedarf aus.
