# Belege einrichten

Expense Management bietet eine Fülle von Optionen, mit denen Sie Ihre Expense-Einrichtung genau an die Bedürfnisse Ihres Unternehmens anpassen können. Sie können all dies während der ersten Implementierung einrichten und dann später alle notwendigen Änderungen vornehmen, um Ihre Geschäftsprozesse anzupassen.
