# Unternehmensrichtlinien für Belege einrichten

Durch das Einrichten von Unternehmensrichtlinien für Belege können Sie bestimmte Aktionen basierend auf vordefinierten Kriterien, wie z. B. Schwellenwerten von Beträgen oder Dokumenttypen, automatisieren. Diese Richtlinien können die Genehmigung von Transaktionen mit geringem Wert automatisieren, Dokumente mit höherem Wert mit Warnungen kennzeichnen oder erstattungsfähige Beträge für bestimmte Belegkategorien begrenzen. Auf diese Weise können geringe Ausgaben effizient verarbeitet werden, während höhere Beträge oder außergewöhnliche Fälle zur Überprüfung markiert werden. Dies gewährleistet Konsistenz und Kontrolle über das Ausgabenmanagement.

Mit Expense Management können Sie Unternehmensrichtlinien definieren, sodass bestimmte sich wiederholende Arbeitsabläufe automatisch ausgelöst werden. Beispielsweise ist es möglich, diese Aktionen auszulösen:

| Aktion                                                    | Beschreibung                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| --------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Dokumente unterhalb des Grenzwerts automatisch genehmigen | Belege unter einem bestimmten Grenzwert werden automatisch genehmigt. Dies kann nützlich sein, wenn Sie bei der Bearbeitung von Belegen mit sehr geringen Beträgen zusätzlichen Genehmigungsaufwand vermeiden möchten. Sie legen einfach ein festes Limit für einen bestimmten Zeitraum fest und alles wird automatisch genehmigt.                                                                        |
| Dokument bis zum Grenzwert zurückerstatten                | Diese Regel legt eine Obergrenze für den erstattungsfähigen Betrag für eine bestimmte Belegart fest. Bei Fahrstrecken ist dadurch eine Kilometerrückerstattung nur bis zu dieser Grenze möglich, wobei die Ausgaben in einen erstattungsfähigen und einen nicht erstattungsfähigen Teil aufgeteilt werden.<br><br>Bei Überschreitung des Limits wird dem Genehmiger und dem Administrator dies angezeigt. |
| Bei Überschreiten des Grenzwerts Warnhinweis hinzufügen   | Ausgaben, die eine bestimmte Grenze überschreiten, wird ein Warnkommentar hinzugefügt. Dadurch wird die Genehmigung oder Buchung von Belegen nicht verhindert. Der Genehmiger und der Administrator werden jedoch darüber informiert, dass das Limit überschritten wurde.                                                                                                                                 |

Unternehmensrichtlinien können für Benutzer oder Benutzergruppen unter [Fahrstreckenfahrzeugen](@EM-27) und auf der Seite **Belegarten** für einen bestimmten Zeitraum definiert werden.

## So richten Sie Unternehmensrichtlinien für Belege ein

Um Unternehmensrichtlinien für Belegarten einzurichten, führen Sie die folgenden Schritte aus:

1. Wählen Sie das Symbol {{search}}, geben Sie **Belegarten** ein und wählen Sie dann die Belegart aus, für die Sie eine neue Unternehmensrichtlinie erstellen möchten.
2. Wählen Sie in der Aktionsleiste **Unternehmensrichtlinien** aus.
   ![Company policies action](/images/EM/company-policies.png)
3. Füllen Sie die Felder nach Bedarf aus. Genehmigen Sie beispielsweise Ausgaben unter einem bestimmten Betrag automatisch für eine bestimmte Benutzergruppe. Das folgende Bild zeigt ein:
