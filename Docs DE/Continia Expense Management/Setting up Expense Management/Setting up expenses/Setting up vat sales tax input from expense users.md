# MwSt.-/Umsatzsteuereingabe für Expense-Benutzer einrichten

In einigen Bereichen müssen Expense-Benutzer zusätzliche Informationen zur Umsatzsteuer/Mehrwertsteuer eingeben, beispielsweise in Fällen, in denen für Ausgaben mehrere Umsatzsteuer-/Mehrwertsteuersätze gelten. Dies ist mithilfe des Felds **MwSt.-Betrag** in der Continia Expense App oder im Continia Expense Portal möglich. Das Feld muss jedoch zuerst hinzugefügt werden.

Wenn dieses Feld hinzugefügt wird, erkennen die Expense App und das Expense Portal automatisch die Umsatzsteuer-/Mehrwertsteuerbeträge und fügen sie diesem Feld hinzu. Bei mehreren Mehrwertsteuersätzen werden die Beträge automatisch auf verschiedene Zeilen aufgeteilt.

Um das Feld **MwSt.-Betrag** zur Expense App und zum Expense Portal hinzuzufügen, folgen Sie diesen Schritten:

1. Wählen Sie das Symbol {{search}}, geben Sie **Konfigurierte Felder** ein, und wählen Sie dann den zugehörigen Link.
2. Fügen Sie in der Spalte **Feldcode** das Feld **MwSt.-Betrag** hinzu, indem Sie es entweder eingeben oder aus der Liste auswählen.

Wenn Sie dieses Feld zur Expense App/zum Expense Portal hinzugefügt haben, wird den Belegkarten in Business Central auch ein entsprechendes Feld hinzugefügt.

> [!TIP]
> Sie können das Feld mit der Aktion **Nach oben** verschieben, um es für den Expense-Benutzer leichter zugänglich zu machen.
