# Unternehmensrichtlinien für Fahrstrecken einrichten

Sie können Unternehmensrichtlinien definieren, sodass bestimmte sich wiederholende Arbeitsabläufe automatisch ausgelöst werden. Beispielsweise ist es möglich, diese Aktionen auszulösen:

| Aktion                                                        | Beschreibung                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| ------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Dokumente unterhalb des Grenzwerts automatisch genehmigen** | Belege unter einem bestimmten Grenzwert werden automatisch genehmigt. Dies kann nützlich sein, wenn Sie bei der Bearbeitung von Belegen mit sehr geringen Beträgen zusätzlichen Genehmigungsaufwand vermeiden möchten. Sie legen einfach ein festes Limit für einen bestimmten Zeitraum fest und alles wird automatisch genehmigt.                                                                        |
| **Dokument bis zum Grenzwert zurückerstatten**                | Diese Regel legt eine Obergrenze für den erstattungsfähigen Betrag für eine bestimmte Belegart fest. Bei Fahrstrecken ist dadurch eine Kilometerrückerstattung nur bis zu dieser Grenze möglich, wobei die Ausgaben in einen erstattungsfähigen und einen nicht erstattungsfähigen Teil aufgeteilt werden.<br><br>Bei Überschreitung des Limits wird dem Genehmiger und dem Administrator dies angezeigt. |
| **Bei Überschreiten des Grenzwerts Warnhinweis hinzufügen**   | Ausgaben, die eine bestimmte Grenze überschreiten, wird ein Warnkommentar hinzugefügt. Dadurch wird die Genehmigung oder Buchung von Belegen nicht verhindert. Der Genehmiger und der Administrator werden jedoch darüber informiert, dass das Limit überschritten wurde.                                                                                                                                 |

Unternehmensrichtlinien können für Benutzer oder Benutzergruppen unter [Fahrstreckenfahrzeugen](@EM-27) und auf der Seite **Belegarten** für einen bestimmten Zeitraum definiert werden.

## Unternehmensrichtlinien für Fahrzeuge einrichten

Um Unternehmensrichtlinien für Fahrzeuge einzurichten, führen Sie die folgenden Schritte aus:

1. Wählen Sie das Symbol {{search}}, geben Sie **Fahrzeuge** ein und wählen Sie dann die Fahrzeuge aus, für die Sie eine neue Unternehmensrichtlinie erstellen möchten.
2. Wählen Sie in der Aktionsleiste **Unternehmensrichtlinien** aus.
3. Füllen Sie die Felder nach Bedarf aus.
