# Fahrstreckensätze einrichten

Bevor Fahrstreckenkosten erstattet werden können, müssen Sie Fahrstreckensätze und andere damit verbundene Einstellungen, wie etwa das Startdatum, festlegen.

Um einen Fahrstreckensatz für ein Fahrzeug zu definieren, gehen Sie wie folgt vor.

1. Wählen Sie das Symbol {{search}}, geben Sie **Fahrstreckensätze** ein, und wählen Sie dann den zugehörigen Link.
2. Wählen Sie in der Aktionsleiste **Neu** aus.
3. Geben Sie unter **Startdatum** das Startdatum für den neuen Fahrstreckensatz ein oder verwenden Sie den Kalender.
4. Wählen Sie unter **Fahrzeugcode** das Fahrzeug aus, für das der neue Fahrstreckensatz gelten soll.
5. Aktivieren Sie unter **Neues Fahrstrecken Jahr** das Kästchen, um die Berechnung der Fahrstrecken zu starten.
6. Legen Sie unter **Satz ID** die Satz-ID fest, damit den Benutzern die gesamten Fahrstreckenkosten für diesen Satz angezeigt werden.

In manchen Ländern und manchen Unternehmen kann sich der Fahrstreckensatz ab einer bestimmten Entfernung entweder aufgrund nationaler Vorschriften oder aufgrund der Unternehmensrichtlinien ändern. Dies kann auch berücksichtigt werden, wenn Sie Fahrstreckensätze in Expense Management einrichten.

Wenn Sie einen anderen Satz anwenden müssen, wenn die gesamte Fahrstrecken eine bestimmte Entfernung überschreitet, gehen Sie wie folgt vor:

1. Wählen Sie in der Aktionsleiste **Neu** aus.
2. Geben Sie unter **Startdatum** dasselbe Startdatum wie für den zuvor erstellten Fahrstreckensatz ein oder verwenden Sie den Kalender.
3. Wählen Sie unter **Fahrzeugcode** das gleiche Fahrzeug aus, das Sie für den ursprünglichen Fahrstreckensatz angegeben haben.
4. Geben Sie unter **Start** die Entfernung ein, ab der der Fahrstreckensatz gilt.
5. Legen Sie unter **Satz ID** die Satz-ID fest, damit den Benutzern die gesamten Fahrstreckenkosten für diesen Satz angezeigt werden.

> [!NOTE]
> Dies ist eine Fortsetzung der ursprünglichen Fahrstreckenberechnung. Achten Sie daher darauf, das Kontrollkästchen **Neues Fahrstrecken Jahr** nicht zu aktivieren, da dadurch eine neue Fahrstreckenberechnung mit einem anderen Satz beginnt.
