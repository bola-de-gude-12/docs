# Expense-Aufgaben weiterleiten

Die effiziente Verwaltung von Ausgaben ist für jedes Unternehmen von entscheidender Bedeutung und ein zentraler Aspekt dieses Prozesses ist die Gewährleistung eines reibungslosen Ablaufs. Weiterleitungen spielen eine wichtige Rolle, um die Einreichung von Spesen so reibungslos wie möglich zu gestalten. Damit wird es bestimmten Personen ermöglicht, Spesen im Namen anderer Benutzer einzureichen. Diese Flexibilität ist insbesondere bei Geschäftsreisen oder teambezogenen Ausgaben von Vorteil.

Die Funktion zur Benutzerweiterleitung ist in der Continia Expense App und im Continia Expense Portal verfügbar. Mit dieser Funktion können Sie die Speseneinreichung an andere Benutzer delegieren und gleichzeitig Ihre eigenen Belege aktualisieren und übermitteln. <br>

## So fügen Sie einen bevollmächtigten Benutzer hinzu

Durch das Hinzufügen eines bevollmächtigten bzw. stellvertretenden Benutzers können Sie Teammitgliedern die Befugnis erteilen, spesenbezogene Aufgaben für andere zu übernehmen. Dies trägt dazu bei, die Effizienz und Teamarbeit innerhalb Ihrer Organisation zu verbessern.

So fügen Sie einen bevollmächtigten Benutzer hinzu:

1. Wählen Sie das Symbol {{search}}, geben Sie **Stellvertretung** ein, und wählen Sie den zugehörigen Link.

2. Wählen Sie in der Aktionsleiste **Neu** aus und geben Sie die folgenden Informationen in die bereitgestellten Felder ein:

   | Feld                           | Beschreibung                                                                                                                                                                                                                                                                                                                                               |
   | ------------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
   | Inhaber Benutzer-ID            | Geben Sie die Benutzer-ID des Benutzers an, der seine Verantwortung für die Speseneinreichung delegiert.                                                                                                                                                                                                                                   |
   | Inhaber Benutzername           | Geben Sie den Namen des Benutzers an, der seine Verantwortung für die Speseneinreichung delegiert.                                                                                                                                                                                                                                         |
   | Weitergeleiteter Benutzer      | Geben Sie die Benutzer-ID der Person an, die stellvertretend für die Einreichung von Belegen verantwortlich ist.                                                                                                                                                                                                                           |
   | Bevollmächtigter Benutzer Name | Geben Sie den Namen der Person an, die stellvertretend für die Einreichung von Belegen verantwortlich ist.                                                                                                                                                                                                                                 |
   | Gültig von                     | Legen Sie das Startdatum des Vertretungszeitraums fest, in dem der stellvertretende Benutzer im Namen des Eigentümers auf Belege zugreifen, diese aktualisieren und einreichen kann. Wenn Sie keinen bestimmten Zeitraum angeben möchten, lassen Sie das Feld **Gültig ab** einfach leer.                                  |
   | Gültig bis                     | Legen Sie das Enddatum des Vertretungszeitraums fest. Während dieser Zeit kann der stellvertretende Benutzer im Namen des Eigentümers auf Belege zugreifen, diese aktualisieren und einreichen kann. Wenn Sie keinen bestimmten Zeitraum angeben möchten, lassen Sie das Feld **Gültig bis** einfach leer. |
   | Deaktiviert                    | Gibt an, ob die Weiterleitung aktiviert ist oder nicht. Eine Weiterleitung wird automatisch deaktiviert, wenn der Benutzer kein Expense Management-Benutzer ist.                                                                                                                                                           |

   Der delegierte Benutzer hat entweder in der App oder im Portal die Möglichkeit, sich als Besitzer des Belegs anzumelden und Ausgaben im Namen des Besitzers einzureichen. Der bevollmächtigte Benutzer hat Zugriff auf alle vom Eigentümer gespeicherten Belege.

## So leiten Sie einen Beleg weiter

Nach dem Hochladen Ihrer Belege können Sie die Spesenabrechnung delegieren. Mit dieser praktischen Funktion können Sie einer anderen Person die Berechtigung erteilen, diese Belege in Ihrem Namen zu verwalten. Dies kann beispielsweise bei einer Teamveranstaltung von Vorteil sein, wenn mehrere Ausgaben eingereicht und zugeordnet werden müssen. Indem Sie die Aufgabe beispielsweise an den Teamleiter delegieren, können Sie den Abrechnungsprozess optimieren und effizient zusammenarbeiten, um sicherzustellen, dass alle teambezogenen Spesen korrekt eingereicht werden.
