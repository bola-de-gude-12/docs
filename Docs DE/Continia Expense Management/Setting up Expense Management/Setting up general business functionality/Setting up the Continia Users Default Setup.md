# Continia Standard Benutzereinrichtung

In Expense Management sind standardmäßig mehrere Einstellungen zugewiesen. Sie können diese Einstellungen jedoch für einzelne Benutzer oder Benutzergruppen anpassen.

In der Continia Standard Benutzereinrichtung können Sie standardmäßige Fahrzeuge für Fahrstrecken, Pauschalgruppen und Erstattungsmethoden festlegen, die automatisch bei bestimmten Belegarten angewendet werden. Alle in der Continia Standard Benutzereinrichtung konfigurierten Einstellungen haben Vorrang vor den in der Fahrzeugliste und Pauschalgruppen festgelegten Standardeinstellungen.

**So konfigurieren Sie die Continia Standard Benutzereinrichtung:**

1. Suchen Sie {{search}} nach **Continia Standard-Benutzereinrichtung** und wählen Sie dann den entsprechenden Link.

2. Wählen Sie in der Aktionsleiste **Neu** aus.

3. Informationen zum Ausfüllen der Felder finden Sie in der folgenden Tabelle:

   | Feld                     | Beschreibung                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
   | ------------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
   | Einrichtungsart          | Wählen Sie, ob die Einrichtung für einen Benutzer oder eine Benutzergruppe gelten soll.                                                                                                                                                                                                                                                                                                                                                                                                                                      |
   | Code                     | Geben Sie den Code für die Einrichtung an. Dies kann je nach gewählter Einrichtungsart ein Benutzercode oder ein Benutzergruppencode sein.                                                                                                                                                                                                                                                                                                                                                                   |
   | Fahrzeugcode             | Geben Sie einen Fahrzeugcode an, um ihn als Standardkonfiguration für den Benutzer oder die Benutzergruppe festzulegen. <br />Bei Fahrzeugen gilt das Feld Standard-Continia-Benutzercode nur für Benutzer, die ein einzelnes Fahrzeug verwenden. Wenn Benutzer über mehrere Fahrzeuge verfügen, ist das Feld stattdessen in Fahrstrecken in der App verfügbar. In solchen Fällen ist der standardmäßige Fahrzeugcode nicht so relevant, da das Feld in der App Vorrang hat. |
   | Pauschal Gruppen Code    | Geben Sie einen Pauschalgruppencode als Standard für den Benutzer oder die Benutzergruppe an.                                                                                                                                                                                                                                                                                                                                                                                                                                |
   | Beleg Erstattung         | Geben Sie die Standardmethode zur Erstattung der Belege für den Benutzer oder die Benutzergruppe an. Die Optionen sind: **Intern** (Benutzer), **Extern** (Externe Lohnabrechnung) und **Beides**.                                                                                                                                                                                                                                                     |
   | Fahrstrecke Erstattung   | Geben Sie die Standardmethode zur Erstattung der Fahrstrecken für den Benutzer oder die Benutzergruppe an. Die Optionen sind: **Intern** (Benutzer), **Extern** (Externe Lohnabrechnung) und **Beides**.                                                                                                                                                                                                                                               |
   | Erstattung der Pauschale | Geben Sie die Standardmethode zur Erstattung der Pauschalen für den Benutzer oder die Benutzergruppe an. Die Optionen sind: **Intern** (Benutzer), **Extern** (Externe Lohnabrechnung) und **Beides**.                                                                                                                                                                                                                                                 |
