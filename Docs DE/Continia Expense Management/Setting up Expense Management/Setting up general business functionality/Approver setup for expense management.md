# Genehmiger für Expense Management einrichten

Zuerst müssen Sie Genehmiger einrichten, um Workflows zur Genehmigung von Belegen aktivieren zu können. Diese Einrichtung erfolgt auf der Seite Continia Benutzereinrichtung, auf der verschiedene Einstellungen für Genehmiger definiert sind, einschließlich des Genehmigungslimits und der Berechtigungen für die Bearbeitung genehmigter Belege. Durch die Konfiguration dieser Parameter können Sie Ihre Genehmigungsprozesse optimieren und sicherstellen, dass Belege ordnungsgemäß geprüft und autorisiert werden.

So fügen Sie einem Benutzer Genehmigungsberechtigungen hinzu:

1. Suchen Sie {{search}} nach **Continia Benutzereinrichtung**.

2. Klicken Sie in der Aktionsleiste auf **Neu**.

3. Verwenden Sie die folgenden Felder, um einem Benutzer Genehmigungsberechtigungen zuzuweisen:
   | Inforegister       | Feld                                     | Beschreibung                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
   | ------------------ | ---------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
   | Allgemein          | **Continia Benutzer-ID**                 | Wählen Sie den Benutzer aus. Alle Genehmiger müssen Business Central-Benutzer sein. Über die Lookup-Schaltfläche in diesem Feld können Sie auf eine Liste der Benutzer zugreifen.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
   | Allgemein          | **Name**                                 | Der Name des Benutzers wird automatisch ausgefüllt, wenn die entsprechende E-Mail-Adresse eingegeben wird, die für Willkommens-E-Mails und Benachrichtigungen verwendet wird.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
   | Allgemein          | **Anzahl der Genehmigungsweiterleitung** | Gibt die Anzahl der Benutzer an, an die die Genehmigung weitergeleitet werden kann.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
   | Allgemein          | **Registrierung erzwingen zulassen**     | Gibt an, ob der Benutzer die Registrierung eines Belegs erzwingen darf und dabei Fehler und Warnungen ignorieren kann, die zu einem Fehlschlagen der Registrierung führen würden.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
   | Allgemein          | **Genehmigungsadministrator**            | Aktivieren Sie diese Option, um dem Benutzer die Möglichkeit zu geben, die Genehmigung von Belegen zu erzwingen. Nach der Aktivierung kann der Benutzer einen Beleg erneut öffnen, bearbeiten und die Genehmigung erzwingen, um den standardmäßigen Genehmigungsablauf zu umgehen. Beachten Sie, dass durch die Aktivierung dieser Option auch automatisch die entsprechende Einstellung in der **Genehmigungsbenutzereinrichtung** aktiviert wird. Außerdem wird dem Benutzer hierdurch zusätzlich zu den Continia-spezifischen Berechtigungen eine Reihe standardmäßiger Genehmigungsadministratorberechtigungen gewährt. Die Berechtigungen werden unter [Expense Management-Berechtigungen](@EM-62) beschrieben. |
   | Web Genehmigungen  | **Genehmigungs-Client**                  | Wählen Sie den Client für die Genehmigung der Belege aus. Wenn das Continia Web Approval Portal ausgewählt ist, kann der Benutzer Dokumente mithilfe von Business Central oder dem Continia Web Approval Portal genehmigen.<br />Die Optionen sind:<ul><li>**Leer** – kein Genehmigungsclient </li><li>**Webclient** – die Genehmigungsanfrage wird innerhalb von Business Central verarbeitet</li><li>**Continia Web Approval Portal** – über dieses Portal haben Sie Zugriff auf alle Dokumente, deren Genehmigung von Continia Document Capture und Continia Expense Management aussteht.   </li></ul>                                                                                                                            |
   | Expense Management | **Kann genehmigte Dokumente bearbeiten** | Aktivieren Sie diese Option, wenn Sie möchten, dass der Benutzer Dokumente bearbeiten darf, die bereits genehmigt wurden. Benutzer, für die diese Option aktiviert ist, können keine Informationen mit Bezug auf den Ausgabenbetrag ändern. Um den Betrag eines genehmigten Belegs zu ändern, müssen Benutzer den regulären Genehmigungsprozess erneut durchlaufen.                                                                                                                                                                                                                                                                                                                                                                                  |
   | Expense Management | **Unbegrenzt**                           | Belege von einem Benutzer mit unbegrenztem Genehmigungslimit erhalten automatisch den genehmigten Status. Diese Option muss für mindestens einen Genehmiger aktiviert werden.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
   | Expense Management | **Betrag**                               | Dieses Feld gilt für Benutzer mit einem angegebenen Genehmigungslimit. Das bedeutet, dass sie Expense-Dokumente nur bis zu einem bestimmten Betrag genehmigen können. Wenn der Ausgabenbetrag eines Belegs dieses Limit überschreitet, wird er entweder abgelehnt oder an einen anderen Genehmiger mit einem höheren Genehmigungslimit oder unbegrenzter Genehmigungsberechtigung weitergeleitet.                                                                                                                                                                                                                                                                                                                                                    |

4. Der letzte Schritt bei der Konfiguration der Genehmiger ist der Export. Klicken Sie dazu in der Aktionsleiste auf **Benutzer exportieren**. Durch diese Aktion werden automatisch Willkommens-E-Mails generiert und an die neuen Genehmiger gesendet. Diese E-Mails enthalten wichtige Links, mit denen sie ihre Genehmigerrolle aktivieren, ein Kennwort erstellen und bequem auf die Expense Mobile App und das Expense Portal zugreifen können.

   > [!NOTE]
   > Sie können sich auch mit Ihren Microsoft 365-Anmeldeinformationen anmelden. In diesem Fall ist die Erstellung eines Passworts nicht erforderlich.
