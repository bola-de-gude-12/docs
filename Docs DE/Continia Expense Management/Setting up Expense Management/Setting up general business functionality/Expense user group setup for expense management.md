# Expense-Benutzergruppen für Expense Management einrichten

Mit Expense-Benutzergruppen können Sie auf einfache Weise mehreren Expense-Benutzern die gleichen Einstellungen zuweisen, statt diese Einstellungen für jeden einzelnen Expense-Benutzer festlegen zu müssen.

Nachdem Sie eine Expense-Benutzergruppe erstellt haben, können sie ihr Benutzer zuweisen, entweder beim Einrichten neuer Continia-Benutzer oder durch Bearbeiten vorhandener Continia-Benutzer.

## Expense-Benutzergruppen einrichten

Um eine Expense-Benutzergruppe für Expense Management einzurichten, folgen Sie diesen Schritten:

1. Wählen Sie das Symbol {{search}}, geben Sie **Expense Benutzergruppe** ein, und wählen Sie den zugehörigen Link.
2. Wählen Sie in der Aktionsleiste **Neu** aus.
3. Geben Sie unter **Code** den Namen der Gruppe ein.
4. Geben Sie unter **Beschreibung** eine Beschreibung der Gruppe ein.

Wenn Sie eine oder mehrere Expense-Benutzergruppen erstellt haben, können Sie auf der Seite **Continia Standard Benutzereinrichtung** die Einstellungen für diese Gruppen festlegen, z. B. **Fahrzeugcode** und Erstattungsmethode für die verschiedenen Ausgabenarten.

Um Einstellungen für eine Expense-Benutzergruppe zu definieren, befolgen Sie diese Schritte:

1. Wählen Sie das Symbol {{search}}, geben Sie **Continia Standard Benutzereinrichtung** ein, und wählen Sie den zugehörigen Link.
2. Wählen Sie unter **Einrichtungsart** die Option **Gruppe**.
3. Wählen Sie unter **Code** die Expense-Benutzergruppe aus, die Sie konfigurieren möchten.
4. Verwenden Sie das Dropdown-Menü in jedem der Felder **Fahrzeugcode**, **Pauschal Gruppen Code**, **Erstattung der Belege**, **Erstattung der Fahrstrecke** und **Erstattung der Pauschale**, um Ihre bevorzugten Einstellungen auszuwählen.
