# Erstattungsmethoden einrichten

Beim Konfigurieren einer Erstattungsmethode können Sie das System angeben, über das Expense-Benutzer ihre Erstattungen erhalten. Sie können die Erstattung innerhalb von Microsoft Dynamics 365 Business Central oder über externe Lohn- und Gehaltsabrechnungssysteme abwickeln. Durch Auswahl der folgenden Erstattungsmethoden wird sichergestellt, dass die entsprechenden Konten basierend auf Ihrem ausgewählten Ausgabesystem ausgefüllt werden.

Folgende Erstattungsmethoden stehen zur Verfügung:

| Methode                                    | Beschreibung                                                                                                                                                                                                                                                                                                       |
| ------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Kreditor (Benutzer)** | Die Rückerstattung an den Benutzer erfolgt direkt in Business Central durch eine Zahlung auf das Konto des Benutzers: Kreditor oder Mitarbeiter.                                                                                                                                   |
| **Externes System**                        | Die Rückerstattung an den Benutzer erfolgt über ein externes Gehaltsabrechnungssystem. Wenn eine Integration mit diesem Lohn- und Gehaltssystem besteht, wird das Konto vor der Buchung auch validiert. Standardmäßig können Sie Ausgaben mit Projekten verbuchen. |
| **Beides**                                 | Die erstattungsfähigen Beträge werden in Business Central gebucht, erscheinen jedoch weiterhin als nicht erstattet, d. h. Sie müssen sie manuell als erstattet markieren.                                                                                          |
