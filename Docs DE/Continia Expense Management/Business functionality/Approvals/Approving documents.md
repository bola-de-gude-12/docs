# Dokumente genehmigen

Wenn ein Dokument genehmigt wurde, ändert sich sein Status in Freigegeben. Mit diesem Status kann das Dokument gebucht werden, sodass dem Benutzer die entsprechenden Forderungen erstattet werden können.

Verwenden Sie für Ihren Dokumentgenehmigungsprozess die entsprechenden Optionen in den Abschnitten zum Genehmigen und zum Anfordern von Genehmigungen.

Der Abschnitt Genehmigen enthält die folgenden Optionen: Genehmigen, Genehmigung erzwingen, Ablehnen, Weiterleiten und auf Abwarten setzen. Der Abschnitt Genehmigung anfordern enthält die Optionen Genehmigen, Genehmigung anfordern, Genehmigungsanfrage senden und Genehmigungsanforderung stornieren sowie einen Link zur Seite Genehmigungen.

So genehmigen Sie ein Dokument:

1. Suchen Sie {{search}} nach **Genehmigungseinträge - Expense Management**.
2. Wählen Sie das entsprechende zu genehmigende Dokument aus.
3. Klicken Sie in der Aktionsleiste auf **Genehmigen**.

## Was Sie beachten müssen

Nachdem ein Dokument genehmigt wurde, kann es nur noch von einem Administrator aktualisiert werden.

Für den Administrator muss die Option **Kann genehmigte Dokumente bearbeiten** aktiviert sein, siehe [Genehmiger für Expense Management einrichten](@EM-33). Andernfalls muss das Dokument erneut geöffnet und erneut zur Genehmigung gesendet werden. Weitere Informationen zum erneuten Öffnen verschiedener Dokumenttypen finden Sie unter:

- [Belege wieder öffnen](@EM-17)
- [Fahrstrecken wieder öffnen](@EM-18)
- [Pauschalen wieder öffnen](@EM-19)
