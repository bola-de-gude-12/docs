# Überblick über Belege

Sobald Ihre Belege in Expense Management importiert wurden, können Sie sofort damit arbeiten. Alle Ihre Belege finden Sie in der Belegliste.

Außerdem können Sie Belege entsprechend Ihrem Unternehmensablauf weiterverarbeiten. Weitere Informationen zu den Hauptprozessen von Continia Expenses finden Sie in der folgenden Tabelle.

| **Thema**                                                                                                                                                  | **Siehe**                                                                               |
| ---------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------- |
| Einreichen von Belegen mit der Continia Expense App, dem Continia Expense Portal, Business Central oder per E-Mail                                         | [Belege einreichen](@EM-5)                                                              |
| Manuelles Verschieben eines Belegs zu einem anderen Mandanten                                                                                              | [Belege in andere Mandanten verschieben](@EM-20)                                        |
| Genehmigen von Dokumenten                                                                                                                                  | [Dokumente genehmigen](@EM-3)                                                           |
| Ablehnen von Dokumenten, um sie in der Datenbank zu behalten (anstatt sie vollständig zu löschen) und Angeben von Ablehnungsbemerkungen | [Dokumente ablehnen](@EM-4)                                                             |
| Erneutes Öffnen von Belegen mit unterschiedlichem Status, um sie beispielsweise erneut abzugleichen oder Fehler zu korrigieren                             | [Belege wieder öffnen](@EM-17)                                                          |
| Buchen einzelner oder mehrerer Belege in einem Stapel                                                                                                      | [Belege buchen](@EM-11)                                                                 |
| Kostenerstattung für Expense-Benutzer oder Expense-Benutzergruppen und Anwendung von Filtern, um die richtigen Expense-Benutzer zu finden                  | [Belege erstatten](@EM-14)                                                              |
| Erstellen eines Kreditors, damit Sie Belege, die bereits mit einer Firmenkreditkarte bezahlt wurden, auf diesen Kreditor buchen können                     | [Einen Kreditor hinzufügen](@EM-45)                                                     |
| Manuelles Herunterladen von Belegen von Continia Online, damit Sie mit deren Bearbeitung beginnen können                                                   | [Belege manuell von Continia Online herunterladen](@EM-23)                              |
| Exportieren von Beleganhängen                                                                                                                              | [Belege buchen: Anhänge exportieren](@EM-11#so-exportieren-sie-anhange) |
| Vermeiden doppelter Buchungen, wenn Sie sowohl Document Capture als auch Expense Management verwenden                                                      | [Einkaufsrechnungen in Expense Management und Document Capture verarbeiten](@EM-277)    |
