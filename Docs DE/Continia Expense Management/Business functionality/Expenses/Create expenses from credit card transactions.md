# Belege aus Kreditkartentransaktionen erstellen

Die Buchung auf einen Geschäftskreditor ist z. B. dann relevant, wenn Sie eine Firmenkreditkarte mit einem bereits im System vorhandenen Kreditor verwenden und die Transaktion diesem Kreditor im Buchungsposten zugeordnet werden soll. Die Funktionalität wird durch das Hinzufügen einer Kreditorennummer zu einem Beleg aktiviert.

So fügen Sie einen Kreditor hinzu:

1. Wählen Sie das Symbol {{search}}, geben Sie **Belege** ein, und wählen Sie dann den zugehörigen Link.
2. Wählen Sie in der Aktionsleiste **Bearbeiten** aus. Ein Kreditor kann einem Beleg unabhängig vom Status des Belegs hinzugefügt werden.
3. Wählen Sie im Inforegister **Buchung** die Option **Mehr anzeigen** aus.
4. Füllen Sie das Feld **Kreditorennr.** aus.

Der Beleg wird nun auf diesen Kreditor gebucht. Da der Beleg bereits über die Kreditkarte bezahlt wurde, entsteht beim Kreditor automatisch eine ausgeglichene Buchung. Dadurch wird sichergestellt, dass der Beleg beim Kreditor nicht als offen angezeigt wird.
