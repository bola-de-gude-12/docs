# Überblick über Pauschalen

Nachdem Sie Pauschalen in Expense Management importiert haben, können Sie sofort damit arbeiten. Sie finden alle Pauschalen in der entsprechenden Liste. Um die Überprüfung zu vereinfachen und die Berechnung vollständig nachzuvollziehen, wird in einer Infobox die Aufschlüsselung des Pauschalenbetrages angezeigt.

Abhängig von Ihren Unternehmensabläufen können Sie die Dokumente weiterverarbeiten. Weitere Informationen zu den wichtigsten Prozessen finden Sie in den folgenden Artikeln.

| **Thema**                                                    | **Siehe**                                                    |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| Einreichen von Pauschalen mit der Continia Expense App, dem Continia Expense Portal oder Business Central oder per E-Mail. | [Pauschalen einreichen](@EM-7)                               |
| Buchen einzelner oder mehrerer Pauschalen in einem Stapel.   | [Pauschalen buchen](@EM-13)                                  |
| Manuelles Verschieben von Pauschalen in einen anderen Mandanten. | [Pauschalen in einen anderen Mandanten verschieben](@EM-22)  |
| Genehmigen von Dokumenten.                                   | [Dokumente genehmigen](@EM-3)                                |
| Ablehnen von Dokumenten, um sie in der Datenbank zu behalten (anstatt sie vollständig zu löschen) und Angeben von Ablehnungsbemerkungen. | [Dokumente ablehnen](@EM-4)                                  |
| Erneutes Öffnen von Pauschalen mit unterschiedlichem Status, um sie beispielsweise erneut abzugleichen oder Fehler zu korrigieren. | [Pauschalen wieder öffnen](@EM-19)                           |
| Manueller Download von Pauschalen von Continia Online.       | [Pauschalen manuell von Continia Online herunterladen](@EM-25) |
| Kostenerstattung für Expense-Benutzer oder Expense-Benutzergruppen und Anwendung von Filtern, um die richtigen Benutzer zu finden. | [Pauschalen erstatten](@EM-16)                               |

| Anpassen von Pauschalen.                                     | [Aktivieren oder Deaktivieren von Zulagen für Pauschalen](@EM-96) |
