# Mehrere Ziele für Pauschalen aktivieren

In vielen Unternehmen sind Reisen mit mehreren Etappen nicht ungewöhnlich. Beispielsweise kann ein Berater verschiedene Kunden an verschiedenen Orten mit unterschiedlichen Sätzen besuchen. Durch die Aktivierung mehrerer Reiseziele für die Pauschale wird die Verwaltung der Sätze für derartige Reisen wesentlich einfacher.

Mit dieser Funktion können Sie für jeden Tagessatz ein oder mehrere Ziele angeben. Der anwendbare Satz wird auf Grundlage des zuletzt besuchten Reiseziels berechnet. Für die Rückreise gelten die örtlichen Sätze.

So aktivieren Sie mehrere Ziele für Pauschalen:

1. Wählen Sie das Symbol {{search}}, geben Sie **Expense Management Einrichtung** ein und wählen Sie dann den zugehörigen Link.
2. Wählen Sie im Inforegister **Pauschale** die Option **Mehrere Reiseziele aktivieren** aus.

> [!IMPORTANT]
> Denken Sie daran, [Pauschalsätze](@EM-86) für alle Ziele einzurichten.
