# Pauschalen anpassen

Sie können die Pauschalen individuell anpassen. Wenn beispielsweise eine Unterkunft nicht genutzt wird, kann sie deaktiviert werden, um nur die relevanten Pauschalendetails anzuzeigen.

## So passen Sie Pauschalen an

So aktivieren oder deaktivieren Sie die Optionen für Pauschalen:

1. Wählen Sie das Symbol {{search}}, geben Sie **Expense Management Einrichtung** ein und wählen Sie dann den zugehörigen Link.
2. Aktivieren oder deaktivieren Sie im Inforegister **Pauschale** unter **Pauschale aktivieren** die entsprechenden Optionen.

> [!IMPORTANT]
> Denken Sie daran, die Sätze für alle verwendeten Pauschaloptionen anzugeben.
