# Steuerpflichtige Sätze für Pauschalen aktivieren

In diesem Artikel erfahren Sie, wie Sie steuerpflichtige Sätze für Pauschalen aktivieren und steuerpflichtige Beträge für Pauschalsätze angeben, falls Ihr Unternehmen seinen Mitarbeitern mehr als die vom Staat festgelegten steuerfreien Sätze erstatten möchte.

## Steuerpflichtige Sätze für Pauschalen aktivieren

So aktivieren Sie steuerpflichtige Sätze für Pauschalen:

1. Wählen Sie das Symbol {{search}}, geben Sie **Expense Management Einrichtung** ein und wählen Sie dann den zugehörigen Link.
2. Wählen Sie im Inforegister **Fahrstrecke** die Option **Steuerpflichtige Sätze aktivieren** aus.

### Steuerpflichtige Beträge für Pauschalsätze angeben

Um steuerpflichtige Sätze anwenden zu können, müssen Sie auch steuerpflichtige Beträge für Pauschalsätze angeben. Und so geht's:

1. Wählen Sie das Symbol {{search}}, geben Sie **Pauschalsätze** ein, und wählen Sie den zugehörigen Link.
2. Wählen Sie in der Spalte **Pauschal Gruppen Code** den Satz aus, für den Sie steuerpflichtige Beträge angeben möchten.
3. Geben Sie im Inforegister **Übernachtung** im Feld **Steuerpflichtige Übernachtungspauschale** einen steuerpflichtigen Betrag an.
4. Geben Sie im Inforegister **Mahlzeiten** im Feld **Steuerpflichtige Verpflegungspauschale** einen steuerpflichtigen Betrag an.

   > [!NOTE]
   > Die Felder **Steuerpflichtige Übernachtungspauschale** und **Steuerpflichtige Verpflegungspauschale** müssen möglicherweise zuerst hinzugefügt werden, damit sie sichtbar sind. Wählen Sie dazu das Symbol {{settings}} > **Entwurf** > **+ Feld**, um den Bereich **Feld der Seite hinzufügen** zu öffnen. Markieren Sie den entsprechenden Zeilenabschnitt und ziehen Sie anschließend das gewünschte Feld aus der Leiste in die Tabelle im ausgewählten Zeilenabschnitt. Wählen Sie **Gestalten einstellen** aus und schließen Sie dann den Bereich **Design fertig stellen**.
