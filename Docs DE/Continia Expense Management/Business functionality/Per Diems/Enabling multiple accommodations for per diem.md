# Mehrere Unterkünfte für Pauschalen aktivieren

Durch die Aktivierung mehrerer Unterkunftsarten für eine Pauschale haben Benutzer die Flexibilität, für jeden Reisetag unterschiedliche Unterkunftsoptionen auszuwählen. Mit dieser Funktion können Mitarbeiter ihre Unterbringung an ihre Bedürfnisse anpassen und so für mehr Komfort und Bequemlichkeit auf Geschäftsreisen sorgen.

So aktivieren Sie mehrere Unterkünfte für Pauschalen:

1. Wählen Sie das Symbol {{search}}, geben Sie **Expense Management Einrichtung** ein und wählen Sie dann den zugehörigen Link.
2. Wählen Sie im Inforegister **Pauschale** die Option **Mehrere Unterkunftsarten aktivieren** aus.
