# Das Einkaufsvertragsarchiv anzeigen

Alle Statusänderungen von Einkaufsverträgen werden im Einkaufsvertragsarchiv protokolliert. Beispielsweise wird automatisch ein neuer Protokolleintrag erstellt, wenn ein Vertrag zur Überprüfung gesendet wird, wenn eine Überprüfung eingereicht, abgebrochen oder abgeschlossen wird oder wenn sich der jährliche Betrag eines Vertrags ändert.

Sie können jederzeit bequem auf das Archiv zugreifen, wenn Sie es für statistische oder andere Zwecke benötigen.

So greifen Sie auf das Einkaufsvertragsarchiv zu:

1. Wählen Sie das Symbol {{search}}, geben Sie **Einkaufsverträge** ein, und wählen Sie dann den zugehörigen Link, um die Seite **Einkaufsverträge** zu öffnen.
2. Wählen Sie in der Vertragsliste den Vertrag aus, dessen Historie Sie anzeigen möchten.
3. Wählen Sie in der Aktionsleiste **Vertrag** > **Archivhistorie**.

Dadurch wird die Seite **Archivierte Einkaufsverträge** geöffnet, die Ihnen einen vollständigen Überblick über die Historie des ausgewählten Vertrags bietet. Jede Statusänderung des Vertrages wird in der Übersichtstabelle durch eine Versionszeile repräsentiert und jede dieser Zeilen ist mit einer Versionsnummer versehen. Wenn Sie weitere Details zu einer bestimmten Version des Vertrags anzeigen möchten, wählen Sie die Zeile dieser Version in der Tabelle aus und wählen Sie dann die Nummer in der Spalte **Nr.** aus, um eine neue Seite mit allen Details der ausgewählten Version zu öffnen.
