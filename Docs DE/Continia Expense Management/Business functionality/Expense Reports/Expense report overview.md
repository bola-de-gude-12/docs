# Überblick über Abrechnungen

Bevor Sie mit Abrechnungen arbeiten können, müssen Sie zuerst die Regeln und Werte konfigurieren, damit sie Ihren Unternehmensrichtlinien entsprechen. Sie können diese während der Erstimplementierung einrichten und später Änderungen vornehmen, um sie an die Anforderungen Ihres Unternehmens anzupassen.

| Thema                                                                                                                                                           | Siehe                                                        |
| --------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------ |
| Aktivieren oder Deaktivieren von Abrechnungen, Konfigurieren von Abrechnungseinstellungen und Anzeigen aller Dokumentenkommentare einer Abrechnung an einem Ort | [Abrechnungen einrichten](@EM-306)                           |
| Erstellen und Übermitteln von Abrechnungen und Überprüfen von Warnungen über das Expense Portal oder in Business Central                                        | [Abrechnungen mit dem Browser verwalten](@EM-294)            |
| Erstellen und Übermitteln von Abrechnungen und Überprüfen von Warnungen über die Expense Mobile App                                                             | [Abrechnungen mit der Expense Mobile App verwalten](@EM-326) |
| Definieren von Abläufen und Regeln für die Vorabgenehmigung von Abrechnungen                                                                                    | [Vorabgenehmigung von Abrechnungen einrichten](@EM-272)      |
| Anzeigen von Kommentaren aus mehreren Dokumenten einer Abrechnung an einem Ort                                                                                  | [Mehrere Kommentare in Abrechnungen anzeigen](@EM-301)       |
