# Überblick über Fahrstrecken

Sobald Ihre Fahrstreckenposten in Expense Management importiert wurden, können Sie sofort damit arbeiten. Alle Fahrstreckenposten finden Sie in der Liste der Fahrstrecken.

In den folgenden Artikeln finden Sie Einführungen in allgemeine Konzepte, die Ihnen den Einstieg in die Verarbeitung von Fahrstrecken erleichtern.

| **Thema**                                                                                                                                                  | **Siehe**                                                     |
| ---------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------- |
| Einreichen von Fahrstrecken mit der Continia Expense App, dem Continia Expense Portal oder Business Central oder per E-Mail                                | [Fahrstrecken einreichen](@EM-6)                              |
| Manuelles Verschieben von Fahrstrecken in einen anderen Mandanten                                                                                          | [Fahrstrecken in einen anderen Mandanten verschieben](@EM-21) |
| Genehmigen von Dokumenten                                                                                                                                  | [Dokumente genehmigen](@EM-3)                                 |
| Ablehnen von Dokumenten, um sie in der Datenbank zu behalten (anstatt sie vollständig zu löschen) und Angeben von Ablehnungsbemerkungen | [Dokumente ablehnen](@EM-4)                                   |
| Erneutes Öffnen von Fahrstreckenabrechnungen mit unterschiedlichem Status, um sie beispielsweise erneut abzugleichen oder Fehler zu korrigieren            | [Fahrstrecken wieder öffnen](@EM-18)                          |
| Buchen einzelner oder mehrerer Fahrstrecken in einem Stapel                                                                                                | [Fahrstrecken buchen](@EM-12)                                 |
| Kostenerstattung für Expense-Benutzer oder Expense-Benutzergruppen und Anwendung von Filtern, um die richtigen Benutzer zu finden                          | [Fahrstrecken erstatten](@EM-15)                              |
| Manuelles Herunterladen von Fahrtkosten von Continia Online, damit Sie mit deren Bearbeitung beginnen können                                               | [Belege manuell von Continia Online herunterladen](@EM-24)    |
| Aktivieren steuerpflichtiger Sätze für Fahrstrecken, um Mitarbeitern mehr als die staatlichen Sätze zu erstatten                                           | [Steuerpflichtige Sätze für Fahrstrecken aktivieren](@EM-93)  |
