# Steuerpflichtige Sätze für Fahrstrecken aktivieren

In diesem Artikel erfahren Sie, wie Sie steuerpflichtige Sätze für Fahrstrecken aktivieren und steuerpflichtige Beträge für Fahrstreckensätze angeben, falls Ihr Unternehmen seinen Mitarbeitern mehr als die vom Staat festgelegten steuerfreien Sätze erstatten möchte.

## Steuerpflichtige Sätze für Fahrstrecken aktivieren

So aktivieren Sie steuerpflichtige Sätze für Fahrstrecken:

1. Wählen Sie das Symbol {{search}}, geben Sie **Expense Management Einrichtung** ein und wählen Sie dann den zugehörigen Link.
2. Wählen Sie im Inforegister **Fahrstrecken** die Option **Steuerpflichtige Sätze aktivieren** aus.

### Steuerpflichtige Beträge für Fahrstreckensätze angeben

Um steuerpflichtige Sätze anwenden zu können, müssen Sie auch steuerpflichtige Beträge für Fahrstreckensätze angeben. Und so geht's:

1. Wählen Sie das Symbol {{search}}, geben Sie **Fahrstreckensätze** ein, und wählen Sie dann den zugehörigen Link.
2. Wählen Sie in der Aktionsleiste **Liste bearbeiten** aus.
3. Geben Sie in der Spalte **Steuerpflichtiger Satz (MW)** den steuerpflichtigen Satz an.
