# Bilder der gespeicherten Anhänge in der Continia Expense App

Eine der Kernfunktionen der Continia Expense App ist die Möglichkeit, Fotos von Quittungen aufzunehmen. In diesem Artikel finden Sie Informationen zur Verwendung der Kamera und zum Anzeigen gespeicherter Bilder in der Expense-App. Darüber hinaus finden Sie Informationen zum Teilen von PDF-Rechnungen und Quittungen aus Apps oder Speicherorten auf Ihrem Telefon mit der Expense App, die Sie dann wie Bilder für Belege usw. verwenden können.

Alle Bilder, die Sie der Liste der gespeicherten Anhänge hinzufügen, werden automatisch mit dem Expense Portal synchronisiert und umgekehrt.

## Kamera öffnen

Um die Kamera in der Continia Expense App zu öffnen und ein Foto aufzunehmen, gehen Sie folgendermaßen vor:

1. Öffnen Sie die Expense App auf Ihrem Gerät.
2. Wählen Sie **Erstellen**.

Oder:

1. Wählen Sie das Symbol der Expense App und halten Sie es gedrückt, bis das Menü der Expense App angezeigt wird.
2. Wählen Sie **Bild**.

## Gespeicherte Bilder anzeigen

Wenn Sie Ihre gespeicherten Bilder in der Expense App anzeigen möchten, gehen Sie wie folgt vor:

1. Öffnen Sie die Expense App auf Ihrem Gerät.
2. Wählen Sie **Offen** und dann erneut **Offen**.

Dadurch gelangen Sie zur Liste der Anhänge. Von hier aus können Sie auch auf die Kamera (Kamerasymbol) und die Kameraaufnahmen (Büroklammersymbol) zugreifen, wenn Sie weitere Bilder hinzufügen möchten.

## PDFs mit der Expense App teilen

Sie können PDF-Rechnungen oder Quittungen aus Apps oder Speicherorten auf Ihrem Telefon – wie etwa E-Mail-Apps, OneDrive, Uber usw. – direkt an die Expense App übertragen. Verwenden Sie einfach die Funktion zum Teilen in der App oder im Speicherort und senden Sie sie direkt an die Expense App. Die PDF wird in der Liste der Anhänge gespeichert und Sie können sie dann wie gespeicherte Bilder anzeigen, wie oben beschrieben. Sie können die PDF auch einem neuen oder bestehenden Beleg, einer Fahrstrecke oder einer Abrechnung hinzufügen, so wie dies mit gespeicherten Bildern möglich ist. <br>

## Bilder mit einem delegierten Benutzer teilen

Als Expense-Benutzer können Sie Dokumentenbilder, beispielsweise von Anhängen, mit einem delegierten Benutzer teilen, damit dieser in Ihrem Namen Spesen einreichen kann. Beispielsweise kann ein Manager seinen Assistenten damit beauftragen, die Bilder seiner Anhänge und anderer Spesenbelege für ihn zu bearbeiten und einzureichen. In diesem Fall sind alle in der Mobile App und im Expense Portal des Managers gespeicherten Belege für den Assistenten sichtbar. Weitere Informationen zur Verwendung dieser Funktion finden Sie unter [Expense-Aufgaben weiterleiten](@EM-166).
