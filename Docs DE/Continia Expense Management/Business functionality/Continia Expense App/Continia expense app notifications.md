# Continia Expense App-Benachrichtigungen

Die automatischen Benachrichtigungen in der Continia Expense App sorgen dafür, dass die Nutzer der App immer über wichtige Änderungen des Dokumentenstatus informiert sind.

Diese Funktion ist von Anfang an aktiviert, wenn Sie bei der Installation der Continia Expense App die Aufforderung zur Berechtigung von Push-Benachrichtigungen akzeptieren. Sie müssen dann nichts einrichten. Sie können diese Benachrichtigungen jederzeit in den Einstellungen der Continia Expense App auf Ihrem Telefon deaktivieren.

Für alle Expense-Dokumente (das heißt Belege, Fahrstrecken, Pauschalen und Abrechnungen) werden Benachrichtigungen mit der Betragsanzeige in der lokalen Währung angezeigt.

Sie erhalten keine Benachrichtigung, wenn Ihre Continia Expense App zum Zeitpunkt einer Änderung im Vordergrund ist. In dem unwahrscheinlichen Fall, dass sich die Continia Expense App zum Zeitpunkt einer Statusänderung für eines oder mehrere Ihrer Expense-Dokumente im Vordergrund befindet, erhalten Sie eine In-App-Benachrichtigung mit der Aufforderung, eine Synchronisierung mit Continia Online durchzuführen, damit alle Änderungen angezeigt werden können.

Nachfolgend finden Sie eine Übersicht darüber, wann Benachrichtigungen an Expense-Benutzer gesendet werden.

| Expense-Benutzer erhalten eine Benachrichtigung in folgenden Fällen: |
| ------------------------------------------------------------------------------------ |
| Ein Beleg/eine Fahrtstrecke/eine Pauschale/eine Abrechnung wird **erstellt**         |
| Ein Beleg/eine Fahrtstrecke/eine Pauschale/eine Abrechnung wird **aktualisiert**     |
| Ein Beleg/eine Fahrtstrecke/eine Pauschale/eine Abrechnung wird **genehmigt**        |
| Ein Beleg/eine Fahrtstrecke/eine Pauschale/eine Abrechnung wurde **abgeschlossen**   |
| Die Vorabgenehmigung einer Abrechnung wurde **genehmigt**                            |
| Die Vorabgenehmigung einer Abrechnung wurde **abgelehnt**                            |
