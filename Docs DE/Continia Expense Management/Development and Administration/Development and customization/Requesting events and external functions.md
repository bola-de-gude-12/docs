# Events und externe Funktionen anfordern

Continia Expense Management bietet Anpassungsmöglichkeiten durch Event Publisher und den Aufruf externer Funktionen. Die meisten Funktionen sind zwar intern, können aber auf Wunsch auch extern angeboten werden. Benutzer können zusätzliche Events oder externe Funktionen anfordern, die, wenn sie akzeptiert werden, in zukünftigen Service Packs enthalten sein werden, normalerweise innerhalb von zwei Monaten. Dieser Ansatz stellt sicher, dass Cloud-Updates problemlos durchgeführt werden können und Rücksicht auf die Kompatibilität und Stabilität von Business Central im Hinblick auf kritische Änderungen genommen wird, um Flexibilität und schnellere App-Updates zu ermöglichen.

## So fordern Sie ein neues Event an

Bevor Sie sich bezüglich neuer Events an das Continia-Supportteam wenden, prüfen Sie bitte in der Liste der veröffentlichten Events, ob es ein Event gibt, das Ihren Anforderungen entspricht. Wenn es ein passendes Event gibt, können Sie dieses Event nutzen, ohne Unterstützung anfordern zu müssen.

Wenn das von Ihnen benötigte Event jedoch nicht aufgelistet ist, wenden Sie sich bitte [an uns](/Continia Expense Management/Getting Started/Help and Support.md) und geben Sie uns die folgenden Informationen für unsere Beurteilung:

- Name des Events
- Zweck des Events
- Die Position im Code, an der das Event ausgelöst werden soll (vorzugsweise mit einem Screenshot oder einem Codeausschnitt)
- Alle relevanten Event Publisher-Parameter
- Parameterbeschreibung

> [!NOTE]
>
> Für jeden neuen Event Publisher, der erstellt und zu Expense Management hinzugefügt wird, verwendet Continia die gleiche Namensstruktur wie Microsoft.

## So fordern Sie eine externe Funktion an

Bevor Sie die externe Bereitstellung einer Funktion anfordern, stellen Sie sicher, dass die Funktion derzeit intern ist. Laden Sie dazu das neueste Service Pack herunter und kompilieren Sie Ihre Expense Management-Erweiterung damit. Wenn bestätigt wird, dass die Funktion nicht verfügbar ist, bitten Sie Ihren Partner, ein Zendesk-Ticket zu erstellen.

So fordern Sie die externe Bereitstellung einer Funktion in Expense Management an:

- Bitten Sie Ihren Partner, ein Zendesk-Ticket mit den folgenden Informationen zu erstellen:
  - Name, Objekttyp und Nummer der Funktion.
  - Der Grund, warum die Funktion extern verfügbar gemacht werden soll.

Continia prüft Ihre Anfrage. Bei Genehmigung wird die Funktion in einem kommenden Service Pack für Expense Management extern bereitgestellt.
