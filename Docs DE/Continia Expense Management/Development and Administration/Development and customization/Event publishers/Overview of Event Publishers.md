# Überblick über Event Publisher

Sie können die Funktionalität von Continia Expense Management mithilfe der vielen verschiedenen Event Publisher (Ereignisherausgeber) anpassen, die Continia bereitstellt.

Ab dem Release von Expense Management 2025 R1 (26.00) wird die Liste der Event Publisher nicht mehr aktualisiert. Es ist jedoch möglich, den in Microsoft Dynamics 365 Business Central integrierten Event Recorder (Ereignis-Aufzeichnung) zu verwenden, um eine Liste der Ereignisse zu erhalten, die als Ergebnis Ihrer Aktionen auf der Plattform ausgelöst wurden. Weitere Informationen finden Sie unter [Discoverability of events](https://learn.microsoft.com/en-us/dynamics365/business-central/dev-itpro/developer/devenv-events-discoverability) (Microsoft-Artikel).

Unten finden Sie eine Liste der Expense Management-Versionen und die entsprechenden Event Publisher:

- [Event Publisher 25.00](@EM-266)
- [Event Publisher 24.00](@EM-218)
- [Event Publisher 12.00](@EM-185)
- [Event Publisher 11.00](@EM-152)
- [Event Publisher 10.00](@EM-102)
- [Event Publisher 9.00](@EM-77)
- [Event Publisher 8.00](@EM-71)
