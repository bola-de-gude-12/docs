# Upgrade von NAV/Business Central mit einer Installation von Expense Management

Wenn Sie Continia Expense Management in einer alten Version von Microsoft Dynamics NAV/Business Central verwenden und NAV/Business Central auf eine neuere Version aktualisieren möchten, müssen Sie je nach verwendeter Version auch Expense Management aktualisieren und/oder zugehörige Aufgaben ausführen.

Die folgenden Artikel enthalten Richtlinien zu den genauen Schritten, die in solchen Szenarien erforderlich sind. Um auf die für Sie relevanten Richtlinien zuzugreifen, wählen Sie einfach die Version von Business Central aus, auf die Sie aktualisieren möchten:

- [Business Central April 2019 (v14, FOB-basiert)](Business Central April 2019, v14.md)
- [Business Central 2020 Release Wave 2 (v17)](Business Central 2020 Release Wave 2, v17.md)
- [Business Central 2021 Release Wave 1 (v18)](Business Central 2021 Release Wave 1, v18.md)
- [Business Central 2021 Release Wave 2 (v19)](Business Central 2021 Release Wave 2, v19.md)
- [Business Central 2022 Release Wave 1 (v20)](@EM-105)
- [Business Central 2022 Release Wave 2 (v21)](@EM-106)
- [Business Central 2023 Release Wave 1 (v22)](@EM-151)
- [Business Central 2023 Release Wave 2 (v23)](@EM-184)
- [Business Central 2024 Release Wave 1 (v24)](@EM-219)
- [Business Central 2024 Release Wave 2 (v25)](@EM-261)
- Business Central 2025 Release Wave 1 (v26) [Link wird in Kürze hinzugefügt]

> [!NOTE]
> Wenn die Version, auf die Sie aktualisieren möchten, in den Anleitungen oben nicht genannt ist, verwenden Sie bitte die in den Anleitungen bereitgestellten Informationen (einschließlich aller Links), um zu den für Sie relevanten Informationen zu gelangen. Informationen zum Upgrade auf Microsoft Dynamics NAV 2018 finden Sie beispielsweise unter [Business Central April 2019 (v14, FOB-basiert)](Business Central April 2019, v14.md). Obwohl dieses spezielle Upgrade in dieser Anleitung nicht direkt beschrieben wird, enthält die Anleitung alle Informationen, die Sie zur Durchführung des Upgrades benötigen.
