# Upgrade mit mehreren installierten Continia-Lösungen (FOB)

Wenn Sie mehrere Continia-Lösungen installiert haben und eine oder mehrere davon aktualisieren möchten, müssen Sie ein paar wichtige Dinge in Bezug auf Continia Core beachten. Diese wichtigen Details werden in diesem Artikel beschrieben:

- [Upgrade von Continia Core (FOB)](Upgrading Continia Core, FOB.md)

Continia Core ist die Voraussetzung dafür, dass alle Continia-Lösungen ordnungsgemäß funktionieren.
