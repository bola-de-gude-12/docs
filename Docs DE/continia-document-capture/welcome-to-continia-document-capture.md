# Welcome to Continia Document Capture

```
id: DC-66
date: 28-10-2025
description: An introduction to Continia Document Capture.
lang: en
```

Document Capture ist eine End-to-End-Lösung für Microsoft Dynamics 365 Business Central, mit der Sie Rechnungen und andere Geschäftsbelege importieren, mit Texterkennung (OCR) verarbeiten, registrieren, genehmigen und archivieren können. Es vereinfacht und automatisiert jeden Schritt Ihres Prozesses zur Dokumentenverwaltung und bietet einen vollständigen Genehmigungsworkflow, der es Benutzern ermöglicht, Belege zu genehmigen, nachdem sie empfangen, gescannt und registriert wurden. Darüber hinaus ermöglicht das digitale Archiv den einfachen Zugriff auf zuvor verarbeitete Belege zur Prüfung, für die Berichterstellung oder für andere Zwecke.
