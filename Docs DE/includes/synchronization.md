Das Herunterladen von {param2} von Continia Online wird in Expense Management als Synchronisierung bezeichnet. Sie können die Synchronisierung von verschiedenen Stellen im System aus starten, unter anderem von der Seite Expense Management Einrichtung.

So führen Sie die Synchronisierung durch:

1. Wählen Sie das Symbol {{search}}, geben Sie **Expense Management Einrichtung** ein und wählen Sie dann den zugehörigen Link.
2. Wählen Sie in der Aktionsleiste **Aktionen** > **Funktion** > **Synchronisiere mit Continia Online**.

Wenn die Synchronisierung abgeschlossen ist, wird ein Dialogfeld zur Bestätigung mit Details zu den heruntergeladenen {param2} angezeigt.

Die regelmäßige Synchronisierung kann oft zeitaufwändig sein. Um diesen Prozess zu optimieren, bieten Aufgabenwarteschlangen eine automatische Ausführung im Hintergrund. Weitere Informationen zu Aufgabenwarteschlangen in Expense Management finden Sie im Artikel [Aufgabenwarteschlangen einrichten](@EM-65).
