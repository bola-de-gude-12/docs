Das Modul Continia-Benachrichtigungen informiert Sie über relevante Aktualisierungen mithilfe von Benachrichtigungen direkt in Microsoft Dynamics 365 Business Central. Wenn beispielsweise ein Betriebsproblem Ihre Continia-Lösungen beeinträchtigen kann, werden Sie mit Continia-Benachrichtigungen direkt darüber informiert. Wenn neue größere Updates für Ihre Continia-Lösungen herausgegeben werden, können Sie direkt über die Benachrichtigung auf eine kurze Einführung und ein Video mit ausgewählten Funktionen zugreifen.

> [!IMPORTANT]
> Wenn Sie eine geschlossene On-Premises-Umgebung haben und auf die neueste Version von Continia {param1} aktualisieren, müssen Sie ausgehenden Datenverkehr zu den Adressen für Continia-Benachrichtigungen zulassen, die unter [Firewall-Anforderungen](@DC-135#firewall-anforderungen) angegeben sind. Andernfalls können Sie keine Benachrichtigungen von Continia empfangen.

> [!NOTE]
> Die Benachrichtigungen werden nur Benutzern angezeigt, für die sie relevant sind. Sie werden an Business Central-Benutzer versendet, basierend auf Parametern wie Benutzerrolle, Continia-Lösung und -Version, Business Central-Version und Land.

## So zeigen Sie Benachrichtigungen an

Es gibt zwei Arten von Continia-Benachrichtigungen:

- Benachrichtigungen über Unterbrechungen oder Aktualisierungen Ihres Systems
- Relevante Informationen zu Releases für Ihre Lösungen

So zeigen Sie die Informationen der Benachrichtigungen an:

1. Wählen Sie im Rollencenter die Continia-Benachrichtigung und dann **Mehr anzeigen** aus. Sie können auch das Symbol zum Suchen verwenden, **Continia-Benachrichtigungen** eingeben und dann den entsprechenden Link auswählen.
2. Auf der Seite **Continia-Benachrichtigungen** sehen Sie eine Liste aller aktiven Benachrichtigungen. Wählen Sie die entsprechende Benachrichtigung aus, um die Details im rechten Bereich anzuzeigen. Aktualisierungen bestehender Benachrichtigungen werden in der ursprünglichen Benachrichtigung angezeigt.

> [!NOTE]
> Die Benachrichtigung wird im Rollencenter angezeigt, bis Sie sie gelesen haben oder bis die Systemunterbrechung behoben wurde.

## So deaktivieren Sie Benachrichtigungen

Wenn Sie keine Continia-Benachrichtigungen mehr über Aktualisierungen zu Ihrem Systemstatus erhalten möchten, können Sie diese deaktivieren. Dabei können Sie Aktualisierungen einer bestehenden Benachrichtigung oder Continia-Benachrichtigungen insgesamt deaktivieren.

So deaktivieren Sie einzelne Benachrichtigungen:

1. Suchen Sie nach der Seite **Continia-Benachrichtigungen** und öffnen Sie sie.
2. Deaktivieren Sie das Kontrollkästchen **Abonniert** für die gewünschte Benachrichtigung.

So deaktivieren Sie alle Continia-Benachrichtigungen:

1. Suchen Sie nach der Seite **Meine Benachrichtigungen** und öffnen Sie sie.
2. Gehen Sie zur Benachrichtigung **Benutzer über Statusaktualisierungen von Continia benachrichtigen** und deaktivieren Sie das Kontrollkästchen **Aktiviert**.

Sie erhalten dann keine Statusaktualisierungen mehr zu Ihren Continia-Lösungen.
