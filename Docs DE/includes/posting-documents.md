Sobald Belegposten genehmigt sind, können sie gebucht werden, um die entsprechenden Konten in Business Central zu aktualisieren und Rückerstattungen zu verarbeiten. Beachten Sie, dass gebuchte Belege endgültig sind und nicht bearbeitet werden können. Sie können einen einzelnen Beleg buchen oder eine Stapelbuchung durchführen.

## So buchen Sie einen Beleg

Durch die Buchung eines genehmigten Belegs wird dessen Verarbeitung abgeschlossen und in Business Central erfasst. Während des Buchungsvorgangs werden durch automatische Prüfungen etwaige Probleme erkannt:

- **Warnungen** – öffnet ein Dialogfeld, das es Ihnen ermöglicht, potenzielle Probleme zu umgehen.
- **Fehler** – müssen behoben werden, bevor der Beleg erfolgreich gebucht werden kann.

So buchen Sie einen einzelnen Beleg:

1. Wählen Sie das Symbol {{search}}, geben Sie **Belege** ein, und wählen Sie dann den zugehörigen Link.
2. Wählen Sie den genehmigten Beleg aus, den Sie buchen möchten, und wählen Sie in der Aktionsleiste **Start** > **Buchen**. Der Beleg wird anschließend gebucht und in den Bereich **Gebuchte Belege** verschoben.

## So buchen Sie mehrere Belege in einem Stapel

Sie können mehrere Belege in einem Stapel buchen. Gehen Sie dazu folgendermaßen vor:

1. Wählen Sie das Symbol {{search}}, geben Sie **Belege** ein, und wählen Sie dann den zugehörigen Link.
2. Wählen Sie in der Aktionsleiste **Start** > **Stapelbuchen**.
3. Ein Dialogfeld wird angezeigt, in dem Sie zusätzliche Angaben machen können:
   - **Nach Abrechnung gruppieren** – wählen Sie diese Option, um Belege in einer neuen Abrechnung zu gruppieren und zusammen zu buchen.
   - **Ersetze Buchungsdatum** – aktivieren Sie diese Option, um das Buchungsdatum für mehrere Belege zu ändern und die bevorzugte Ersetzungsmethode auszuwählen.
   - **Filter: Beleg** - in diesem Abschnitt können Sie Filter festlegen, nach denen Belege gebucht werden sollen.
4. Wählen Sie **OK** aus, um die Belege zu buchen, und verschieben Sie sie in **Gebuchte Belege**.

> [!NOTE]
> Alle Einträge mit Warnungen oder Fehlern werden beim Stapelbuchen von Belege übersprungen, bis der Administrator die Probleme behebt. Der Administrator erhält eine Rückmeldung zu den entsprechenden Problemen und muss diese Posten einzeln bearbeiten, bevor sie erfolgreich gebucht werden können.

## So exportieren Sie Anhänge

Sie können die Anhänge gebuchter Belege exportieren. Dies kann beispielsweise bei der Rückerstattung der Mehrwertsteuer praktisch sein. In vielen Ländern können Unternehmen eine Rückerstattung der Mehrwertsteuer auf bestimmte Ausgaben beantragen, die im Rahmen ihrer Geschäftstätigkeit anfallen. Die Steuerbehörden verlangen häufig Belege, wie Quittungen, Rechnungen und andere kostenbezogene Anlagen, um diese Ansprüche nachzuweisen.

So exportieren Sie Anhänge:

1. Wählen Sie das Symbol {{search}}, geben Sie **Gebuchte Belege** ein, und wählen Sie dann den zugehörigen Link.
2. Wählen Sie in der Aktionsleiste **Zugehörig** > **Belege** > **Anhänge herunterladen**.
