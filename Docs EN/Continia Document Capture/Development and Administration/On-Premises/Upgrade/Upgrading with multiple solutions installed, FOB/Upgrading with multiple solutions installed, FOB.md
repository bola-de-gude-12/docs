# Upgrading with Multiple Continia Solutions Installed (FOB)

When you have multiple Continia solutions installed and want to upgrade one or more of these, there are some important things to note relating to Continia Core and the Continia Delivery Network. These important details are described in the following articles:

* [Upgrading Continia Core (FOB)](Upgrading Continia Core, FOB.md)
* [Upgrading the Continia Delivery Network (FOB)](Upgrading the Continia Delivery Network, FOB.md)

Continia Core is essential for all Continia solutions to work properly, whereas the Continia Delivery Network is currently only relevant to Continia Document Capture and Continia Document Output.
