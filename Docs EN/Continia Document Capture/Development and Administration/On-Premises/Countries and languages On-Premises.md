# Country/Regional availability and supported languages for FOB
This article provides an overview of what countries/regions Continia Document Capture is available in, and what languages are supported for FOB versions of Microsoft Dynamics 365 Business Central. For information on country and language availability for app versions of Business Central, see [Country/Regional availability and supported languages](@DC-1).

> [!NOTE]
> The last version to include support for FOB versions of Business Central is Document Capture 2024 R2 (v25). This means that the below information on countries, regions, and languages will not be updated any further.

## Countries/regions and languages
The following table lists all countries/regions where Document Capture is available, along with the corresponding language/region codes and language tags. Any applicable comments are provided in footnotes.

|   |   |   |
| --- | --- | --- |
| **Country/region** | **Language/region code** | **Tag** |
| Australia | AU/AP | en-AU |
| Austria | AT | de-AT |
| Belgium | BE | nl-BE, fr-BE |
| Canada | NA | en-CA |
| Denmark | DK | da-DK |
| Finland<a href="#footnote-2"><sup>2</sup></a> | FI | fi-FI |
| France<a href="#footnote-1"><sup>1</sup></a> | FR | fr-FR |
| Germany | DE | de-DE |
| Netherlands | NL | nl-NL |
| New Zealand | NZ/AP | en-NZ |
| Norway | NO | no-NO |
| Poland<a href="#footnote-2"><sup>2</sup></a> | PL | pl-PL |
