# Keyboard shortcuts

This article provides an overview of the keyboard shortcuts available in Continia Document Capture.

Keyboard shortcuts can enhance your productivity and improve overall accessibility. By reducing reliance on the mouse, shortcuts allow you to input data and execute commands more quickly and efficiently. This is particularly valuable in environments where repetitive tasks and multi-monitor setups are common.

>[!TIP]
>For the full list of keyboard shortcuts available in Microsoft Dynamics 365 Business Central, see [Keyboard shortcuts](https://learn.microsoft.com/en-us/dynamics365/business-central/keyboard-shortcuts) (Microsoft article).
