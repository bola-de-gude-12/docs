# Support for Additional XML Formats

## Business value
With more XML document formats supported, you can receive invoices and credit memos from more countries without having to perform any manual template modifications or similar.

## Feature details
Support for the following XML document formats will be added:

* Finvoice for Finland
* SimplerInvoicing for Belgium/Netherlands
* Fattura for Italy
* ZUGFeRD/X-Rechnung for Germany
* Facture-X for France
* UTS format and functionality for Denmark
* PINT, the future international Peppol format (if launched in time for the next Continia release)

> [!NOTE]
> Some of the above may be delivered in a service pack for Continia Document Capture 6.50.
