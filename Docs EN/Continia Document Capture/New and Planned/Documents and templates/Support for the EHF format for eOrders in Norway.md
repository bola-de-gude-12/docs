# Support for the EHF Format for eOrders in Norway

| Feature | Public preview | General availability online |
| --- | :-: | :-: |
| Support for the EHF format for eOrders in Norway | - | Oct 2025 |

>[!NOTE]
>This feature will be released in Document Capture 2025 R2.

## Business value

This feature will allow you to meet the Norwegian requirement of sending and receiving business documents in the EHF format using Continia eDocuments. You can read more about the legal requirements in [Managing Electronic Documents in Norway](@DC-186).

## Feature details

With the implementation of the feature, Continia eDocuments will support the sending and receipt of eOrders in the Norwegian EHF format using the Continia Delivery Network. This will only apply to orders.
