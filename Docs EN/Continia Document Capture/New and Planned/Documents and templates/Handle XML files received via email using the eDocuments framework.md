# Handle XML Files Received via Email Using the eDocuments Framework

| Feature | Public preview | General availability online |
| --- | :-: | :-: |
| Handle XML files received via email using the eDocuments framework | - | Oct 2025 |

>[!NOTE]
>Due to the complexity involved in the proper implementation of this feature, its release has been moved to October 2025.

## Business value

This update will unify the handling of XML files in Document Capture. Regardless of the source of the XML file (that is, email or Continia Delivery Network), the XML will be processed via the eDocuments feature.

## Feature details

Currently, XML files received via email are handled using the basic XML import functionality. This update will allow you to import and handle XML files using the new eDocuments feature – taking advantage of its extended functionality.
