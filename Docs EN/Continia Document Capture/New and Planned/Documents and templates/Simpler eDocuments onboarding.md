# Simpler eDocuments Onboarding

| Feature | Public preview | General availability online |
| --- | :-: | :-: |
| Simpler eDocuments onboarding | - | Oct 2025 |

>[!NOTE]
>Due to the complexity involved in the proper implementation of this feature, its release has been moved to October 2025.

## Business value

The simplified onboarding procedure will significantly improve the overall experience of users registering with the Continia eDocuments network.

## Feature details

A new assisted setup guide will be created to make it easier for users to onboard to the Continia Delivery Network. The guide will make the onboarding less technical and ensure an overall smoother experience.
