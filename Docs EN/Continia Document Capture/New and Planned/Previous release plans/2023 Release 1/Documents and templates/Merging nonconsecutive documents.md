# Merging Nonconsecutive Documents

| Feature | Public preview | General availability online |
| --- | :-: | :-: |
| Merging nonconsecutive documents | {{checkmark}} Mar 2023 | {{checkmark}} Apr 2023 |

## Business value
Currently, you can only merge multiple nonconsecutive documents by applying a filter as a somewhat tedious workaround. With this feature, such workarounds will no longer be necessary.

## Feature details
On the **Split and Merge Documents** page, it will now be possible to merge documents that aren't immediately next to each other. Simply select the relevant documents by pressing Ctrl+Click, and then select **Merge** on the action bar.
