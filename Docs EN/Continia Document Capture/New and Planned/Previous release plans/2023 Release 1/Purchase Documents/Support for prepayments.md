# Support for Prepayments

| Feature | Public preview | General availability online |
| --- | :-: | :-: |
| Support for prepayments | {{checkmark}} Mar 2023 | {{checkmark}} Apr 2023 |

## Business value
With the added support for prepayments in Continia Document Capture, it will be possible for you to handle vendors demanding that goods and services are paid for in advance.

## Feature details
When an invoice that has been either identified as or marked as a prepayment invoice is registered, Document Capture will automatically post it as a prepayment invoice that's linked to an associated purchase order. A purchase order number must be entered or captured in order for this link to be made.

For more information, see [To register prepayment invoices](@DC-67#to-register-prepayment-invoices).
