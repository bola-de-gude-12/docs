# Receiving OIOUTS Documents via the Continia Delivery Network

| Feature | Public preview | General availability online |
| --- | :-: | :-: |
| Receiving OIOUTS documents via the Continia Delivery Network | {{checkmark}} Mar 2023 | {{checkmark}} Apr 2023 |

## Business value

This feature will enable you to receive OIOUTS (UtilityStatement) documents from your vendors directly via the Continia Delivery Network.

## Feature details

You'll be able to receive OIOUTS documents via the Continia Delivery Network. Your vendors can easily send these documents using the open e-business infrastructure NemHandel, and you'll then have them available for further processing in Continia Document Capture, just like other types of incoming documents.
