# Configuration of More Comments for Assignment

| Feature | General availability on-premises | General availability online | Public preview |
| --- | :-: | :-: | :-: |
| Configuration of more comments for assignment |{{checkmark}} Mar 2022 | {{checkmark}} Apr 2022 | - |

## Business value

This feature gives you added flexibility, as it allows you to configure more comments and have errors forwarded to specific users.

## Feature details

At the moment, it's possible to configure some document comments as either **Information**, **Warning**, or **Error** comments. As the batch registration process currently skips documents that have **Warning** comments, this means that you're able to control which comments can block the batch registration process, and you can even determine if documents with **Error** comments should be assigned to a certain user.

In the future, it will be possible to configure more comments than before. However, note that not all types of comments will have the same level of configurability.
