# Amount Limits for Four-Eyes Approval

| Feature | General availability on-premises | General availability online | Public preview |
| --- | :-: | :-: | :-: |
| Amount limits for four-eyes approval | {{checkmark}} Mar 2022 | {{checkmark}} Apr 2022 | - |

## Business value
As an alternative to having all invoices go through four-eyes approval, this feature will make it possible for you to apply four-eyes approval exclusively to invoices that are above a certain amount. All invoices with lower values will only need a single approver rather than two, resulting in an overall faster approval process.

## Feature details
With the feature, you'll be able to configure a threshold that triggers four-eyes approval for all invoices exceeding that threshold.

For more information, see [Enabling Purchase Approval](@DC-22).
