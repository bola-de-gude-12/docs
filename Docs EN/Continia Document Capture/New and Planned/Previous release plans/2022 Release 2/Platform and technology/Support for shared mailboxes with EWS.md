# Support for Shared Mailboxes with Exchange Web Services (EWS)

| Feature | Public preview | General availability online |
| --- | :-: | :-: |
| Support for shared mailboxes with Exchange Web Services (EWS) | - | {{checkmark}} Dec 2022 | - |

## Business value
By supporting the use of shared mailboxes when using Exchange Web Services (EWS), this feature will make it possible for you to keep your email service costs low. 

## Feature details
An additional configuration field will be added to the **Document Category** card and the OCR configuration file, allowing you to select **Shared Mailbox** when you configure email addresses using EWS.
