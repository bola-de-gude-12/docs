# Matching on Vendor Shipment and Vendor Order Numbers

| Feature | Public preview | General availability online |
| --- | :-: | :-: |
| Matching on vendor shipment and vendor order numbers | - | {{checkmark}} Oct 2022 |

## Feature details
The current functionality allows users to limit the auto-matching routine so that it only considers orders with specific order numbers. This functionality will be expanded to include filters on:

* **Vendor Order No.**
* **Vendor Shipment No.**
