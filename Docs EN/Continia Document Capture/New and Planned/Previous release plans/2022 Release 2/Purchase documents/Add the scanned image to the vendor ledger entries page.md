# Add the Scanned Image to the Vendor Ledger Entries Page

| Feature | Public preview | General availability online |
| --- | :-: | :-: |
| Add the scanned image to the vendor ledger entries page | - | {{checkmark}} Oct 2022 |

## Business value
Including the document image FactBox on the standard Business Central vendor ledger entries page makes it easier for you to view the posted documents in Business Central.

## Feature details
The document image will be added to page 29 – Vendor Ledger Entries.
