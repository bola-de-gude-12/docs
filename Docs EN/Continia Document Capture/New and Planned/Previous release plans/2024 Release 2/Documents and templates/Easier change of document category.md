# Easier Change of Document Category

| Feature | Public preview | General availability online |
| --- | :-: | :-: |
| Easier change of document category | {{checkmark}} Sep 2024 | {{checkmark}} Oct 2024 |

## Business value

Sometimes documents end up in the wrong category by mistake. Currently, you have to add the **Category** column to the document journal before you can change the category of a document.

The soon-to-be-added **Change Category** action allows you to quickly accomplish the same result, while also ensuring that the fields of the moved document are automatically recognized.

## Feature details

The capability to change the category of a document will be added to the document journal.

Selecting **Document** > **Change Category** on the action bar will present you with a list of categories that match the **Source Record Table ID** of the selected document(s). You'll then be able to move the document(s) to the correct category, which also results in an automatic field recognition.
