# Support for New Document Types in the Advanced Ordering Flow

| Feature | Public preview | General availability online |
| --- | :-: | :-: |
| Support for new document types in the advanced ordering flow | {{checkmark}} Sep 2024 | {{checkmark}} Oct 2024 |

## Business value

With support for additional document types, the advanced Continia eDocuments flow can be fully automated, meaning that you'll no longer have to reach out to your vendors using external means of communication (like email or phone calls) when they make changes to your orders. You'll be able to handle everything related to electronic ordering inside Microsoft Dynamics 365 Business Central if you wish to do so.

## Feature details

In the advanced Continia eDocuments flow, support will be added for two new document types:

* **eOrder Cancelation** documents
* **eOrder Change** documents

These types of documents will allow buyers to respond directly to sellers if they approve or reject any changes sent by the sellers. When a seller submits a change using an eOrder response document, the buyer will then be able to accept the change with an **eOrder Change** document or to reject the change with an **eOrder Cancelation** document.
