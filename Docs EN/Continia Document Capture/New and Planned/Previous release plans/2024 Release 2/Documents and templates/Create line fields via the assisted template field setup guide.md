# Create Line Fields via the Assisted Template Field Setup Guide

| Feature | Public preview | General availability online |
| --- | :-: | :-: |
| Create line fields via the **Assisted Template Field Setup Guide** | {{checkmark}} Sep 2024 | {{checkmark}} Oct 2024 |

## Business value

When using the **Assisted Template Field Setup Guide**, you now have the option to create not only header fields, but also line fields. This change simplifies and speeds up the resolution of a common task.

## Feature details

Prior to this, creating a template line field required going to the **Document Card Lines** subpage and then selecting **Create new Template Field**. Now both types of template field can be created from the same setup guide, so the **Template** action group has been removed from the **Document Card Lines** subpage.

To create a template line field:

1. Search ({{search}}) for and select **Document Categories**.
2. Select the **PURCHASE** code to open the document journal.
3. On the action bar, click **Template > Create new Template Field** to open the **Assisted Template Field Setup Guide** and select **Next** to get started.
4. In the **Template Field Type**, select **Line**.
5. Select the type of field you want to create and select **Next** to proceed with its configuration.
