# New Master Template Fields for Continia Sustainability

| Feature | Public preview | General availability online |
| --- | :-: | :-: |
| New master template fields for Continia Sustainability | {{checkmark}} Sep 2024 | {{checkmark}} Oct 2024 |

## Business value

Continia's new solution, Continia Sustainability, is closely integrated with Continia Document Capture. To facilitate your carbon accounting, several fields related to emissions are being added to Document Capture.

## Feature details

The following master template header and line fields are being added to Document Capture to enable its integration with Sustainability:

* Environmental Account Type
* Environmental Account No.
* Emission Type
* Emission Code
* Emission Unit Type
* Emission Unit Code
* Emission Quantity

Note that these fields are not required by default, and they're not automatically inserted in new templates. All other field properties carry the default values.
