# Date Calculation in the Formula Field

| Feature | General availability on-premises | General availability online | Public preview |
| --- | :-: | :-: | :-: |
| Date calculation in the formula field | {{checkmark}} Sep 1, 2021 | {{checkmark}} Oct 1, 2021 | - |

## Feature details
This feature will make it possible for you to use formulas in template date fields in order to combine recognized values (for example, **DOCDATE+DATEFIELD**), and to add/subtract a date value from a recognized value (for example, **DOCDATE+2W**).
