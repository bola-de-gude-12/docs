# Ability to View PDF/XML Files Directly from the Vendor Ledger Entries Screen

| Feature | General availability on-premises | General availability online | Public preview |
| --- | :-: | :-: | :-: |
| Ability to view PDF/XML files directly from the Vendor Ledger Entries screen | {{checkmark}} Mar 1, 2021 | {{checkmark}} Apr 1, 2021 | - |

## Business value
Having all relevant details easily available in one place is generally desirable, as this means you won't have to spend valuable time looking for them elsewhere. That's exactly the point of this feature, which provides quick insights into the details of any invoice or credit memo posted in Microsoft Dynamics 365 Business Central.

## Feature details
It will be possible to view PDF and XML files related to any specific vendor ledger entry directly from the **Vendor Ledger Entries** screen. In other words, you'll have access to more document details in the same convenient place. This makes it easier for you to find what you're looking for, thereby freeing up time for you to spend on other important matters.

> [!NOTE]
> This change will be developed using page extensions, meaning that it will only be available for Microsoft Dynamics 365 Business Central 2019 release wave 2 (BC15) or later.
