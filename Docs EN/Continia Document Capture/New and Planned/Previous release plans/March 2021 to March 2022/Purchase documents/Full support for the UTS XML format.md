# Full Support for the UTS XML Format

| Feature | General availability on-premises | General availability online | Public preview |
| --- | :-: | :-: | :-: |
| Full support for the UTS XML format | {{checkmark}} Jun 1, 2021 | {{checkmark}} Jun 1, 2021 | - |

## Business value
With more XML document formats supported, you can receive invoices and credit memos from more countries without having to perform any manual template modifications or similar.

## Feature details
It will be possible to process UTS files imported into the document journal. When a UTS file is registered, it will be removed from the document journal and attached to the related XML invoice/credit memo. It will also be possible to undo the action if a UTS file is attached to the wrong XML invoice/credit memo.
