# Direct Upgrade to Document Capture 2021 R2 (8.00)

| Feature | General availability on-premises | General availability online | Public preview |
| --- | :-: | :-: | :-: |
| Direct upgrade to Document Capture 2021 R2 (8.00) | {{checkmark}} Sep 1, 2021 | {{checkmark}} Oct 1, 2021 | - |

> [!NOTE]
> Previously, this feature was planned as a direct upgrade to Continia Document Capture 2021 R1 (7.00), but this has now been changed to Document Capture 2021 R2 (8.00).
> 
> The feature was released as planned on September 1, 2021. For a full description of it, including a step-by-step guide on how to carry out the direct upgrade, see [Upgrading from Versions 4.50–6.50 to Version 8.00](@DC-4).

## Business value
This feature will drastically reduce the time you spend on upgrading Continia Document Capture. It enables you to upgrade from any version of Document Capture from 3.50.07 onward to Document Capture 2021 R2 (8.00) in just two steps. 

## Feature details
With the current upgrade process, you can upgrade any Document Capture version from 3.50.07 onward directly to version 4.50, but from Document Capture 4.50 you must perform a stepped upgrade in order to upgrade to Document Capture 8.00, meaning that you must go through all versions in between. For example, if you want to upgrade from Document Capture 3.50.07 to 8.00, you must:

1. Upgrade from 3.50.07 to 4.50
1. Upgrade from 4.50 to 5.00
1. Upgrade from 5.00 to 5.50
1. Upgrade from 5.50 to 6.00
1. Upgrade from 6.00 to 6.50
1. Upgrade from 6.50 to 7.00
1. Upgrade from 7.00 to 8.00

With the new upgrade process, you'll be able to upgrade from any version of Document Capture starting with 3.50.07 to Document Capture 8.00 in just two steps. 

For example, if you want to upgrade from Document Capture 3.50.07 to 8.00, you'll only have to:

1. Upgrade from 3.50.07 to 4.50
1. Upgrade from 4.50 to 8.00
