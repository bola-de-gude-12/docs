# Support for Additional Storage Types

| Feature | General availability on-premises | General availability online | Public preview |
| --- | :-: | :-: | :-: |
| Support for additional storage types | {{checkmark}} Sep 1, 2020 | {{checkmark}} Oct 1, 2020 | - |

## Feature details
A number of events have been added in order to make it easy for partners to extend Document Capture to use other storage types. Azure Blob storage has also been added.
