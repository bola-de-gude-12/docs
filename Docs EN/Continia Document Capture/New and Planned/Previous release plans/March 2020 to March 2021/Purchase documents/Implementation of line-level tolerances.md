# Implementation of Line-Level Tolerances

| Feature | General availability on-premises | General availability online | Public preview |
| --- | :-: | :-: | :-: |
| Implementation of line-level tolerances | {{checkmark}} Sep 1, 2020 | {{checkmark}} Oct 1, 2020 | - |

## Feature details
Line-level tolerances have been implemented. Until now, Continia Document Capture has only featured a tolerance amount/percentage for totals, but this has now also been added for lines.

For more information, see [Automatic Document Matching](@DC-70).
