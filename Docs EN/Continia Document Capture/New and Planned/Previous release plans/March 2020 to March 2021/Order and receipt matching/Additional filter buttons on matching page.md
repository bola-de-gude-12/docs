# Additional Filter Buttons on Matching Page

| Feature | General availability on-premises | General availability online | Public preview |
| --- | :-: | :-: | :-: |
| Additional filter buttons on matching page | {{checkmark}} Sep 1, 2020 | {{checkmark}} Oct 1, 2020 | - |

## Feature details
Two new filter buttons, **Filter on Unmatched Only** and **Filter on Match Differences**, have been added to the matching page in order to enable users to:

* View lines with matching differences (unit cost).
* View unmatched lines only.
