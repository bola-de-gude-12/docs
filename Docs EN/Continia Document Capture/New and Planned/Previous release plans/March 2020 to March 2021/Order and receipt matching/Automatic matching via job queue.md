# Automatic Matching via Job Queue

| Feature | General availability on-premises | General availability online | Public preview |
| --- | :-: | :-: | :-: |
| Automatic matching via job queue | {{checkmark}} Sep 1, 2020 | {{checkmark}} Oct 1, 2020 | - |

## Feature details
With automatic matching via job queue, the matching routine is executed periodically (for example every night). This means that if invoices aren't matched upon import, for instance in case the purchase receipt is missing, they'll be matched automatically later, after a purchase receipt has been posted.
