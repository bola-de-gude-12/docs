# End Dates for Purchase Contract Lines

| Feature | Public preview | General availability online |
| --- | :-: | :-: |
| End dates for purchase contract lines | {{checkmark}} Mar 2024 | {{checkmark}} Apr 2024 |

## Business value

If you have multiple different purchase contracts associated with a single vendor, you can choose to have them represented as purchase contract lines on the **Purchase Contract** page. As the contracts are typically billed separately and function as separate entities, they're likely to have different end dates, but these end dates aren't immediately visible in the current user interface. With this feature, they'll be displayed for each line in the contract, thereby making it very easy for you to know when each of the contracts is about to expire. This is relevant for sorting and filtering purposes and provides a clear overview for improved contract management.

## Feature details

A new column, **End date**, will be added to the table in the **Lines** section of the **Purchase Contract** page in order to display the termination date for each purchase contract line. This not only makes it easy for you to view and track contract end dates, but also enables you to filter and sort on these, resulting in a significantly improved contract overview that allows you to manage your contracts and plan ahead with no hassle.

For more information, see [Creating Purchase Contracts](@DC-86).
