# Aligning the Purchase Contracts Module with North America

| Feature | Public preview | General availability online |
| --- | :-: | :-: |
| Aligning the Purchase Contracts module with North America | {{checkmark}} Mar 2024 | {{checkmark}} Apr 2024 |

## Business value

With this feature, the Purchase Contracts module will be aligned to work correctly with North American taxes. It will also provide North American customers with a good overview of their tax details and statistics and better control over their purchase contracts and subscriptions in general.

## Feature details

This feature will add sales tax calculation to the Purchase Contracts module and make it possible to define tax differences for each individual purchase contract line. You'll be able to do this using the newly added **Statistics** page, which will be accessible from the **Purchase Contract** page. Any taxes defined like this from the Purchase Contracts module will be considered when you register documents in the document journal.

For more information, see [Purchase Contracts for North America](@DC-192).
