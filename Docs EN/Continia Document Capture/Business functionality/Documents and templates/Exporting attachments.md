# Exporting attachments

This article describes the process involved in exporting document attachments in bulk, be them PDF or XML files.

This feature is particularly useful during audits, as multiple attachments can be retrieved at once – based on their vendor name, number, posting date, etc.

## To export attachments

To export document attachments in Continia Document Capture:

1. Search ({{search}}) for and select **Export Attachments**.
2. In the **Export Attachments** popup, configure the following settings:
   * **Export Related Documents** - if enabled, the drag-and-drop attachments related to the original documents (i.e.: the invoices) are included in the export.
   * **File Type to Export** - select the type of attachment you want to export. If XML is selected, the styled HTML file is also included in the export.
3. You can then use the **No.**, **Name**, and **Date** filters to narrow down your selection of attachments – as well as other filters. To export all document attachments, leave these filters empty.
4. Click **OK** to begin the export process. Alternatively, click **Schedule** to choose a different date and/or time for the export.

When the task is completed, Document Capture lists the number of attachments and vendors processed – and the exported document attachments are automatically saved as a ZIP file and downloaded. Each folder in the ZIP file represents a vendor, which can be identified by its vendor number.
