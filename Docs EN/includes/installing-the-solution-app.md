Before you can start using Continia {param1}, you must install it in your Microsoft Dynamics 365 Business Central tenant. If you’re not completely sure about using this product, sign up for a free 30-day trial with no strings attached. This allows you to explore the features before deciding on a subscription. You’re free to cancel the trial anytime if you find that it doesn’t meet your needs.

Continia {param1} is available from Microsoft AppSource, and there are various ways to install it. For more information, see [Customizing Business Central using extensions](https://docs.microsoft.com/dynamics365/business-central/ui-extensions) (Microsoft article).

> [!IMPORTANT]
> To install {param1}, you must be assisted by an approved Continia partner. During the activation process, a person from your partner’s organization will be required to log in using their own user credentials.
> 
> Also, to install extensions from AppSource or add per-tenant extensions, you must have the right permissions. You must either be a member of the D365 EXTENSION MGMT user group, or you must have the D365 EXTENSION MGMT permission set.

To install {param1}:
1. Search for ({{search}}) **Continia {param1}**, then select the related link under **Get from Microsoft AppSource**.
1.	Click **Free Trial** and complete the installation. For more information, see [Customizing Business Central using extensions](https://docs.microsoft.com/dynamics365/business-central/ui-extensions) (Microsoft article).
1.	**Optional:** To follow the installation status, search ({{search}}) for and select **Extension Management**.
1.	When the {param1} installation is complete, you receive a notification on the Role Center that {param1} has been installed.
1.	In the notification, click **Activate now**.
1.	Follow the steps in the **Activation Guide**.
1.	When the guide is finished, you see the **Continia {param1} Assisted Setup**, which helps you set up the solution.
