# Viewing Full Details in a Document Card from an Expense Report

| Feature                                 | General availability on-premises | General availability online |     Public preview     |
| --------------------------------------- | :------------------------------: | :-------------------------: | :--------------------: |
| Viewing full details in a document card |      {{checkmark}} Apr 2025      |   {{checkmark}} Apr 2025    | {{checkmark}} Mar 2025 |

## Business value
Being able to navigate directly to an expense document from an Expense Report enhances transparency and efficiency in expense management.

## Feature details
- Users can now open an expense in the Expense Report to view the complete Expense Document card.
- Offers a comprehensive view of all details related to each expense, beyond the list view.

 For more information, see [Submitting and managing expense reports](@EM-294) and [Setting up Expense Reports](@EM-306).
