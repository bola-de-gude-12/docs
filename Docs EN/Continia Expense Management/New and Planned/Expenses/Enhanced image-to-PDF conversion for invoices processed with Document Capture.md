# Enhanced Image-to-PDF Conversion for Invoices Processed with Document Capture

| Feature | Public preview | General availability |
| --- | :-: | :-: |
| Enhanced image-to-PDF conversion for invoices processed with Document Capture | {{checkmark}} Mar 2025 | {{checkmark}} Apr 2025 |

## Business value
This feature streamlines the invoice processing workflow by automatically converting image-based invoices to PDF format for seamless integration with Continia Document Capture. As you'll no longer need to manually convert files, this will save you time and reduce potential errors in invoice handling.

## Feature details
When you process expenses and select **Process with Document Capture**, supported image-format documents will automatically be converted to PDF before they're sent to Document Capture. This will allow for a more efficient and accurate invoice processing experience. For more information, see [Processing Purchase Invoices across Expense Management and Document Capture](@EM-277).
