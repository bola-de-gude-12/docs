# Expense reimbursement page shows all expense types


| Feature                              | General availability on-premises | General availability online |     Public preview     |
| ------------------------------------ | :------------------------------: | :-------------------------: | :--------------------: |
| Improved visibility of expense types |      {{checkmark}} Apr 2025      |   {{checkmark}} Apr 2025    | {{checkmark}} Mar 2025 |

## Business value
This feature improves visibility of the different expense types that are reimbursed to the Expense user.

## Feature details
For greater visibility, all expense types are now shown as columns on the Expense Reimbursement page. For more information, see [Reimbursing expenses](@EM-14).
