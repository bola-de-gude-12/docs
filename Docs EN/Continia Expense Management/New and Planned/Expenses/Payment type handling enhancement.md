# Payment Type Handling Enhancement

| Feature | Public preview | General availability |
| --- | :-: | :-: |
| Payment type handling enhancement | {{checkmark}} Mar 2025 | {{checkmark}} Apr 2025 |

## Business value
Businesses are evolving and with them their payment methods. They require more flexibility when assigning and unassigning payment types, better handling of old payment types, more flexibility when changes occur on payment types, and more control when things don’t go as expected. This feature will deliver a more seamless experience when working with payment types and expenses. 

## Feature details
Payment type-related processes will be redesigned, focusing on user experience. For more information, see [Setting up Payment Types](@EM-81).
