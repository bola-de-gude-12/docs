# Signing in with SSO via Azure Active Directory

| Feature | General availability on-premises | General availability online | Public preview |
| --- | :-: | :-: | :-: |
| Signing in with SSO via Azure Active Directory | {{checkmark}} Mar 1, 2021 | {{checkmark}} Apr 1, 2021 |-|

## Business value
Having multiple different sign-in credentials can be a hassle. So far, this has been necessary for expense users when they sign in to the Continia Expense Portal and the Continia Expense App, but this feature makes it possible to sign in to both of these using single sign-on (SSO), thereby making it a lot easier for users to access all relevant applications.

## Feature details
The feature will enable you as an expense user to sign in to both the Continia Expense Portal and the Continia Expense App via Azure Active Directory (AD) using the same email and password combination.

It will also allow admins to control which users have access to these applications through Azure AD.
