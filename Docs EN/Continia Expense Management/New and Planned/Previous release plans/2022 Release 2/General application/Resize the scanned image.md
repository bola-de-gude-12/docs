# Resize the Document Attachment Page

| Feature | Public preview | General availability |
| --- | :-: | :-: |
| Resize the document attachment page | - | {{checkmark}} Oct 2022 |

## Business value
Working with Expense Management using larger monitors, you often have unused space between the header field section and the document attachment. This feature allows the user to adjust the size of the page part showing the document, giving you a better overview of the entire document page.

## Feature details
You can now resize the document attachment factbox directly in the Document page and Expense, Mileage and Expense Reports. Adjusting the document attachment size will affect all pages where that particular factbox is displayed.
