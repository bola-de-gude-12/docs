# Expanded options for calculating per diem sub rates

| Feature | Public preview | General availability |
| --- | :-: | :-: |
| Expanded options for calculating per diem sub rates | {{checkmark}} Mar 2023 | {{checkmark}} Apr 2023 |

## Business value
More accurate application of per diem sub rates.

## Feature details
With this feature, you can choose between two options for calculating per diem sub rates, Daily or Last Day Only. In previous versions, sub rates were applied to the last day as default.
