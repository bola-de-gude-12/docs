# Bank Transaction Descriptions Follow Expenses

| Feature | Public preview | General availability |
| --- | :-: | :-: |
| Bank transaction descriptions follow expenses | {{checkmark}} Sep 19 2024 | {{checkmark}} Oct 2024 |

## Business value
Customers would like to keep bank transaction information on expenses, and this feature delivers that.

## Feature details
Bank transaction descriptions will no longer be overwritten by users.
