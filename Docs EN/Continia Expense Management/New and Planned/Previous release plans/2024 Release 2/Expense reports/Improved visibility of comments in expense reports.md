# Improved Visibility of Comments in Expense Reports

| Feature | Public preview | General availability |
| --- | :-: | :-: |
| Improved visibility of comments in expense reports | {{checkmark}} Sep 19 2024 | {{checkmark}} Oct 2024 |

## Business value
This feature provides easier access to comments in expense reports for expense users, as all comments will be visible at the same time from the Expense Report page instead of expense users having to look at each line to see them.

## Feature details
Comments are visible per document in an expense report, meaning that expense users will need to go to each document and then select Comments to view the specific comments belonging to the document. With this feature, expense users now have access to all comments belonging to an expense report via the Comment tab on the Expense Report page. For more information, see [Setting up expense reports](@EM-306).
