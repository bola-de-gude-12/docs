# Boolean Value Support in Field Dependencies

| Feature | Public preview | General availability |
| --- | :-: | :-: |
| Boolean value support in field dependencies | {{checkmark}} Mar 2025 | {{checkmark}} Apr 2025 |

## Business value
This enhancement allows you to create field dependencies using **Yes**/**No** (Boolean) values, providing more flexibility in defining conditional fields based on binary choices.

## Feature details
Field dependencies will now support the addition of Boolean data types. The Expense App and Expense Portal will seamlessly handle the logic for these dependencies, maintaining the same functionality and user experience as with existing field types. This will enable more tailored and efficient data entry configurations. For more information, see [Setting up field dependencies](@EM-250).
