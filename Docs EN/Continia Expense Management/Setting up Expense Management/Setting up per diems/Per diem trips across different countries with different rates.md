# Per diem trips across different countries with different rates

Managing per diem trips through countries with different rates has become much easier with the addition of this feature because all trip information can be added to one per diem instead of having to make a per diem for each country.

Expense users will be able to create one per diem trip covering trips through multiple countries. In the Expense App and the Expense Portal, expense users will have the option of selecting via points whenever they create a per diem.

## Setting up different rates for different countries

Expenses differ in cost from country to country, therefore it is best practice to set up a per diem rate that is country-specific.

To set up different per diem rates for different countries:

1. Search for {{search}} **Expense Management Setup**.
2. Under **Per Diem**, enable **Enable Multiple Destinations**.
3. Search for {{search}} **Expense Management Countries/Regions**.
4. On the action bar, click **Actions** (if you cannot see this, click **More options**).
5. Click **Copy Countries/Regions**.
6. Search {{search}} for **Per Diem Groups**.
7. Click **New**.
8. Fill out the fields as necessary to create rates for each country needed.
