# Customizing the Web Approval Portal

The procedure for customizing the Continia Web Approval Portal is the same for both online and on-premises installations of Microsoft Dynamics 365 Business Central. 

To customize the portal:

1. Search {{search}} for **Expense Management Setup**, then choose the related link.
1. On the action bar, select **Setup** > **Web Approval** to open the **Expense Management Setup / Continia Web Approval** page.
1. Under **General** > **Fields to Show**, select which fields (if any) you want to display in addition to the mandatory standard fields during approval in the Web Approval Portal.
1. Under **Custom Approval Fields**, you can enable your own custom fields, which will display in the Web Approval Portal. If you need additional fields, select **Show more** in the upper-right corner.
1. Select **Close** when you're done to save your changes and close the page.
