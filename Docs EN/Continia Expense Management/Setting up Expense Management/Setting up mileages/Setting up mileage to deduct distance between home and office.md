# Setting up Mileage to Deduct Distance between Home and Office

Certain companies don't reimburse users for the commute between home and place of work. You can set up Expense Management to automatically deduct the distance between home and office, meaning that users will only be reimbursed for distances travelled beyond this regular commute.

## Setting up home-to-office distance deduction

To set up this functionality, follow these steps:

1. Choose the {{search}} icon, enter **Expense Management Setup**, and then choose the related link.
2. In the **Mileage** FastTab, enable **Deduct distance between home and office.** 



Next step is to set up the distance on each individual user on their User Setup Card.



1. Select the {{search}} icon, enter **Continia User Setup**, and then select the related link.
2. Open the user card for each user. Under the **Expense Management** FastTab, enter the user's regular commuting distance in **Distance from home to office**. Users will be asked to specify their home address when creating mileages.

In Business Central, a comment is added to the mileage, and the specified distance is subtracted from the calculated distance automatically.
