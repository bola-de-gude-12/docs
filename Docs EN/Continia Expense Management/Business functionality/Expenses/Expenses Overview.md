# Overview of Expenses

After importing expenses into Expense Management, you can immediately start working with them. All of your expenses are in the Expenses list.

You can also further process expenses according to your company flow. For more information about the main processes of Continia Expenses, refer to the following table.


| **To** | **See** |
| --- | --- |
| Submit expenses using the Continia Expense App, the Continia Expense Portal, Business Central, or via email | [Submitting Expenses](@EM-5) |
| Move an expense to another company manually | [Moving an Expense to Another Company](@EM-20) |
| Approve documents | [Approving Documents](@EM-3) |
| Reject documents, thereby dismissing them but retaining them in the database (instead of deleting them entirely), and specify rejection comments |  [Rejecting Documents](@EM-4) |
| Reopen expenses with different statuses, for instance to match them again or to correct errors | [Reopening Expenses](@EM-17) |
| Post a single expense or multiple expenses in a batch | [Posting Expenses](@EM-11) |
| Reimburse expense users or expense user groups and apply filters to find the right expense users | [Reimbursing Expenses](@EM-14) |
| Create a business vendor so you can post expenses that were already paid for using a corporate credit card to this vendor | [Adding a Business Vendor](@EM-45) |
| Download expenses from Continia Online manually so you can begin processing them | [Downloading Expenses from Continia Online Manually](@EM-23) |
| Export expense attachments | [Posting Expenses: Export expense attachments](@EM-11#to-export-expense-attachments) |
| Learn how to avoid duplicate postings when using both Document Capture and Expense Management | [Processing Purchase Invoices in Document Capture and Expense Management](@EM-277) |
