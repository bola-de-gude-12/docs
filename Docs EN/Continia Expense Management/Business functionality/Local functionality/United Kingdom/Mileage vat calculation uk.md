# Mileage VAT calculation (UK)

Mileage claims are a popular way for businesses to reimburse employees for using their own car for business-related purposes. In the UK, it's possible to include VAT on mileage claims, though VAT can only be reclaimed for the fuel element of the mileage claim.

>  [!NOTE]
>  This functionality is only enabled for the UK localization of Expense Management.

To set up a mileage VAT calculation so expense users can reclaim VAT for the fuel element on mileage expenses, you must set a posting account for the fuel for a vehicle, and specify a mileage rate for that fuel.

To set up a fuel posting account for a vehicle:

1. Search for {{search}} **Vehicles**.
1. Select the vehicle that you want to set a fuel posting account for.
1. On the action bar, click **Setup**.
1. In the **Fuel Posting Account No.** field, enter the account number you want to use.

To specify a mileage rate for the fuel:

1. Search for {{search}} **Mileage Rates**.
1. In the **Fuel Rate (LCY)** column, enter the fuel rate. Repeat for each line as needed.
