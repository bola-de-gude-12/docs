# Item tracking   capture multiple values per item line

| Feature                                               | Public preview | General availability online |
| ----------------------------------------------------- | :------------: | :-------------------------: |
| Item tracking - capture multiple values per item line |        -       |           Apr 2026          |

### Business value

This addition will enhance the [item tracking capture feature](/broken/pages/05cd595cb22169940e110affed17f586c1fbc57d), providing you with more flexibility and allowing for tracking data in more invoices to be automatically interpreted.

### Feature details

Support for capturing multiple tracking values (i.e.: **Expiration Date**, **Lot No.**, **Package No.**, **Serial No.**, and **Warranty Date**) per item line will be added.

Examples:

* An item line with 3 pieces and 3 serial numbers.
* An item line with 10 pieces, 6 with Lot = A and 4 with Lot = B.
