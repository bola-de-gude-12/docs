# Item tracking   update order

| Feature                      | Public preview | General availability online |
| ---------------------------- | :------------: | :-------------------------: |
| Item tracking - update order |        -       |           Apr 2026          |

## Business value

This change will extend the functionality of the [item tracking capture feature](/broken/pages/9b07aab9ef452cba7e6992aa80e0c1f521391536). By allowing you to update an existing order with tracking data from an invoice, this feature will save you time and ensure the consistency of data across documents.

## Feature details

When using the **Match & Update Order** option in the **Invoice Reg. Step 1** field, in the **PURCHASE** category, item tracking information captured from a purchase invoice will be added to the matched purchase invoice.
