# Batch import improvements

| Feature                   | Public preview | General availability online |
| ------------------------- | :------------: | :-------------------------: |
| Batch import improvements |        -       |           Apr 2026          |

## Business value

The process of importing documents in batches will be improved, providing you with additional transparency and a more failproof field recognition flow. With a reliable batch import functionality, you’ll have less manual work and a more streamlined flow.

## Feature details

The document processing functionality will be redesigned, allowing for a more transparent and stable document import process. Not only will this revamp prevent errors from stopping the field recognition process, but also provide a log listing the documents that failed.
