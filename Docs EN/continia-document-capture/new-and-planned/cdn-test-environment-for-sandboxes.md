# CDN test environment for sandboxes

| Feature                            | Public preview | General availability online |
| ---------------------------------- | :------------: | :-------------------------: |
| CDN test environment for sandboxes |        -       |           Apr 2026          |

## Business value

This feature will enable you to perform end-to-end testing of the exchange of electronic documents in your own sandbox environment.

The goal is to shorten the user acceptance testing (UAT) period and speed up the implementation of Continia eDocuments, allowing for errors to be caught and addressed earlier in the implementation process. These sandboxes could also be used as proof-of-concept systems.

## Feature details

When this test environment is implemented, you’ll be able to:

* Onboard test participations.
* Send eDocuments between different sandbox environments, as long as both parties are using the same Continia solution and version.

Documents will be sent to a test version of the Continia Delivery Network, replicating the entire process involved in the transmission of documents.
