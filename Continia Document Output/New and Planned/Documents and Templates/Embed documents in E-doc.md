# Embed Documents in E-Doc

| Feature | Public preview | General availability |
| --- | :-: | :-: |
| Embed documents in E-doc | {{checkmark}} Mar 2025 | {{checkmark}} Apr 2025 |

## Business value

This new feature enables support of documentation for documents in Document Output, such as invoices. This documentation could, for example, consist of an easy-to-read PDF file based on the posted invoice or attached documents supporting an invoice when these can't be sent separately, as this is not possible when invoicing electronically though Peppol or OIOUBL.

## Feature details

With this extension of the feature released in Document Output 2024 R2 (DO25), it will add the option to embed all document attachments for Business Central into the XML file intended to be sent electronically. It will be configurable whether the PDF that will be embedded will be based on the report selection, posted document attachments or both or no embedding.
