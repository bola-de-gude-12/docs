# Report Selection Override Feature On Email Template Lines 

| Feature | Public preview | General availability |
| --- | :-: | :-: |
| Report selection override feature on email template lines | {{checkmark}} Mar 2023 | {{checkmark}} Apr 2023 |

## Business value

This features provides expanded options and more flexibility in report selection.

## Feature details

Every email template has a standard report selection attached. If a report selection is selected on an email template line, this selection will override the standard selection.

On email templates, you can select a standard report selection for a document type, which will be used when the email attachment is generated. If you select a report selection directly on an email template line, it enables you to use differentiated reports per, for example, customer group or any other way that email template lines are differentiated.
