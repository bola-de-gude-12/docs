# Add Handled (DO) and Output Profiles to Reminders

| Feature | Public preview | General availability |
| --- | :-: | :-: |
| Add Handled (DO) and Output Profiles to reminders | {{checkmark}} Mar 2025 | {{checkmark}} Apr 2025 |

## Business value

This new feature extends the usage of the Handled (DO) monitoring field to also cover the document type Reminder to ensure identical handling of document types throughout Document Output.

## Feature details

Adding Handled (DO) and Output Profiles to reminders yields the same search field options as you get on other document types, making it easy to identify if a document was printed, emailed or sent electronically (if the document type is supported). This is also a preparation step to add improved functionality for reminder handling in the Periodic Documents section of Document Output.
