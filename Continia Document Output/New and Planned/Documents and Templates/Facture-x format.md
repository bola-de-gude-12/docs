# Factur-x Format (FR)

| Feature | Public preview | General availability |
| --- | :-: | :-: |
| Facture-x format (FR) | {{checkmark}} Mar 2023 | {{checkmark}} Apr 2023 |

## Business value
This feature provides support for the Factur-x format for invoicing in France, and it will remain a key invoicing format for companies in France until PEPPOL Bis3 becomes the dominant format in B2B invoicing.

## Feature details
France has a country-specific document variant called Factur-x. It's a PDF file, though the attached XML file is the same format as can be send using the PEPPOL network.
