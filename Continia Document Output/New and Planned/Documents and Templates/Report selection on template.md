# Report Selection on Template

| Feature | Public preview | General availability |
| --- | :-: | :-: |
| Report selection on template | - | {{checkmark}} Oct 2022 |

## Business value
A template can use the same report selection as used for a document type in standard documents without configuring them on the template and in standard. If this report is changed at some point, the change will automatically also take effect on the template.

## Feature details
This feature will ensure that default templates are connected with a Report Selection, like Posted Invoice. When users change the Report ID in Standard Report Selection, they do not need to change this on the template.
