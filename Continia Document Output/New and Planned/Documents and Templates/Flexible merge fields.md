# Flexible Merge Fields

| Feature | General availability on-premises | General availability online | Public preview |
| --- | :-: | :-: | :-: |
| Flexible merge fields | {{checkmark}} Sep 1, 2021 | {{checkmark}} Oct 1, 2021 | - |

##Business value
With the ability to create new merge fields, you'll have more options for merging information into your emails.

##Feature details
On the **Merge Fields** page, you can set up all the fields you need for merging information into emails. With this feature, it will be easier for you to create new merge fields on that page.
