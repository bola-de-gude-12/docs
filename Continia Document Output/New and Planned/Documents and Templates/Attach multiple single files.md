# Attach Multiple Single Files

| Feature | Public preview | General availability |
| --- | :-: | :-: |
| Attach multiple single files | {{checkmark}} Mar 2023 | {{checkmark}} Apr 2023 |

## Business value

It may cause problems at the receiving end if attached open documents are combined into one PDF or attached as a zip file.

## Feature details

A new option on email templates makes it possible to attach all open documents as individual files.
