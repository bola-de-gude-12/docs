## Merge Multiple Attachments or Send Separately

| Feature | Public preview | General availability |
| --- | :-: | :-: |
| Merge multiple attachments or send separately | {{checkmark}} Mar 2024 | {{checkmark}} Apr 2024 |

## Business value
Document types like reminders or statements can have multiple related documents attached to them, and this feature provides functionality for choosing whether to send attachments as multiple documents or a single one in one email.

## Feature details
On email template lines, there are currently features available to combine and merge documents and also to use Background PDF functionality. This feature adds the option to keep all the documents split into several attachments for easier handling when received by a financial software solution such as Continia Document Capture.

Read more about this feature in the article [Combining Document](@DO-46).
