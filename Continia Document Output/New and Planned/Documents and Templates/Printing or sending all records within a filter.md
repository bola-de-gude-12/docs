# Printing or Sending All Records Within a Filter

| Feature | General availability on-premises | General availability online | Public preview |
| --- | :-: | :-: | :-: |
| Printing or sending all records within a filter | {{checkmark}} Dec 1, 2020 | {{checkmark}} Dec 1, 2020  | - |

##Business value
This feature will make it possible for you to process multiple records in one go – at the click of a button. It doesn't get more user-friendly than that.

##Feature details
The button **Print/E-Mail all** will be added, allowing you to easily print or send all records within the current filter. Records that have an email address will be emailed using SMTP, whereas records without an email address will be printed.
