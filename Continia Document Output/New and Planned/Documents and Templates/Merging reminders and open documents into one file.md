# Merging Reminders and Open Documents into one File

| Feature | Public preview | General availability |
| --- | :-: | :-: |
| Merging reminders and open documents into one file | {{checkmark}} Mar 2023 | {{checkmark}} Apr 2023 |

## Business value

Some document management systems have a limitation of only one attachment file per email. This feature can be enabled to combine certain documents into one file attachment, ensuring that the email won't be caught in a filter because of a limit on number of attachments.

## Feature details

The feature will be attached to an email template and be enabled on request. For statements and reminders, the open documents are picked up based on date filter and status, meaning that the “Combine Documents to one PDF” functionality can’t be used.
