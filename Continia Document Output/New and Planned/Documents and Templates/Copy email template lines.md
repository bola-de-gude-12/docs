# Copy Email Template Lines

| Feature | Public preview | General availability |
| --- | :-: | :-: |
| Copy email template lines | {{checkmark}} Sep 19 2024 | {{checkmark}} Oct 2024 |

## Business value

Email template lines can be used to generate various email-based dimensions or other variant options. This provides the ability to target the email content directly to a specific customer group with different email text lines.
However, in the current Document Output solution, it's not possible to copy the full content and configuration of an email template line and make a new variant/line with a minimal change, which will be possible with this new feature, speeding up the process and cutting down on the hassle when you create email template variants.

## Feature details

This feature will give you the option to make an exact copy of a selected email template line. The new line will not initially be enabled, and cannot be enabled until the line varies in some way from the original line, as this would cause a system error when emails are automatically created and distributed.

Read about how to use this functionality in [Working with email templates](@DO-52).
