# Multiple Sender Addresses (SMTP Servers)

| Feature | General availability on-premises | General availability online | Public preview |
| --- | :-: | :-: | :-: |
| Multiple sender addresses (SMTP servers) | {{checkmark}} Feb 15, 2021 | {{checkmark}} Feb 15, 2021  | - |

##Business value
This feature will give you even more flexibility when [customizing the sender of outgoing emails](Customizing the sender of outgoing emails.md).

##Feature details
The feature will enable you to create multiple **From E-Mail** addresses and to link them to different templates.
