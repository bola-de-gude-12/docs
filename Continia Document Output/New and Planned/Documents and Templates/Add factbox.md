# Improved FactBox with attachment names on template lines

| Feature | Public preview | General availability |
| --- | :-: | :-: |
| Improved FactBox with attachment names on template lines | - | {{checkmark}} Jun 2023 |

## Business value
You can attach files such as campaign flyers or Terms and Conditions in email templates. However, it is not convenient to view the attached documents as it requires entering the HTML template editor. This feature update aims to enhance the visualization of attachments per template line (variant), providing a more user-friendly experience.

## Feature details
In the current version of Document Output, you can only see the attached documents by accessing the HTML template editor. This update will add an attachment list to the template card, displaying the attachments for the selected email template line. Additionally, we are considering the possibility of allowing the addition of new attachments directly within the template, eliminating the need to enter the HTML editor whenever possible.

Read more about FactBoxes in email templates in [Working with Email Templates](@DO-52).
