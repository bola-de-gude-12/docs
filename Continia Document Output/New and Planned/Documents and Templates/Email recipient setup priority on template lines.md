# Email Recipient Setup Priority on Template Lines

| Feature | Public preview | General availability |
| --- | :-: | :-: |
| Email recipient setup priority on template lines | - | N/A |

> [!NOTE]
> The release of this feature has been postponed. A new release date has not yet been determined.

## Business value

This feature will enhance the option to make email template variants for different types of receivers, solving the challenge to find the right email address for a specific customer type if it differs from other receivers.

## Feature details

The list of email recipient options will still be implemented overall on the template, but the lists of priorities are attached to each email template line individually, providing the option to set different priorities on each variant. Blank priorities (or non-configured) will still indicate that the current implementation setting has the same priorities for all lines and will still be a valid configuration.
