# Option to Build a Password Based on Merge Fields to Protect PDF Files

| Feature | Public preview | General availability |
| --- | :-: | :-: |
| Option to build a password based on merge fields to protect PDF files | - | N/A |

>  [!IMPORTANT]
>  Unfortunately, it hasn't been possible to complete this feature in time for it to be included in Continia Document Output 2024 R1. However, the feature will be added as soon as possible.

## Business value

When the Password Protect PDF functionality is used, you need to share the password with the recipient. This feature lets you define rules for a password that consists of data unknown to anyone except the intended recipient.

## Feature details

Flexible configuration of passwords based on merge fields from the email template provides the option to indirectly share the password as clear text in emails. You could, for example, configure the PDF password so that it consists of a company name and a customer number.
