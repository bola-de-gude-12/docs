#Release of Document Output for Business Central Cloud

| Feature | General availability on-premises | General availability online | Public preview |
| --- | :-: | :-: | :-: |
| Release of Document Output for Business Central cloud | {{checkmark}} Nov 12, 2020 | {{checkmark}} Nov 12, 2020  | - |

##Feature details
With the release of Continia Document Output for Microsoft Dynamics 365 Business Central cloud, Document Output is no longer available only to users of Business Central on-premises.

The cloud version of Document Output is fully featured and includes almost all the same capabilities as the on-premises version, with only a few minor exceptions.
