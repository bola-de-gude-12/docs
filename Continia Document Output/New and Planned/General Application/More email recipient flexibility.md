# More Email Recipient Flexibility

| Feature | Public preview | General availability |
| --- | :-: | :-: |
| More email recipient flexibility | - | {{checkmark}} Sep 2023 |

## Business value

It is now easier to choose recipients when using email templates. It can be difficult to locate the right email address for all recipients of a specific document type. With this new feature, you can specify a set of fields to search for email addresses in a particular order of importance.

## Feature details
We will be replacing the E-mail Recipients Field with a table in the Recipient section of the email template. In this table, you can add the field numbers in order of priority. If the search for recipients leads to an email with a priority 1 field number, the remaining fields in the priority list will be skipped.

Read more about email recipients in [Working with Email Templates](@DO-52).
