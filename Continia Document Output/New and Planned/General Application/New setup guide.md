# New Setup Guide

| Feature | General availability on-premises | General availability online | Public preview |
| --- | :-: | :-: | :-: |
| New setup guide | {{checkmark}} Sep 1, 2021 | {{checkmark}} Oct 1, 2021  | - |

## Business value
With virtually no user interaction, the new setup guide makes Continia Document Output easier for you to use than ever before – less configuration, less hassle.

## Feature details
The new setup guide will automatically check whether the required email setup has been carried out correctly in Microsoft Dynamics 365 Business Central. In case something is missing, the setup guide will alert you to the specific issue(s).
