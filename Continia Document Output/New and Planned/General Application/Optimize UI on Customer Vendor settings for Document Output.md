# Optimize UI on Customer/Vendor Settings for Document Output

| Feature | Public preview | General availability |
| --- | :-: | :-: |
| Optimize UI on customer/vendor settings for Document Output | {{checkmark}} Mar 2025 | {{checkmark}} Apr 2025 |

## Business value

To ensure a better feeling of Document Output as a standard integrated feature to standard Business Central, setup fields will be added directly to customer settings and not to a separate setup card. 

## Feature details

In the present version of Document Output, there are two places where you can configure settings, such as customer settings, in relation to sending preferences, receiver setup and similar.

The most common setup items are set up on the Document Output Setup Card, but the Document Output FactBox also gives access to customer actions, which has a similar naming but mainly contains statistics fields from customer entries as well as supporting functions for sending statements. This will be redesigned in a more convenient way for the supported app versions, extending the standard customer card.
