# Continia Delivery Network

| Feature | General availability on-premises | General availability online | Public preview |
| --- | :-: | :-: | :-: |
| Continia delivery network | {{checkmark}} Mar 11, 2021 | {{checkmark}} Mar 11, 2021  | - |

##Feature details
Continia Delivery Network is a service that integrates with various electronic data exchange networks such as the PEPPOL eDelivery Network and various value-added networks, thereby enabling customers to export business documents via these.

If, for example, some of your customers are already connected to the PEPPOL eDelivery Network, they can send documents from Continia Document Output via this network using the Continia Delivery Network.
