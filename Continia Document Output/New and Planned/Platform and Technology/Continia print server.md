# Continia Print Server

| Feature | General availability on-premises | General availability online | Public preview |
| --- | :-: | :-: | :-: |
| Continia Print Server | {{checkmark}} Dec 1, 2020 | {{checkmark}} Dec 1, 2020  | - |

##Business value
With the new Continia Print Server, you'll be able to print documents (with background images) and to merge PDF files with colors, company logos, company information, sales and delivery terms, watermarks, and much more.

##Feature details
In addition to the above, the Print Server will make it possible for you to print directly from queues – without any manual handling – thereby allowing you to keep working without any interruptions. It will also support the [**Print/E-Mail all** functionality](/Continia Document Output/New and Planned/Documents and Templates/Printing or sending all records within a filter.md), which enables you to send or print all selected documents at the click of a button.

Stored locally at your premises, the Print Server will connect printers to the cloud through a network. Via the cloud network, it will be able to accept print jobs from your computers and to send them to the relevant printers.
