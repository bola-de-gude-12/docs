# OAuth Support in Document Output Service

| Feature | Public preview | General availability on-premises | General availability online |
| --- | :-: | :-: | :-: |
| OAuth support in Document Output Service | - | {{checkmark}} Apr 2022 | {{checkmark}} Apr 2022 |

## Business value
OAuth support in Document Output Service will replace Basic Authentication as authentication protocol.

## Feature details
When Basic Authentication is no longer supported in O365, we need to use OAuth instead. 

This requires a special setup, and you can find the detailed setup guide in the article [Setting up Oauth for Document Output Service](@DO-22).
