# Improved support for ZUGFeRD and Factur-x formats

| Feature | Public preview | General availability |
| --- | :-: | :-: |
| Improved support for ZUGFeRD and Factur-x formats | - | {{checkmark}} Jun 2023 |

## Business value
We will optimize the process of sending of invoices in the ZUGFeRD or Factur-x formats to recipients in Germany or France. This update will simplify the configuration process and optimize the selection of the Output profile, resulting in improved efficiency.

## Feature details
With this update, you can effortlessly send invoices in the ZUGFeRD or Factur-x formats, specialized PDF formats containing embedded XML files in a CII format. While these formats were already supported, previously, a specific setup was required on each customer card to utilize them. However, with this release, the ZUGFeRD/Factur-x profile is readily available for immediate use without additional setup.
