# Business Functionality

As an end-to-end solution for automating and streamlining the distribution of business documents, Continia Document Output offers considerable benefits to virtually any business.

Document Output currently consists of a single comprehensive module but is likely to be expanded with more modules in the future. The key features are listed below, along with an overall description of the solution.

> [!IMPORTANT]
> Not all features will be available in all versions of Document Output. Refer to [Comparison of Features by Document Output Version](@DO-18) to see features available by version of Document Output.

With Document Output, you get all the features you need to simplify your document output and distribution. You can easily batch-print business files, design your own professional-looking email templates using the HTML editor, bulk-email sales invoices, reminders, and other business documents – and much more.

Built directly inside Business Central, Document Output is easy to integrate and set up and comes with a host of features to make your life easier. The following are some of the key features:
