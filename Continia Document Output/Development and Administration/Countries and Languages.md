# Country/Regional Availability and Supported Languages
This page provides an overview of what countries/regions Continia Document Output is available in and what languages are supported for *app* versions of Microsoft Dynamics 365 Business Central. You can find information on country and language availability for _FOB_ versions of Business Central [here](@DO-2).

## Countries and regions
There are no localized versions of Document Output, meaning that it's available in all countries that are supported by Microsoft Dynamics 365 Business Central. You can see the general availability [here](https://learn.microsoft.com/en-us/dynamics365/business-central/dev-itpro/compliance/apptest-countries-and-translations).
