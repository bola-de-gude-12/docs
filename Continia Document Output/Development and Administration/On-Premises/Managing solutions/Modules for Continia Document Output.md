# Modules for Continia Document Output

Continia Document Output comprises a number of modules (sometimes referred to as *granules*) that each represent an area of functionality which you may or may not need, depending on your organizational needs and wants. For a list of these and all additional Continia modules, see [ISV modules to add to NAV/Business Central license](https://partnerzone.continia.com/isv-modules-to-add-to-navbusiness-central-license/) (only available to Continia partners).

To have a module added to your Document Output license, please contact [your dedicated Continia partner](@DO-11).

> [!NOTE]
> The base module for Document Output is mandatory, and you'll be charged according to the number of modules added by your partner.
