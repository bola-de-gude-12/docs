# Country/Regional Availability and Supported Languages for FOB
This page provides an overview of what countries/regions Continia Document Output is available in and what languages are supported for *FOB* versions of Microsoft Dynamics 365 Business Central. You can find information on country and language availability for _app_ versions of Business Central [here](@DO-1).

> [!NOTE]
>
> The latest version of Document Output to include support for FOB versions of Business Central is Document Output 2024 R2. This means that the below information on countries, regions and languages will not be updated any further.

## Countries/regions and languages
The following table lists all countries/regions where Document Output is available, along with the corresponding language/region codes and language tags. Any applicable comments are provided in footnotes.

|   |   |   |
| --- | --- | --- |
| **Country/region** | **Language/region code** | **Tag** |
| Australia | AU/AP | en-AU |
| Austria | AT | de-AT |
| Belgium | BE | nl-BE, fr-BE |
| Canada | NA | en-CA |
| Denmark | DK | da-DK |
| France | FR | fr-FR |
| Germany | DE | de-DE |
| Netherlands | NL | nl-NL |
| New Zealand | NZ/AP | en-NZ |
| Norway | NO | no-NO |
| Spain | ES | es-ES |
| Sweden | SE | sv-SE |
| Switzerland | CH | de-CH, fr-FR |
| United Kingdom | GB | en-GB |
| United States | NA | en-US |
| Worldwide | W1 | en-US |
