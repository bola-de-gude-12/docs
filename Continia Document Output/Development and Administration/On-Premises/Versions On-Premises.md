# Supported On-Premises Versions of NAV and Business Central
This page provides an overview of all supported *on-premises* versions of Microsoft Dynamics NAV and Microsoft Dynamics 365 Business Central. For information on what countries/regions Continia Document Output is available in and what languages are supported for these on-premises versions, see [this article](Countries and languages On-Premises.md).
