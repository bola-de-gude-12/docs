# How can I be sure that my e-document (Peppol/OIOUBL) arrived?

## Question

How can I be sure that my e-document (Peppol/OIOUBL) arrived?

## Answer

The various electronic invoicing platforms manage statuses and feedback in different ways.

In the case of Nemhandel, feedback is generated only if OIOUBL documents contain errors. Consequently, if there's no feedback within 48 hours, the documents are assumed to have been sent successfully.

On the other hand, Peppol documents prompt feedback for both successful and unsuccessful transfers, and feedback is typically received within a few hours.

To monitor the status of an e-document, and that of all sent e-documents, you can check **Outgoing Network Documents**.

If you want to view all documents sent to a particular customer, navigate to **Customers** and select that customer.
