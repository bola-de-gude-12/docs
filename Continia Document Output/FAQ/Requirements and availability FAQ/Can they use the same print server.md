# If I Have Multiple Companies in Document Output Cloud, Can They Use the Same Print Server?
## **Question**

If I have multiple companies in Document Output Cloud, can they use the same print server?

## Answer

Yes, you can even have multiple environments using the same printer server (service).  
In the configuration, you point to the webservice per company. That webservice also holds the reference to an environment (~database).
