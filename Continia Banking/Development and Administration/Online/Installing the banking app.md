# Installing the Continia Banking App

To use Continia Banking, you must first install it in your Microsoft Dynamics 365 Business Central tenant. If you're unsure about using Continia Banking, don't worry, as you can sign up for a free 30-day trial before starting a subscription. You can stop the trial if Continia Banking isn't suitable for you for any reason.

Continia Banking is available from Microsoft AppSource, and there are various ways to install it.
