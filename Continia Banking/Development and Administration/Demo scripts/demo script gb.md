# Continia Banking Demo Script

> [!IMPORTANT]
>
> The demo script for Continia Banking is currently in development. Stay tuned for updates!

This demo script provides a detailed description of how to demo the standard features in Continia Banking. The purpose of the Continia demo systems is to give Continia partners an environment to educate end customers and to present and test the latest versions of Continia Banking. 

This demo environment is already set up and ready to use, and it's always up-to-date with the latest version of Banking. Working with the demo script requires basic knowledge of Banking.
