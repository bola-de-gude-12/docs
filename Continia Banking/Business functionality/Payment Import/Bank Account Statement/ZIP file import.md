# ZIP file import

The new ZIP file import feature allows users to import a compressed file directly into the journal. The system automatically processes the ZIP contents, matching relevant PDFs to specific journal lines. This functionality supports CAMT53 and CAMT54 files and can be accessed from both the Bank Account Reconciliation List and Payment Reconciliation Journal List Pages through a dedicated fact box.
