# Delegating approval tasks

If you can't approve payments or need help during busy times, you can assign your approval tasks to someone else. Until you return, this person will handle approvals for you, ensuring that there are no delays and decisions are made in a timely manner.

To set up approval sharing:

1. Search ({{search}}) for and select **Payment Approval Sharing**.
2. On the action bar, select **Edit List** to make the fields editable.
3. Fill in the following fields:
   * **Owner User ID** and **Shared to User ID** - enter the user ID of the person who's sharing approval tasks – hereafter referred to as the original approver – and that of the substitute approver.
   * **Valid From** and **Valid To** - enter or select the start and end dates of the period in which approval tasks should be shared.
   * **Copy to all Companies** - by selecting this checkbox, the approval sharing is automatically copied to all companies where both the owner and shared-to user are present.
