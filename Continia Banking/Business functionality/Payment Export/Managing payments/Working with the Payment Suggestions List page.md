# Working on the payment suggestions page

If you have [Advanced Payment Suggestion mode](@CB-23) switched on, you work from the **Payment Suggestions** page, where you can view, adjust, and manage payment proposals, including direct debits. This custom view allows for detailed review before manually transferring suggestions to the standard Payment Journal for final processing.

To create new payment suggestions using advanced mode:
1. Search ({{search}}) for and select **Payment Suggestions**.
2. On the **Payment Suggestions** page, select **Suggest Vendor/Customer/Employee Payments**.
3. On the **Suggest Vendor/Customer/Employee Payment** page, you can apply filters. Optionally, you can use templates to save specific filter settings for payment suggestions. For more information on Payment Suggestion Templates, refer to the [Setting up Payment Suggestion Templates](@CB-89) article. 
4. You can now review details in the FactBox pane:
   * The **Suggestion Header FactBox** gives you an overview of each payment suggestion header’s status. For example, it shows you how many suggestions are open, (in)valid, transferred to a journal, and posted. 
   * The **Payment Totals FactBox** gives you a summary of payment details for each method.

1. If no adjustments are necessary, select **Create Gen. Journal Lines** to generate the payment journal entries. To make modifications, open the **Payment** card, where you can add or remove lines, adjust payment amounts, or apply discounts as needed.
