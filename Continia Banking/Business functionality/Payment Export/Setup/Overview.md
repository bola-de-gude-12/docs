# Overview - Export Settings

When you enable Continia Banking, you will be prompted to activate it. Activating Banking takes you through the necessary steps to set it up. This process takes you through the assisted setups of Bank Account Setup, Payment Bank Account Setup, and Payment Posting Setup. Alternatively, you can use the search icon and enter Banking Export Setup, Payment Bank Account Setup, and so on.

The following table provides an overview of the articles available for setting everything related to export in Continia Banking.

| ------------------------------------------------------------ | --------------------------------------------- |
| Set up your company's bank accounts for Continia Banking through the assisted bank account setup. | [Setting up bank accounts](@CB-37)            |
| Learn more about export settings and their customization options. | [Configuring export settings](@CB-23)         |
| Select the appropriate balance account for different payments. | [Setting up the payment bank account](@CB-41) |
| Set up payment postings and the corresponding accounts to be used. | [Setting up payment postings](@CB-39)         |
| Configure several options for the payment journal.           | [Setting up payment journals](@CB-40)         |
| Set up banks effectively for accurate communication.         | [Setting up banks](@CB-38)                    |
