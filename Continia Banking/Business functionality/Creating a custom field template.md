# Creating a custom field template

To create a custom template:

1. Search ({{search}}) for and select the template type that you want to customize. For example, enter *Description Templates* and select the related link.
2. On the action bar, select **New** or select an empty cell in the Template Type column.
3. Select a **Template Type** or enter a name.
4. Enter a **Template Code** and a **Template Description**.
5. In the **Description Pattern** column, select the three dots on the far right to access the **Field Selection** page, where you can choose from various fields to include. Select these fields and click **OK**.
6. Once selected, the pattern based on your chosen fields will be displayed. Select the three dots nest to the pattern to open the **Template Field Properties** page. You can now change the formatting for all of the selected fields:
   * **Length** - specify the maximum number of characters for the field.
   * **Place Before/After** - define whether the field appears before or after another field in the pattern.
   * **Opposite Sign** - use this option to invert the sign of numeric values (e.g., change positive to negative).
   * **Truncate From Left** - remove characters from the left side of the field if it exceeds the specified length.
   * **Date Format Type** - choose how dates should be displayed, such as full date, short date, or a custom format.
