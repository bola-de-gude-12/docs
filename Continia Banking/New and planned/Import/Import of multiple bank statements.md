# Import of multiple bank statements

| Feature | General availability online | Public preview |
| --- | :-: | :-: |
| Import of multiple bank statements |          {{checkmark}} June 2025          |    -    |



## Business value

This feature allows users to import multiple bank statements for the same bank account into a single reconciliation, streamlining the process of reconciling payments over a specified period. It saves time by enabling the user to consolidate statements from a week or month, improving efficiency and reducing manual handling.

## Feature details

Users can import multiple bank statements for the same bank account into a single reconciliation or payment journal. The system allows users to select a period (e.g., week or month) for which bank statements can be imported together. If a bank statement falls outside the selected period, a setting can automatically create a new reconciliation for it. This feature ensures that users can handle multiple daily statements in one process while maintaining accurate, period-based reconciliations.

For more information, see the [Importing multiple bank statements](@CB-245) article.
