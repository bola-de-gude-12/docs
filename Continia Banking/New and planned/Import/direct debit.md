# Direct Debit Handling in Statement Import

| Feature | General availability online | Public preview |
| --- | :-: | :-: |
| Direct Debit (including reversals) handling in statement import |          Apr 2025          | - |

## Business Value

This feature simplifies direct debit management for finance teams by automating the processing of payments and reversals. It reduces manual workload, minimizes errors, and enhances cash flow transparency.


## Feature Details

The feature allows users to manage direct debits directly within the reconciliation process. Returned direct debits can also be processed from the reconciliation, eliminating the need for manual intervention. Users can configure different handling options based on specific return reasons.
