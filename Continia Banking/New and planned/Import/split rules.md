# Split rules

| Feature | General availability online | Public preview |
| --- | :-: | :-: |
| Split rules |          {{checkmark}} Apr 2025          |    {{checkmark}} Mar 2025    |



## Business value

Automates the process of splitting incoming payments into multiple accounts.

## Feature details

Allows payment lines to be split into multiple entries based on predefined rules. These rules enable automatic distribution into different G/L accounts, assign custom posting groups, and apply specific dimensions.

To learn more about applying split rules, refer to the [Introducing Split Rules](@CB-213) article.
