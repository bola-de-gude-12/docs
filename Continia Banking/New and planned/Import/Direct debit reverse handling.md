# Direct Debit reverse handling

| Feature | General availability online | Public preview |
| --- | :-: | :-: |
| Direct Debit reverse handling |        {{checkmark}} Jun 2025        | - |

## Business Value

The Direct Debit Return feature will allow businesses to efficiently handle direct debit reversals, providing better control over payment corrections.


## Feature Details

This functionality will enable the system to process direct debit reversals automatically. When a payment is reversed, the system will update the accounting records accordingly. It will reduce manual intervention and speed up the resolution of payment issues.

For more information, see the articles [Introducing returned direct debits](@CB-242) and [Handling direct debit returns](@CB-243).
