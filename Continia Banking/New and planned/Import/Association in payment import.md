# Working with associations in Payment Import

| Feature | General availability online | Public preview |
| --- | :-: | :-: |
| Working with associations in Payment Import |          Oct 2025          |    -    |



## Business value

By handling associations in payment imports, this feature ensures that payments are correctly matched to their respective invoices or contracts, which is critical for maintaining accurate financial records and improving audit trails.

## Feature details

This feature will enable the system to recognize and handle associations between payments and related records, such as invoices or contracts, during the import process. This reduces the need for manual matching and improves the accuracy of financial data processing.
