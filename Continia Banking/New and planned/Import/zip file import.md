# Importing ZIP files

| Feature | General availability online | Public preview |
| --- | :-: | :-: |
| Importing ZIP files |          Oct 2025          |    -    |

## Business value

This feature will simplify the process of importing files by allowing users to drag and drop ZIP files containing multiple documents, improving efficiency and ease of use.

## Feature details

Users can drag and drop ZIP files into the system, which will automatically extract and process the files inside. This feature reduces the complexity of file uploads and supports seamless integration with external systems by handling compressed files in an efficient manner.
