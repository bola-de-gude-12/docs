## Business Value

The integration of CAMT.054 into Continia Banking brings more efficiency to the payment import workflows by providing a lot more detailed information for the matching process. Importing CAMT.054 files directly into the Payment Reconciliation journals helps users reduce manual data entry, minimize errors, and streamline payment processing through more automation.




## Feature Details

The CAMT.054 Import feature allows for the import of payments into payment reconciliation journals with greater detail than CAMT.053 files. When a CAMT.054 file is imported into Business Central, Continia Banking automatically matches the payments to the corresponding open entries.

For more information, refer to the [Importing payments](@CB-163).
