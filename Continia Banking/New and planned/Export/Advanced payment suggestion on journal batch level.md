# Advanced Payment Suggestion on Journal Batch Level

| Feature | General availability online | Public preview |
| --- | :-: | :-: |
| Advanced Payment Suggestion on Journal Batch Level |        {{checkmark}} Feb 2025        | - |

## Business Value

This feature empowers users to manage both simple and advanced payment modes within a single company. To enhance flexibility in payment processes, you can assign different modes to individual journal batches. For instance, companies can set the default mode to **Standard** in the Banking Export Setup for efficiency, while enabling **Advanced Mode** for specific journal batches that require more detailed handling.


## Feature Details

This functionality allows for the processing of various payment types with differing levels of detail across different journal batches. To adjust the settings, go to the **General Journal Batches** page. For more information, refer to the [Advanced Payment Suggestions](@CB-145) article.
