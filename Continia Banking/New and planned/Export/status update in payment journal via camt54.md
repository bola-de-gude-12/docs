# Status update in the payment journal via CAMT.054

| Feature | General availability online | Public preview |
| --- | :-: | :-: |
| Status update in Payment Journal via CAMT.054 |        {{checkmark}} Apr 2025        | {{checkmark}} Mar 2025 |

## Business value

Enhances reconciliation efficiency by allowing manual updates of payment statuses using CAMT.054 files.


## Feature details

Integrates CAMT.054 data for real-time journal status updates. If the account uses manual communication and the bank supports it, you can upload a CAMT.054 file to [update payment statuses](@CB-112#to-update-the-status).
