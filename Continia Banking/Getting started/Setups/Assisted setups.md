# Assisted setups for Continia Banking

You can configure Continia Banking through guided assisted setups. These setups are available for various modules and the general setup. You can access the setups on the Continia Banking Setup page, through a direct search using the {{search}} icon, or on the dedicated module pages.

Below are the assisted setups that are currently available:

* **Set up bank accounts** (Required) </br>
  Before you can proceed with using Continia Banking, it is mandatory to run the [Set up bank accounts](@CB-37) assisted setup.

- **Set up payment balance account** (Required)</br>
  See [Managing Balance Accounts for Different Currencies](@CB-41) for more information.

- **Set up recipient payment information** (Required) </br>
  This setup helps adapt recipient payment information for use within Continia Banking, ensuring accurate processing of payments. For more information, see [Setting up Recipient Payment Information](@CB-32).

- **Set up security** (Optional, extra module)</br>
  Guides you through the security setup, an optional module that provides additional security features. For more information, see [Working with Payment Approvals](@CB-28).

- **Set up bank closing day** (Optional)</br>
  Use this if the standard holidays are not sufficient for your setup. This allows for specifying additional bank closing days. For more information, see [Adding Bank Closing Days](@CB-42).
