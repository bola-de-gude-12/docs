# Quickstart guide – Setting up Continia Banking

The following steps take you through the required configurations before using Continia Banking with Microsoft Dynamics 365 Business Central.


| To                                         | Task                                                         |
| ------------------------------------------ | ------------------------------------------------------------ |
| Prepare your setup                         | To make it easier to start working with Continia Banking, you must prepare your setup. Refer to the [Preparing your setup](@CB-11) article for more details. |
| Prepare for direct or manual communication | If you want to use manual communication, go ahead and order the relevant files from your bank. Navigate to your bank from the [Bank integration](@CB-79) article to see which files you must order. If you want to use direct communication, refer to the prerequisites in the onboarding article for your bank to see if you need to request any subscriptions or services with the bank. The Direct Communication module is an optional add-on that must be included in your subscription separately. |
| Install and activate Continia Banking      | Continia Banking is available from Microsoft AppSource, and there are various ways to install it. For more information, see [Customizing Business Central online using apps](https://docs.microsoft.com/dynamics365/business-central/ui-extensions).<br />You can install a free trial version of Continia Banking following these instructions: [Sign up for a trial](@CB-05).<br />For more information on activating Continia Banking, refer to the [Solution Management](@CB-55) article. |
| Set up bank accounts                       | If you use manual communication, run the [Set up Bank Accounts](@CB-37) assisted setup guide. If you use direct communication, enable the Direct Communication service and follow the onboarding article for your bank service. Navigate to the relevant article from the Onboarding banks article. |
| Set up Recipient Payment Information       | Run the [Recipient Payment Information](@CB-32) assisted setup guide. |
| Set up Payment Balance Account             | Continia Banking lets you verify payment information and [select the appropriate balance account for different payments](@CB-41). |
| Set up Security                            | Create and set up a [payment approval workflow](@CB-28) for a payment journal. The Security module is an optional add-on that must be included in your subscription separately. |
