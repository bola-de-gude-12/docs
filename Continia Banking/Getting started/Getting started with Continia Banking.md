# Getting started with Continia Banking

Continia Banking is an end-to-end solution for Microsoft Dynamics 365 Business Central that enables you to streamline and automate payments and bank account reconciliation processes. It’s easy to sign up and get started, and there’s no need for complex integrations, as Continia Banking is already fully integrated into Business Central.

To begin using Continia Banking:

1. Search ({{search}}) for and select **Continia Banking Setup**. 

1. The **Continia Banking Setup** page offers access to the assisted and manual setup options:
   * For more information on the available assisted setup guides and their purpose, see the [Overview](@CB-34) article.
   * For more information on the manual setups, see the [Overview](@CB-125) article.

1. Use the **Setup Update Method** field to specify whether the bank system setup in Business Central should be updated manually or automatically. 

1. Additionally, set the **Setup Update Time** to schedule the automatic updates for a specific time each day.
