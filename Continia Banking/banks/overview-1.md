# Overview of supported banks (Airtable test)

To use Continia Banking effectively, it's important to understand how it integrates with different bank systems. The integration process varies based on the system your bank uses to handle financial data. You can either manually import and export bank files or use direct bank communication, depending on what your bank supports.

With Continia Banking, you can use manual communication to send bank data between the bank and Microsoft Dynamics 365 Business Central. To learn more about how to set up manual communication for a bank, refer to the [Setting up bank accounts](../../Continia%20Banking/Banks/@CB-37/) article.

To learn more about configuring direct communication, refer to the table below and select your bank from the left column to access the onboarding article specific to that bank.
