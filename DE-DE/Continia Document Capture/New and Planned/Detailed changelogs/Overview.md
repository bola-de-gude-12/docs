# Detaillierte Changelogs

Bei jeder Aktualisierung von Continia Document Capture wird ein detailliertes Changelog veröffentlicht. In diesem Changelog sind die neuen oder geänderten Funktionen und Fehlerbehebungen aufgeführt, die in der Version enthalten sind. Jeder Artikel enthält eine kurze Release Note, in der der Zweck und die Auswirkungen der Änderung erläutert werden.

Changelogs sind auch für das Continia Web Approval Portal, das Continia Delivery Network, den Document Capture Service und Continia Docs verfügbar.

- [Continia Document Capture 2025 R2](@DC-389)
- [Continia Document Capture 2025 R1](@DC-447)
- [Continia Document Capture 2024 R2](@DC-234)

Um ein vorheriges Changelog zu sehen, navigieren Sie über das Inhaltsverzeichnis auf der linken Seite direkt zum entsprechenden Abschnitt. Wenn Sie ein Mobilgerät verwenden, tippen Sie oben links auf der Seite auf die Schaltfläche **Inhalt**.

> [!TIP]
> Als Continia-Partner können Sie über neue Versionen und Service Packs von Document Capture benachrichtigt werden, wenn diese veröffentlicht werden. Informationen, um sich für diesen Service zu registrieren, finden Sie unter [Receive a notification upon new Document Capture releases](https://continia.zendesk.com/hc/en-us/articles/360005355079-Receive-a-notification-upon-new-Document-Capture-releases) (nur für Partner verfügbar).
