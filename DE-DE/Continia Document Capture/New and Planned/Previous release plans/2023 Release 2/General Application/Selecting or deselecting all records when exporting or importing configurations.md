# Aus- und Abwählen aller Datensätze beim Exportieren oder Importieren von Konfigurationen

| Funktion                                                                                 | Öffentliche Vorschau |     Allgemeine Verfügbarkeit Online     |   |
| ---------------------------------------------------------------------------------------- | :------------------: | :-------------------------------------: | - |
| Aus- und Abwählen aller Datensätze beim Exportieren oder Importieren von Konfigurationen |           -          | {{checkmark}} Okt. 2023 | - |

## Vorteile

Durch die Möglichkeit, alle Datensätze gleichzeitig aus- oder abzuwählen, sparen Sie viel Zeit, da Sie nicht mehr jedes einzelne Kontrollkästchen für Konfigurationsdatensätze manuell aktivieren oder deaktivieren müssen, um solche Stapelvorgänge durchzuführen.

## Details zur Funktion

Wenn Sie eine Konfiguration mithilfe des **Document Capture Konfigurationsassistenten** importieren oder exportieren, können Sie jetzt ganz einfach alle Konfigurationsdatensätze auf einmal aus- oder abwählen. Sie können dies oben links auf der Seite **Wählen Sie aus, was eingeschlossen werden soll** einstellen, indem Sie **Daten** und dann **Alle auswählen** oder **Alle abwählen** auswählen. Dadurch werden alle Kontrollkästchen in der Spalte **Inkludiert** aktiviert oder deaktiviert, je nachdem, welche der beiden Optionen Sie auswählen.

> [!NOTE]
> Ein Artikel, der den **Document Capture Konfigurationsassistenten** beschreibt, einschließlich Informationen über die oben genannte Funktion, wird unmittelbar nach dem Release von Continia Document Capture 2023 R2 veröffentlicht.
