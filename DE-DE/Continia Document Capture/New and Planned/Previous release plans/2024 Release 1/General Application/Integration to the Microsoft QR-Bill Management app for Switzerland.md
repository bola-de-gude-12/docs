# Integration in die Microsoft QR-Bill Management App für die Schweiz

| Funktion                                                            |   Öffentliche Vorschau  |     Allgemeine Verfügbarkeit Online     |
| ------------------------------------------------------------------- | :---------------------: | :-------------------------------------: |
| Integration in die Microsoft QR-Bill Management App für die Schweiz | {{checkmark}} März 2024 | {{checkmark}} Apr. 2024 |

## Vorteile

Diese Funktion ermöglicht eine direkte Integration zwischen Continia Document Capture und der von Microsoft für den Schweizer Markt entwickelten QR-Bill Management-App. Das bedeutet, dass Sie für die Nutzung von QR-Bill Management keine Software von Drittanbietern mehr erwerben und installieren müssen.

## Details zur Funktion

Immer wenn eine Einkaufsrechnung mit einem QR-Code in Document Capture importiert wird, wird der QR-Code automatisch von Document Capture gelesen. Allerdings ist aktuell noch eine Drittanbieter-App nötig, damit bei Verwendung der QR-Bill Management App die QR-Daten den richtigen Rechnungsfeldern zugeordnet werden.

Die neue Funktion integriert ähnliche Funktionen wie die Drittanbieter-App direkt in Document Capture und macht damit Drittanbieter-Software überflüssig. Wenn Sie die QR-Bill Management App installiert haben, sorgt die Funktion automatisch dafür, dass alle erfassten QR-Daten direkt in QR-Bill Management kopiert werden.

Weitere Informationen finden Sie unter [Integration der QR-Bill Management App](@DC-217).
