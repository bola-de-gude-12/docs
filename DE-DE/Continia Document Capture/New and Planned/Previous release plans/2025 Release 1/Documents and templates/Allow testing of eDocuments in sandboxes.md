# Testen von eDocuments in einer Sandbox

| Funktion                               |   Öffentliche Vorschau  |     Allgemeine Verfügbarkeit Online     |
| -------------------------------------- | :---------------------: | :-------------------------------------: |
| Testen von eDocuments in einer Sandbox | {{checkmark}} März 2025 | {{checkmark}} Apr. 2025 |

## Vorteile

Mit dieser Funktion können Sie die eDokumente-Funktionen von Continia in einer Sandbox verwenden und so Anpassungen und Abläufe testen, ohne Ihre Produktionsumgebung zu beeinträchtigen.

## Details zur Funktion

Wenn diese Funktion implementiert ist, wird das Senden und Empfangen von elektronischen Dokumenten in Testumgebungen unterstützt. Obwohl das Onboarding einer Teilnahme von einer Sandbox aus nicht möglich ist, können Sie eine Teilnahme kopieren und sie beim Erstellen der Sandbox aus der Datenbank als Testteilnahme markieren.

Weitere Informationen finden Sie unter [Continia eDocuments einrichten](@DC-179).
