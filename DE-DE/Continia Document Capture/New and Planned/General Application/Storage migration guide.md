# Speichermigrationsassistent

| Funktion                    | Allgemeine Verfügbarkeit On-Premises | Allgemeine Verfügbarkeit Online | Öffentliche Vorschau |
| --------------------------- | :----------------------------------: | :-----------------------------: | :------------------: |
| Speichermigrationsassistent |        {{checkmark}} März 2022       |     {{checkmark}} Apr. 2022     |           -          |

## Vorteile

Dieser neue Assistent unterstützt Sie beim Wechsel Ihrer Speichermethode und wird Ihnen diese Aufgabe erheblich erleichtern.

## Details zur Funktion

Dieser Assistent unterstützt Sie bei der Migration Ihres Speichers innerhalb von Continia Document Capture. Er bietet eine Reihe von Funktionen, zum Beispiel:

- Unterstützung der Migration zwischen zwei gleichen Speichertypen, z. B. **Datei** > **Datei** oder **Azure Blob** > **Azure Blob**
- Zusätzliche Überprüfung der Azure Blob-Anmeldeinformationen sowie ein Verbindungstest
