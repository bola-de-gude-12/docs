# Upgrade auf Continia Document Capture 2025 R1 (26.00) für FOB-Versionen

In diesem Artikel wird beschrieben, wie Sie Document Capture für FOB-Versionen von Microsoft Dynamics 365 Business Central aktualisieren. Informationen zum Upgrade von Document Capture für App-Versionen von Business Central finden Sie unter [Upgrade von Continia Document Capture für App-Versionen von Business Central](@DC-6).

Wenn Sie Ihre alte Version von Document Capture (DC) auf eine neuere Version aktualisieren, hängt der Aktualisierungsprozess davon ab, von welcher Version Sie aktualisieren und welche Version von Microsoft Dynamics NAV/Business Central Sie haben. Möglicherweise ist ein direktes Upgrade nicht möglich und Sie müssen möglicherweise auch Ihre Version von NAV/Business Central aktualisieren, um die Kompatibilität sicherzustellen. Informationen zum Aktualisieren Ihrer alten Version von NAV/Business Central unter Sicherstellung der Kompatibilität mit Document Capture finden Sie unter [Upgrade von NAV/Business Central mit einer Installation von Document Capture](@DC-7).

Mithilfe der folgenden Tabelle können Sie genau bestimmen, welche Aufgaben in Ihrem speziellen Szenario ausgeführt werden müssen, um Ihre alte Version von Document Capture auf Document Capture 2025 R1 (26.00) zu aktualisieren.

> [!IMPORTANT]
> Beachten Sie, dass Document Capture 2025 R1 (26.00) nur verfügbar ist, wenn Sie eine der folgenden Versionen von Business Central verwenden:
>
> - Business Central 2025 Release Wave 1 (BC v26)
> - Business Central 2024 Release Wave 2 (BC v25)
> - Business Central 2024 Release Wave 1 (BC v24)
> - Business Central April 2019 (v14, FOB-basiert)

> [!NOTE]
> Wenn Sie mehrere Continia-Lösungen in einer FOB-Version von Business Central installiert haben und eine oder mehrere dieser Lösungen aktualisieren möchten, müssen Sie beim Importieren von Objekten bestimmte Schritte ausführen. Weitere Informationen finden Sie unter [Upgrade mit mehreren installierten Continia-Lösungen (FOB)](/Continia Document Capture/Development and Administration/On-Premises/Upgrade/Upgrading with multiple solutions installed, FOB/Upgrading with multiple solutions installed, FOB.md).
