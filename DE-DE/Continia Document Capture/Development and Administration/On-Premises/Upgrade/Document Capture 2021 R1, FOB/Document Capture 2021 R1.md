# Upgrade auf Continia Document Capture 2021 R1 (7.00) für FOB-Versionen

In diesem Artikel wird beschrieben, wie Sie Document Capture für _FOB_-Versionen von Microsoft Dynamics 365 Business Central aktualisieren. Informationen zum Upgrade von Document Capture für _App_-Versionen von Business Central finden Sie unter [Upgrade von Continia Document Capture für App-Versionen von Business Central](@DC-6).

Wenn Sie Ihre alte Version von Document Capture (DC) auf eine neuere Version aktualisieren, hängt der Aktualisierungsprozess davon ab, von welcher Version Sie aktualisieren und welche Version von Microsoft Dynamics NAV/Business Central Sie haben. Möglicherweise ist ein direktes Upgrade nicht möglich und Sie müssen möglicherweise auch Ihre Version von NAV/Business Central aktualisieren, um die Kompatibilität sicherzustellen. Informationen zum Aktualisieren Ihrer alten Version von NAV/Business Central unter Sicherstellung der Kompatibilität mit Document Capture finden Sie unter [Upgrade von NAV/Business Central mit einer Installation von Document Capture](@DC-7).

Mithilfe der folgenden Tabelle können Sie genau bestimmen, welche Aufgaben in Ihrem speziellen Szenario ausgeführt werden müssen, um Ihre alte Version von Document Capture auf Document Capture 2021 R1 (7.00) zu aktualisieren.

> [!NOTE]
> Wenn Sie mehrere Continia-Lösungen in einer FOB-Version von Business Central installiert haben und eine oder mehrere dieser Lösungen aktualisieren möchten, müssen Sie beim Importieren von Objekten bestimmte Schritte ausführen. Weitere Informationen finden Sie unter [Upgrade mit mehreren installierten Continia-Lösungen (FOB)](@DC-54).

> [!NOTE]
> Es ist jetzt möglich, von Document Capture 4.50 oder höher direkt auf [Document Capture 8.00](@DC-10) zu aktualisieren. Dadurch entfallen die oben beschriebenen schrittweisen Upgrades vollständig und Sie können Ihre Software wesentlich einfacher auf den neuesten Stand bringen. Beachten Sie jedoch, dass dies erst ab Document Capture 8.00 möglich ist – ein direktes Upgrade von älteren Versionen auf Document Capture 7.00 ist nicht möglich.
