# Module für Continia Document Capture

Continia Document Capture wird mit einer Reihe von Modulen (auch als Granules bezeichnet) geliefert, die jeweils einen Funktionsbereich darstellen, den Sie je nach Ihrem geografischen Standort und den Anforderungen und Wünschen Ihres Unternehmens möglicherweise benötigen oder nicht. Eine Liste aller zusätzlichen Continia-Module finden Sie unter [ISV modules to add to NAV/Business Central license](https://partnerzone.continia.com/partnerzone/my-continia/isv-modules/) (nur für Continia-Partner verfügbar).

Um Ihrer Document Capture-Lizenz Module hinzuzufügen, wenden Sie sich bitte an [Ihren Continia-Partner](@DC-15).

> [!NOTE]
> Das Basis-Modul für Document Capture ist obligatorisch und Ihnen werden Kosten entsprechend der Anzahl der von Ihrem Partner hinzugefügten Module berechnet.
