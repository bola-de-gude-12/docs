# Eingebettete PDF-Dateien in der Dokumentenvorschau standardmäßig für XML anzeigen

Wenn Sie lieber eingebettete PDF-Anhänge als Original-XML-Dateien im Belegjournal anzeigen möchten, können Sie die Dokumentenvorschau so einrichten, dass diese standardmäßig angezeigt werden. Gehen Sie dazu folgendermaßen vor:

1. Suchen Sie {{search}} nach **Belegkategorien** und wählen Sie dann den entsprechenden Link aus.
2. Um die Kategorie für Einkaufsbelege zu öffnen, wählen Sie die Zeile **EINKAUF** (nicht den Code **EINKAUF**), und wählen Sie dann **Bearbeiten** in der Aktionsleiste.
3. Wählen Sie im Inforegister **Vorlagen** in der Liste der Vorlagen die XML-Mastervorlage aus, die Sie bearbeiten möchten, und wählen Sie dann **Bearbeiten** aus, um die Vorlagenkarte zu öffnen.
4. Aktivieren Sie im Inforegister **Allgemein** die Option **Eingebettete PDF-Dateien als Standard anzeigen**.

Die Dokumentenvorschau im Belegjournal zeigt jetzt standardmäßig alle PDF-Dateien an, die in eine importierte XML-Datei eingebettet wurden, anstatt das ursprüngliche XML anzuzeigen.
