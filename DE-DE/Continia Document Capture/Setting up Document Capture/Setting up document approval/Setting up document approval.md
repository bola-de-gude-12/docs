# Überblick über das Einrichten der Beleggenehmigung

Continia Document Capture bietet einen vollständigen Genehmigungsworkflow, der es Ihnen ermöglicht, Belege zu genehmigen, nachdem sie empfangen, gescannt und registriert wurden. Unter den Links in der folgenden Tabelle finden Sie Informationen, die Ihnen dabei helfen, die Genehmigung von Belegen den Anforderungen Ihres Unternehmens entsprechend einzurichten:

| Thema                                                                                                                                                                                | Siehe                                                                                             |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------- |
| Aktivieren und Konfigurieren von Continia-Genehmigungsworkflows, einschließlich dem Erzwingen der Genehmigung, Vier-Augen-Genehmigung sowie Genehmigungsprüfungen und -validierungen | [Einkaufsgenehmigung aktivieren](@DC-22)                                                          |
| Konfigurieren von Benutzern als Genehmiger, einschließlich Genehmigungslimits und Berechtigungen                                                                                     | [Continia Benutzereinrichtung für Genehmigungen](@DC-23)                                          |
| Einrichten von Genehmigungsgruppen, um Genehmigungen gemäß den Dimensionen von Zeilen in Einkaufsrechnungen durchzuführen                                                            | [Genehmigungsgruppen](@DC-24)                                                                     |
| Einrichten der automatischen geteilten Genehmigung, zum Beispiel bei Abwesenheit der Genehmiger                                                                                      | [Geteilte Genehmigung einrichten](@DC-37)                                                         |
| Erstellen einer Liste verfügbarer Genehmiger für jeden Benutzer, die zum Weiterleiten von Belegen zur Genehmigung ausgewählt werden können                                           | [Benutzerspezifische Listen von Genehmigern für die Genehmigungsweiterleitung erstellen](@DC-138) |
| Einrichten von E-Mail-Benachrichtigungen für alle entsprechenden Aktionen (Genehmigen, Ablehnen und Stornieren)                                                   | [E-Mail-Benachrichtigungen](@DC-26)                                                               |
| Erstellen und Bearbeiten von Listen mit Genehmigern, die für die Genehmigung von Einkaufsbelegen erforderlich sind                                                                   | [Genehmigungsabläufe erstellen](@DC-30)                                                           |
| Aktivieren und Einrichten der erweiterten Genehmigung, um Belegzeilen basierend auf Dimensionen und Grenzbeträgen zu genehmigen                                                      | [Erweiterte Genehmigung einrichten](@DC-31)                                                       |
