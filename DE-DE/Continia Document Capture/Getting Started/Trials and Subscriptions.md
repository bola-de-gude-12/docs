# Testversionen und Abonnements für Continia Document Capture

Es gibt verschiedene Möglichkeiten, Document Capture zu nutzen und einige davon können sogar kombiniert werden. Die verschiedenen Methoden werden im Folgenden ausführlicher beschrieben.

## Registrieren Sie sich für eine Testversion

Wenn Sie Document Capture testen möchten, bevor Sie ein Abonnement starten, können Sie sich [hier](https://appsource.microsoft.com/de-de/product/dynamics-365-business-central/PUBID.continia365%7CAID.6da8dd2f-e698-461f-9147-8e404244dd85%7CPAPPID.6da8dd2f-e698-461f-9147-8e404244dd85?tab=Overview) bei einer kostenlosen Testversion registrieren. Klicken Sie einfach oben links auf **Kostenlose Testversion**. Mit dieser kostenlosen Testversion erhalten Sie Zugriff auf alle Document Capture-Funktionen für bis zu 30 Tage oder bis die maximale Anzahl an Dokumenten der kostenlosen Testversion verarbeitet wurde. Nach dem Testzeitraum oder zu einem beliebigen Zeitpunkt während des Testzeitraums können Sie Document Capture ganz einfach abonnieren. Mit dem Abonnement sind Sie bereits startklar und können anschließend mit den gleichen Daten wie im Testzeitraum weiterarbeiten.

Informationen zum Kennenlernen von Document Capture finden Sie unter [Erste Schritte mit Continia Document Capture](Overview.md).

Wenn Sie Fragen haben, wenden Sie sich bitte an Ihren zertifizierten Continia-Partner. Weitere Informationen finden Sie unter [Finden eines Partners für Continia Document Capture](@DC-15).

## Ein Abonnement starten

Um die Vollversion von Document Capture dauerhaft und ohne Einschränkungen oder Bedingungen zu verwenden, können Sie jederzeit ein Abonnement abschließen und sofort beginnen. Auf diese Weise können Sie statt der Testversion sofort von der App profitieren. Die benutzerfreundliche Oberfläche erleichtert den Einstieg.

Um Document Capture zu abonnieren, wenden Sie sich einfach an Ihren Microsoft Dynamics 365 Business Central-Partner. Ihr Partner hilft Ihnen bei allen Fragen rund um das Abonnement und sorgt dafür, dass Sie schnell startklar sind. Die Unterstützung Ihres Partners bei der Aktivierung von Document Capture ist wichtig, unabhängig von der Version von Document Capture. So wird garantiert, dass Sie problemlos starten können und alles vorschriftsmäßig eingerichtet ist. Weitere Informationen finden Sie unter [Finden eines Partners für Continia Document Capture](@DC-15).

Ihr Partner steht Ihnen bei allen Fragen zur Verfügung. Wenn Sie mehr über Document Capture erfahren möchten, finden Sie unter [Erste Schritte mit Continia Document Capture](Overview.md) zahlreiche nützliche Informationen.

## Ein Testmodul aktivieren

Wenn Sie [über ein aktives Abonnement](#ein-abonnement-starten) für ein oder mehrere Module verfügen, können Sie ein anderes Modul 30 Tage lang völlig kostenlos in Ihrer Produktionsumgebung ausprobieren. Diese Option ist derzeit für das Einkaufsverträge-Modul verfügbar, wird aber in Zukunft möglicherweise auch für andere Module angeboten.

Um eine kostenlose 30-Tage-Testversion eines Moduls zu aktivieren, gehen Sie folgendermaßen vor:

1. Suchen Sie ({{search}}) nach **Continia Lösungsverwaltung** und wählen Sie dann den entsprechenden Link aus.
2. Wählen Sie in der FactBox **Abonnementmodule** rechts auf der Seite das entsprechende Modul.
3. Klicken Sie auf {{vertical-dots}}, um ein Dropdown-Menü für das Modul zu öffnen, und klicken Sie dann auf **Testversion starten**, um das Modul im Testmodus zu aktivieren.

Wenn Sie möchten, können Sie die Testversion innerhalb des 30-tägigen Testzeitraums jederzeit beenden. Folgen Sie dazu der Anleitung oben, wählen Sie im Dropdown-Menü stattdessen jedoch **Testversion beenden** aus.

> [!NOTE]
> Nach Ablauf des Testzeitraums wird die Testversion automatisch in ein kostenpflichtiges Abonnement umgewandelt, sofern Sie es nicht wie oben beschrieben aktiv kündigen.

Möglicherweise können Sie Ihre kostenlose 30-Tage-Testversion auch über eine [In-App-Benachrichtigung](@DC-131) aktivieren, falls das System Muster bei Belegen erkennt und vorschlägt, dass die Verwendung des Moduls für Sie von Vorteil sein könnte.

## Eine Sandbox einrichten

Suchen Sie nach einer sicheren Möglichkeit, Funktionen in Document Capture zu testen und auszuprobieren? Dann ist eine Sandbox möglicherweise das Richtige für Sie. Mit Sandboxes können Sie üben und experimentieren, ohne das Risiko einzugehen, wertvolle Daten zu verlieren oder den Produktionsablauf zu beeinträchtigen, und Sie können darin Belege völlig kostenlos verarbeiten.

Ihr Administrator kann zusätzlich zu Ihrer Produktionsumgebung eine oder mehrere Sandbox-Umgebungen einrichten, auch wenn Sie sich für eine Testversion registriert haben. Von den Sandbox-Umgebungen werden keinerlei Daten in Ihre Produktionsumgebung übertragen, sodass das Experimentieren mit Sandbox-Daten völlig risikofrei ist.

Um mit einer Sandbox zu beginnen, wenden Sie sich bitte an Ihren Business Central-Partner. Weitere Informationen finden Sie unter [Finden eines Partners für Continia Document Capture](Resellers and Partners/Find a Reseller.md).
