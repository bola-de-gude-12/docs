# Überblick über die Verwaltung elektronischer Dokumente in Document Capture

Der direkte Austausch elektronischer Dokumente mit Continia Document Capture ist der einfache Weg zur Optimierung Ihres Unternehmens. Immer mehr Behörden auf der ganzen Welt schreiben den elektronischen Austausch von Dokumenten vor, und dieser erfreut sich auch bei Business-to-Business-Organisationen immer größerer Beliebtheit, weshalb er ein wesentlicher Bestandteil von Document Capture ist.

Ihre Optionen zum Austausch elektronischer Dokumente in Document Capture hängen davon ab, welche elektronischen Dokumente und Dateien Sie austauschen möchten. Sie können Dateien problemlos austauschen, indem Sie sich mit dem Continia Delivery Network verbinden. Darüber hinaus können Sie Dateien an eine von Document Capture überwachte E-Mail-Adresse senden.

Das Continia Delivery Network ist ein Service, der in das Peppol eDelivery Network integriert ist und Ihnen so das Senden und den Empfang elektronischer Dokumente direkt in Microsoft Dynamics 365 Business Central ermöglicht. Continia ist ein zertifizierter Peppol Access Point, der die Integration Ihres Unternehmens in das Peppol eDelivery Network durch Drittanbieter überflüssig macht. Wenn Sie sich beim Continia Delivery Network registrieren, werden Sie automatisch als Empfänger im globalen Peppol eDelivery Network eingerichtet. Weitere Informationen zum Herstellen einer Verbindung mit dem Peppol eDelivery Network oder dem NemHandel-Netzwerk finden Sie unter [Das Continia Delivery Network einrichten](@DC-75).

Wenn Sie eine von Document Capture überwachte E-Mail-Adresse verwenden, können Sie elektronische Dokumente in vielen Formaten importieren. Die gängigsten Formate werden unterstützt, jedoch nicht alle. Eine Liste der unterstützten Formate finden Sie unter [Unterstützte Formate](@DC-55).
