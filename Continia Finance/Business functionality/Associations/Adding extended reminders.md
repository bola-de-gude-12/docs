# Adding Extended Reminders

If you use the Associations module, you can set reminders that take into account associations and customer/vendor links.

To create an extended  reminder:
1. Use the {{search}} icon, enter **Create Reminders** and select the related link.
2. Continia Finance offers the following additional options:
   * **Incl. Association** - if enabled, all customers of an association are printed. A prerequisite for this is that the selected customer is assigned to an association.
   * **Incl. Linked Vendors** - if enabled,  when reminding the customer, the vendor ledger entries of the corresponding customer/vendor link are also included in the reminder and offset.
   * **With Backlog** - if enabled,  a reminder will also be created if there is a credit balance in the customer/vendor link
