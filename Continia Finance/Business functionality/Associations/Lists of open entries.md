# Lists of Open Entries for Associations

With the Association module in Continia Finance, you can view, send and print lists in different formats with the open customer or vendor entries of your associations. You can customize the lists in a variety of ways.

## To send, view or print a list of open entries for associations

1. Choose the {{search}} icon, enter **Associations**, and then choose the related link.
1. Select the association for which you want to send, view or print a list of open entries.
1. In the action bar, select **Association - Open Entries**.
1. To customize your list, enable the required fields. This is a description of some of the most important fields:
   | **Field**                | **Description**                                              |
   | :------------------------ | :----------------------------------------------------------- |
   | Incl. Comments                      | Enabling this field means that any existing entry comments will be printed below the respective entry. |
   | Incl. Customers                      | This field is only active if you've selected a vendor association. Enabling this field means that linked customer entries will be included in the report. |
   | Incl. Vendors | This field is only active if you've selected a customer association. Enabling this field means that linked vendor entries will be included in the report. |
   | Print only Accounts with                      | Set a balance filter. The options are:<br><ul><li>Balance > 0</li><li>Balance &lt; 0</li><li>Balance = 0</li></ul> |
   | Subtotals                      | Include the subtotal for each customer or vendor |
   | Print Current Balance                      |  Include the current balance in addition to the balance on the reference date. |
1. Options for filtering for both vendor and customer ledger entries can also be found here:
   | **Field**                | **Description**                                              |
   | :------------------------ | :----------------------------------------------------------- |
   | Document Type                      | Filter by document type |
   | Document Date                      | Filter by document date |
   | Document No.                      | Filter by document No. |
   You can also add a custom filter.
1. When you're done, choose one of the options at the bottom of the page:
   | **Action**                | **Description**                                              |
   | :------------------------ | :----------------------------------------------------------- |
   | Send to...                      | Export the list to PDF, XML, Word document, Excel document (data and layout), or Excel document (data only) |
   | Print                     | Print the list |
   | Preview                     | Get a preview of the list |
