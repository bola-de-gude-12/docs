# Applying Partial Disposal of Assets

To execute a partial disposal for an asset, you can use the Continia Finance Partial Disposal feature available on the FA Card.

To activate the partial disposal feature:

1. Use the {{search}} icon, enter **Fixed Asset Setup**, and select the related link.
2. To activate the partial disposal feature, you need to specify a Reposting Journal Template and a Reposting Journal Name. On the action bar, select **Journal Templates** > **FA Journal Templates** to specify a journal. 
3. Similarly, on the action bar, select **Journal Templates** > **FA Reclass. Journal Templates** to specify a reposting journal. 

To execute a partial disposal:

1. On the FA Card, select **FA Partial Disposal**. 
2. In the **Insert New Fixed Asset via** field, enter the number of the new asset using the system's number series or manually enter a fixed number, which can also apply to a sub-asset.
3. On the **Reclassification** tab, you can specify the date and the entry type (e. g. reposting acquisition or regular depreciation), indicate if it is a partial disposal for an asset with quantities and enter the disposal quantity.
4. On the **Disposal via FA Journal** tab, you can indicate if it is a sale or a scrap. If you select *Scrapping*, the other fields can no longer be edited. If you have completed your entries, on the action bar, select **Reclassify Fixed Asset**. Once you have selected **Repost Asset**, a new asset will be created with the specified quantity and the formerly existing asset quantity will be reduced accordingly.
