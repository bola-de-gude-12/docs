# Using Fixed Assets Templates

Continia Finance allows you to use asset templates within the Fixed Assets Template Card to speed up the creation of new assets. These templates streamline asset creation and management by standardizing fields, depreciation books, and inventory settings. They offer a consistent framework, saving time and ensuring uniformity when handling assets.

## To create a fixed assets template

To add a fixed asset template:

1. Use the ![Search for page or report](/images/search_small.png) icon and search for **Fixed Asset Template Card**, then select the related link. 
2. Select the template code, set the depreciation start date, and name the asset. Use auto-generated or manual Fixed Asset numbers. Enable **Show Fixed Asset Card** for immediate access to asset details.

## To add a fixed asset using a template

You can add a fixed asset by using the Fixed Asset templates in various ways:

* Via the **Fixed Asset Gen. Journal** > **Create FA from Template.**
* Via the **Fixed Asset Journal** > **Create FA from Template.**
* Directly from a purchase invoice or purchase order > 
  **Create FA from Template**
* Via the **Fixed Asset Card** > Create FA from Template

To add a fixed asset using a template:

1. On the action bar, select **Create Fixed Asset** to generate the asset. 
2. On the **Create FA from Template** page, select the Fixed Asset template code to copy the data from. 
3. Go to the **Deprecation Starting Date** field and select the starting date of regular depreciation.
4. In the **Fixed Asset Description** field, enter the Fixed Asset name.
5. In the **Serial No./Dimensions field**, you can use a number series automatically generated by the system or specify the FA number manually.
6. After you specify the fields, go to the action bar and select **Create Fixed Asset** to create the Fixed Asset automatically. If you enable the **Show FA Card** checkbox, the system will open the Fixed Asset card.
