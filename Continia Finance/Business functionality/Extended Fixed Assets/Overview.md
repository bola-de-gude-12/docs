# Introduction to the Extended Fixed Assets module

Managing fixed assets efficiently is crucial for any organization's financial health. Continia's Extended Fixed Assets module offers robust tools within Microsoft Dynamics Business Central to streamline asset management processes.

* **Extended Fixed Assets Setup** - refer to the [Setting up fixed assets](@CF-09) article to learn more about the essential steps required to set up the module. It covers how to access the setup window and provides a detailed explanation of each field within it. 

* **FA Template** - refer to the [Using fixed asset templates](@CF-86) to learn more about how to define and utilize FA templates effectively. It discusses various methods for accessing and applying FA templates, offering insights into optimizing asset creation workflows.

* **FA Card Features** - The FA Card serves as a central hub for managing individual fixed assets. Refer to the [Adding a fixed asset](@CF-110) article to learn more about the additional fields and functionalities available within the FA Card.
