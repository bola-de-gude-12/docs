# Using Ledger Entry  Comments

You now have the option to use comments in all your ledger entries. This can be helpful if you want to explain entries that are still open, like when there are differences between payments and invoices that need more investigation.

> [!NOTE]
>
> The Using Ledger Entry Comments feature is part of the Essential module that you can use free of charge. Go to the [Introduction to the Essential Module](@CF-74) article to learn more about activating the module.

To add a comment:

1. Go to General Journals, select a journal line, and on the action bar, select **Continia Finance** > **Ledger Entry Comments**.

2. In the **Comment** column of the page that now opens, enter your comment.

3. If you now post the journal line, go to the **G/L Registers** and select the general ledger entry, on the right panel; in the **Entry Comments** section, you can see the comment you just added. Click the comment to open the **G/L Entry Card** and print the report. 

   > [!NOTE]
   >
   > You can add comments to a line anytime, including when posting, applying, or reopening.
