# Using Fast Posting Codes

To make it easier to assign business transactions to the correct G/L accounts, you can use the Fast Posting Codes feature. This feature allows you to quickly and efficiently assign accounts with just a keyword. It also adds the Fast entry code column to the Cash Book and the Extended Incoming Payments Ledger for even smoother entries.

> [!NOTE]
>
> The Using Fast Posting Codes feature is part of the Essential module that you can use free of charge. Go to the [Introduction to the Essential Module](@CF-74) article to learn more about activating the module. 

To add a fast posting code:

1. Select the {{search}} icon, enter **Fast Posting Codes**, and then select the related link.

2. Select **New**.

3. For the new Fast Posting Code, enter the following information:

   | Field               | Description                                                  |
   | ------------------- | ------------------------------------------------------------ |
   | Code                | Enter a useful code, for example *Utility Germany*.          |
   | Posting Description | Enter the posting text for the book sheet line. For example, *Water, Electricity, and Gas (DE)*. |
   | Account Number      | Select the account or multiple accounts to which you want to allocate the expense. |

If you access a general journal, the Fast Posting Code column will appear. Once you enter the code, the system will automatically add the account number and details.
