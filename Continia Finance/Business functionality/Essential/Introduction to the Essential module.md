# Introduction to the Essential Module

The Continia Finance Essential Module is a free add-on designed to enhance your financial management. With this module, you'll gain access to various features that improve clarity, efficiency, and accuracy in your financial tasks.

To start using the Essential Module, go to the **Continia Finance Setup** page. In the **General** section, you can enable features like keeping original currency codes, adding comments to ledger entries, using VAT key codes, quickly capturing data, improving line readability with section lining, and using colored hyperlinks for better interaction with reports.

To activate the Essential module's features:

1. On the Continia Finance Setup page, navigate to the **General** FastTab to enable the following features:
   * **Maintain Original Currency Code in G/L Entry** - when activated, this feature includes the original currency code and amount in G/L entries.
   * **Use Ledger Entry Comments** - when activated, you gain the ability to utilize comment lines in all ledger entries. This is especially useful if you wish to add explanations for open entries, such as discrepancies between payments and invoices requiring further investigation.
   * **Use VAT Key Codes** - activating VAT key codes allows for the automatic population of VAT fields.
   * **Use Fast Posting** - when activated, you can employ predefined accounting rules by utilizing a specific keyword in journals.
   * **Print Alternating Shading** - when you activate alternating shading, it shades alternate lines in a subtle grey, significantly improving the legibility and focus on individual lines.
   * **Colored Hyperlinks in Reports** - when activated, the master data records are underlined in distinct colors, serving as interactive links to their corresponding reports.
   * **CSV ports** - use CSV ports to easily import or export data to or from Business Central.
   
1.  Navigate to the **Balance Confirmation** FastTab, to activate the feature that helps you efficiently manage year-end financial tasks. You can generate balance confirmation letters for both customers and vendors. The inclusion of Report IDs for Customer and Vendor Balance Confirmations ensures clear and easily identifiable communication.
