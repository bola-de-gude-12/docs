# Setting the Last Posting Date

This article describes the **Set up Last Posting at** feature, which populates the last posting date across your G/L accounts, customers, and vendors.

This feature is useful when you first install Continia Finance, for example, or when you need an overview of customers or vendors with whom you haven't done business in a number of months or years.

When the **Set up Last Posting at** action is selected, Continia Finance goes through your G/L accounts, customers, and vendors, checks the date of the corresponding last posting, and populates the **Last Posting At** column accordingly. This column is visible in the **Chart of Accounts**, **Customers**, and **Vendors** lists.

   >[!NOTE]
   >This only needs to be done once. Afterward, the system automatically keeps this data up to date.

## To set the last posting date
To set the last posting date across your G/L accounts, customers, and vendors:

1. Choose the {{search}} icon, enter **Extended Financial Reports Setup**, and then choose the related link.
2. In the action bar, select **Set up Last Posting at**. Select **Yes** to initiate the process.

   >[!NOTE]
   >Depending on the number of the G/L accounts, customers, and vendors in your system, the process may take a while to finish.
