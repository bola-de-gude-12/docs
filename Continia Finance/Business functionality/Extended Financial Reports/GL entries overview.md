# G/L Entries Overview
This article describes the **G/L Entries Overview** in Continia Finance.

The **G/L Entries Overview** is part of the **Extended Financial Reports** module in Finance. It shows a combined list of G/L account balances based on various criteria, such as posting groups or department codes. If you use the [affiliation or filters features](@DC-113), the **G/L Entries Overview** can also display financial data from different companies.

To access the **G/L Entries Overview**:

1. Select **Extended Financial Reports** > **Reconcile**, then select **G/L Entries Overview**.
2. At first, the overview is empty. To populate it, select **Refresh Table** in the action bar.
3. On the **Cumulate Field Content** FastTab, select the data you want to gather in the overview and select **OK**.
4. The overview is populated accordingly, allowing you to sort the data based on the many columns presented.

By clicking a value under the **Amount**, **Debit Amount**, **Credit Amount**, or **VAT Amount** columns, you can see all entries in the related G/L account.

>[!TIP]
>To transfer your chart of accounts to an e-balance sheet, select **Create Mapping** in the action bar. This creates entries in the table for each combination of G/L account and tax group, cumulating for each VAT posting group. It's also possible to establish subaccount groups in the mapping table, which can then be exported to an Excel spreadsheet. To see the created G/L account mapping, select **Show Mapping** in the action bar.
