# Setting up the Factoring Module

Setting up the Factoring module is easy with the help of the Assisted Setup Guide. 

1. Before you start, make sure that you have the correct company open in Business Central. You must be in the company in which you want to manage factoring. This can easily be an operational company that participates in factoring itself. If factoring is only to be managed from within one company, it is advisable to select this company as the factoring company.
2. Use the {{search}} icon, enter **Continia Finance Factoring Assisted Setup Guide**, and select the related link.
3. Complete the steps in the wizard and select **Finish**.

To use the **Factoring Setup** page:

1. Use the {{search}} icon, enter **Factoring Setup**, and select the related link.
2. Go to the **Factoring Company** field and select the company that must collect the factoring information.
3. Go to the **Factoring Proposal Nos**, and select the number series for the proposals.
