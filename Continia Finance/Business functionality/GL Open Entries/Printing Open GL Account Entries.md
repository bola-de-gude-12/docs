# Printing Open G/L Account Entries 

The G/L Entries module provides a valuable feature that efficiently allows you to print a list of open entries for your General Ledger (G/L) Accounts. This functionality is especially useful when you need to review or export this information for further analysis. 

To print open G/L Entries or export them to Excel:

1. From the main menu, navigate to **G/L Open Entries** > **G/L Account** > **Open Entries**.

2. Now, it is possible to set multiple options. For example:

   | Field           | Description                                                  |
   | --------------- | ------------------------------------------------------------ |
   | Output in Excel | Activate to export the report contents to an Excel file.     |
   | Show Date       | Decide whether you want to display the posting date or the document date. |
   | No.             | Indicate the G/L accounts you want to consider in the report. |
   | Date Filter     | Specify the date filter that should be used. For instance, if you open this report with a To Date filter, the Open Entries List will show entries based on that date. For example, if the date filter is set to ..31.12.2019, all open G/L entries due as of 31.12.2019 will be displayed. |
   | Document Type   | Indicate the G/L entries to be included in the report.       |
   | G/L Entry tab   | Use the **G/L Entry tab** to indicate the G/L entries to be included in the report. |

4. Now you can either print the report or send it. Alternatively, if you want to export the report without printing it, you can select **Preview**.
