# Use Installments in Journal Lines

| Feature | Public preview | General availability online |
| --- | :-: | :-: |
| Use installments in journal lines | - | {{checkmark}} Mar 2025 |

## Business value

With this feature, users can set up installment payments in journal lines – just as the existing functionality for setting up installment payments for documents – making it easier and quicker to create installment plans.

## Feature details

This feature introduces UI elements in journal lines for easy access to and use of the installment payments functionality. The same functionality exists for documents already, and this is now extended to journal lines.

Read about setting up installment plans from journal lines in [Applying Installment Plans](@CF-132).
