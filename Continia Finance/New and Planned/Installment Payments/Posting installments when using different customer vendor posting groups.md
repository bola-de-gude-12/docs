# Ability to Post Installments when Using Different Customer/Vendor Posting Groups

| Feature | Public preview | General availability online |
| --- | :-: | :-: |
| Ability to post installments when using different customer/vendor posting groups | - |      {{checkmark}} Oct 2024 |

## Business value

This new feature ensures that different posting groups for customers and vendors are respected during transactions. Specifically, the posting group of an installment rate will align with the posting group of the corresponding invoice. This enhancement supports accurate accounting and reporting by adhering to the defined posting groups for all related transactions.

## Feature details

The feature modifies the handling of installment rates by using the posting group of the original invoice instead of the standard customer or vendor posting group.
