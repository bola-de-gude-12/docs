# Improved Check VAT Code Functionality 

| Feature | Public preview | General availability online |
| --- | :-: | :-: |
| Improved Check VAT Code functionality | - | {{checkmark}} Nov 2024 |



## Business value

The new enhancement to the **Check VAT Code** feature introduces greater flexibility and control for businesses by allowing them to tailor the validation process to their specific needs. Instead of running checks on all posting configurations, companies can now select which settings, such as the VAT Business Posting Group, are relevant for their operations.

## Feature details

Previously, the **Check VAT Code** functionality performed validations across all posting configurations defined in the Continia Finance Setup. With the enhanced feature, businesses can now define a predetermined range of posting settings for validation. For example, companies can configure the system to run checks solely on the VAT Business Posting Group while excluding other parameters.
