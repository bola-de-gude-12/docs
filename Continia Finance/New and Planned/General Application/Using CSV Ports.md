# Using CSV Ports

| Feature | Public preview | General availability online |
| --- | :-: | :-: |
| Using CSV ports | - |{{checkmark}} Oct 2024|

## Business value

This new feature enhances the data import and export capabilities in Business Central by allowing users to handle files with a .csv structure. This improvement simplifies the process of importing and exporting data, making it more efficient and user-friendly. All data types can be imported directly into any table – often a journal batch – streamlining data processing and entry.

The feature facilitates export of data to other solutions within the company and significantly speeds up data exchange compared to the standard Business Central features, improving overall efficiency.

## Feature details

You can define validation rules to ensure data accuracy and mapping rules to align .csv fields with Business Central fields, ensuring data is correctly translated between .csv files and Business Central, reducing errors and mismatches.

With this feature, you can also create dedicated ports for different data structures, allowing for specialized handling of various data types.

Read about how to use CSV ports for data import/export in [Using CSV ports](@CF-55).
