# Supported On-Premises Versions of Business Central
This page provides an overview of all supported *on-premises* versions of Microsoft Dynamics 365 Business Central. For information on what countries/regions Continia Finance is available in and what languages are supported for these on-premises versions, see [this article](@CF-128). For information on the types of support Continia does and doesn't provide, see [Scope of Support Services](@CF-185).

> [!NOTE]
>
> Every new major version of Continia Finance is only released for the last three major versions of Business Central.

The following table provides an overview of what versions of Continia Finance (CF) are compatible with the different versions of Business Central:
