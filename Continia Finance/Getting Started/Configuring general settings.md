# Configuring Continia Finance

To set up Continia Finance, activate the app in your company first. When you open Business Central for the first time post-installation, you'll see a notification in the Role Center to activate the app. 

## To set up Continia Finance
Before you run a Continia Finance module, you can determine on the general settings using the Setup page.

To add general settings:

1. Use the ![Search for page or report](/images/search_small.png) icon, search for **Continia Finance Setup,** and select the related link. 
2. On the **General** FastTab, you can, for example, activate the following:
   * **Use Section Lining** - when activated, every alternate line is subtly shaded in grey, significantly enhancing legibility and focus on individual lines.
   * **Colored Report-Hyperlinks** - activate this option to add a link to the master data record in the print preview.
4. On the **Balance Confirmation** FastTab, you can add settings for the free module that helps simplify year-end financial tasks. For more information, refer to the [Using Balance Confirmation](@CF-79) article.
5. On the **Modules** FastTab, you can view which Continia Finance modules are enabled in your license.

## The assisted setups

You can configure the Continia Finance modules through guided assisted setups. There are setups available for the various modules and for the general setup. For example, the Continia Finance Setup helps you configure the settings for balance confirmation letters. To learn more about the assisted setups that are available for Continia Finance, refer to [The Continia Finance Assisted Setups](@CF-91) article.

To initiate an assisted setup:

1. On the Role Center, on the action bar, select **Setup**.
2. On the **Continia Finance Setup** page, on the action bar, select **Assisted Setup** and select the module you want to run the setup for. Alternatively, select the {{search}} icon, enter **Assisted Setup**, navigate to the Finance section, and select the setup that you want to run.
