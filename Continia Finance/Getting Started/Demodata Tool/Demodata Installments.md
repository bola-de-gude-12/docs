# Installments Demo 

With the Installments module within the Continia Finance demo data tool, you can present how to create open entries with different due dates based on templates. You can use the template while or after posting the invoice.

The start a demo scenario for Installments:

1. Select the search icon, enter **Continia Demo Scenarios**, and select the related link.
2. Select **Installments: Sales invoice,** and on the action bar, select **Run Scenario**.
3. The demo scenario enables you to:
   * View a installment templates or create your own.
   * Create an invoice with an installment and post it.
   * View the posted entries.
