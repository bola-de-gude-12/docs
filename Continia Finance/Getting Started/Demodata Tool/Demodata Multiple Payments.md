# Demo data Multi-Level Payment Discount

With the Multiple Payment Discount demo data, you can generate a sales invoice with a multi-level payment discount. In addition, the necessary settings are made.

The start a demo scenario for Multiple Payment Discounts:

1. Select the search icon, enter **Continia Demo Scenarios**, and select the related link.
2. For the Associations module, there are two demo scenarios available:
   * **Multi-Level Payment Discount: Sales invoice** - generates a sales invoice with a multi-level payment discount.
3. Select the scenario, and on the action bar, select **Run Scenario**. The created sales invoice opens instantly.

   Payment Discount 3: 30 D - 1%
