# Essential demo

The Continia Demo Scenarios are designed to help you quickly and easily run demos in a sandbox environment and test the functionalities of Continia Finance in your own environment. There is a guided scenario available for all the different modules. 

To run a demo scenario:

1. Use the search icon, enter **Continia Demo Scenarios**, and select the related link. 
2. On the **Continia Demo Scenarios** page, you can select a specific demo and select **Run Scenario** for a guided tour. The following scenarios are available for Continia's Finance Essential modules:
   * Check VAT Keys
   * Maintain Orig. Currency Code
   * Fast Posting Codes
   * VAT Key Codes
   * Balance confirmation
   * Ledger Entry Comments
