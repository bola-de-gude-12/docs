# Treasury Demo

With the Treasury module within the Continia Finance demo data tool, you can present how you set up the Treasury module and view available treasury lists.

The start a demo scenario for Treasury:

1. Select the search icon, enter **Continia Demo Scenarios**, and select the related link.
2. Select Treasury: Create Views and on the action bar, select **Run Scenario**.
