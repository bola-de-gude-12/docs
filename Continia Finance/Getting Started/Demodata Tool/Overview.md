# Continia Finance Demo Scenarios

This document outlines the demo data tool that we offer. This tool is designed to help you quickly and easily run demos in a sandbox environment and test the functionalities of Continia Finance in your own environment. There are different ways to start and use the Demodata tool: You can give demonstrations of specific scenarios using the demo data, or you can follow the guided scenarios for the specific modules.

## Starting a demo scenario

If you want to start a demo scenario, here's how you do it:

1. Use the search icon, enter **Continia Demo Scenarios**, and select the related link.
1. On the **Continia Demo Scenarios** page, you can select a specific demo and select **Run Scenario** for a guided tour.
