# Support for the Business Central Universal Code Initiative

| Feature | Public Preview | General availability on-premises | General availability online |
| --- | :-: | :-: | :-: |
| Support for the Business Central Universal Code Initiative | - | {{checkmark}} Mar 2022 | - |



## Business value

It will be possible to continue to use all Payment Management on-premises features without paying Universal Code Initiative fees to Microsoft.



## Feature details

The Payment Management on-premises solution is already a cloud-optimized extension and will continue to work as usual, without the module for not cloud-optimized extensions.
