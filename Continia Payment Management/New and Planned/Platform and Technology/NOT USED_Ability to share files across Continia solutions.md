# Ability to share files across Continia solutions

##Business value

This feature will be useful for businesses that have several Continia solutions installed, which needs to import the same bank files into Business Central using direct bank communication. This could be a bank file with customer payments which is needed both by Continia Collection Management and Continia Payment Management, thus it will be possible for both solutions to import the file using direct bank communication.

##Feature details

The feature makes it possible to store the bank files in a temporary staging table, from where the files can be imported by several Continia solutions before it is deleted.
