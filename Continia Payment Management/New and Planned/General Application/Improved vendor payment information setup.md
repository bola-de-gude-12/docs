# Improved Vendor Payment Setup 

| Feature | General availability online | Public preview |
| --- | :-: | :-: |
| Improved vendor payment information setup | {{checkmark}} Oct 2022 | - |

## Business Value

The Vendor Payment Information Setup guide has been improved to equip users to set up their vendors quickly without needing support, thereby getting a head start with Payment Management. 




## Feature Details

The flow and user experience of the assisted Vendor Payment Information Setup guide has been enhanced, now called Vendor Payment Setup. We've carefully reviewed and updated all steps and interface texts, emphasizing essential configurations. We removed a redundant step and eliminated unnecessary fields for listed vendors. Additionally, we've revamped the user interface text to be more concise and user-friendly, ensuring clearer and more focused guidance for users.

To learn more, see the [Setting up vendor payment](@PM-5) article.
