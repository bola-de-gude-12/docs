# Improved Bank Account Setup

| Feature | Public preview | General availability online |
| --- | :-: | :-: |
| Improved Bank Account Setup | - | {{checkmark}} Oct 2022 |

## Business Value

With the improved assisted Bank Account Setup guide, Continia increases the guidance and support that users receive directly in the assisted setup guide. The users receive instructions and help specifically targeted the bank they are using, including any steps that must be completed before or after they run the assisted setup guide. The improvements will make it easier for customers to set up their bank accounts with direct communication and require less support. 

 


## Feature Details

The flow and user experience of the assisted Bank Account Setup guide have been updated. All the steps and user interface text of the assisted guide have been reviewed and updated to include all the relevant information. Regardless of the bank, the user will be assisted through the guide with clear and concise directions and additional links to Continia Docs or relevant external websites.

To learn more about the Bank Account Setup, see the [Setting up bank accounts](@PM-3) article.
