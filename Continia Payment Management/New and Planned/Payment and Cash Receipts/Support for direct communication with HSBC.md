# Support for Direct Communication with HSBC

| Feature | General availability on-premises | General availability online | Public preview |
| --- | :-: | :-: | :-: |
| Support for direct communication with HSBC | - | - | - |

> [!IMPORTANT]
>
> The release date for this feature is yet to be determined.



## Business value

To integrate with HSBC, besides manual communication and direct communication through [Yapily](@PM-268), Payment Management now also offers direct integration through [HSBC Connect](@PM-383).

## Feature details

This integration increases our support for HSBC, helping customers to send and receive bank files automatically between HSBC and Microsoft Dynamics 365 Business Central.

For more information on how to set up communication with HSBC, see the [HSBC overview article](@PM-285)
