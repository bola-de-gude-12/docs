# Support for File Import From Third-Party Payment Services

| Feature | General availability online | Public preview |
| --- | :-: | :-: |
| Support for file import from third-party payment services | {{checkmark}} Jan 2023 | - |

## Business value

With an extension of the importing services, we enable finance departments to follow the company cash flow with real-time updated customer payment transactions from third-party payment service providers (PSPs).

## Feature details

Payment Management will support file import from PSPs by making it possible to import the customer payment files directly from these parties into the cash receipt journal. The new file import feature is a journal update function that supports the import of cash receipt journals into Business Central. The feature will be developed as a generic and custom-friendly function that can import multiple PSP payment files. As the feature is non-specific to specific providers, this function can be used for various import scenarios and formats.

To learn more about working with PSPs, see the [Payment Service Providers](@PM-300) overview article.
