# Bellin

With Payment Management, you can use manual communication to send bank data between Bellin and Microsoft Dynamics 365 Business Central. 

This article lists the file format requirements for the manual communication method. To learn more about how to set up manual communication for Bellin, refer to the [Setting up bank accounts](@PM-3#to-set-up-bank-accounts-to-use-manual-communication) article.
