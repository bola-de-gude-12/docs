# Nordea Corporate Netbank

With Payment Management, you can use manual communication to send bank data between Nordea Corporate Netbank and Microsoft Dynamics 365 Business Central. 

Order the files listed in the table below, and then follow this link to see how you set it up:

- [Manual communication](@PM-3#to-set-up-bank-accounts-to-use-manual-communication)

> [!TIP]
>
> Bank system code: UNITEL02
