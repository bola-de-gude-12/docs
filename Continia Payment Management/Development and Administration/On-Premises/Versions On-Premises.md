# Supported On-Premises Versions of Business Central

The following table provides an overview of all Business Central versions that are compatible with Payment Management. 

> [!NOTE]
>
> Every new major version of Payment Management will only be released for the last three major versions of Business Central.

For more information about in which countries/regions Continia Payment Management is available and which languages are supported, see [The Countries and Languages](/Continia Payment Management/Development and Administration/Countries and languages.md) article. 

| Business Central version | Supported up to<br />Payment Management version |
| --- | --- |
| Microsoft Dynamics 365 Business Central 2019 release wave 2 (BC v15)<a href="#footnote-1"><sup>1</sup></a> | 3.3.0.0 |
| Microsoft Dynamics 365 Business Central 2020 release wave 1 (BC v16) | 3.3.0.0 |
| Microsoft Dynamics 365 Business Central 2020 release wave 2 (BC v17) | 4.3.1.0 |
| Microsoft Dynamics 365 Business Central 2021 release wave 1 (BC v18) | 5.3.1.0 |
| Microsoft Dynamics 365 Business Central 2021 release wave 2 (BC v19) | 5.3.1.0 |
| Microsoft Dynamics 365 Business Central 2022 release wave 1 (BC v20) | 6.2.4.0 |
| Microsoft Dynamics 365 Business Central 2022 release wave 2 (BC v21) | 7.2.2.0 |
| Microsoft Dynamics 365 Business Central 2022 release wave 2 (BC v22) | 24.2.6 |
| Microsoft Dynamics 365 Business Central 2023 release wave 1 (BC v23) | 25.5.0 |
| Microsoft Dynamics 365 Business Central 2024 release wave 1 (BC v24) | latest version |
| Microsoft Dynamics 365 Business Central 2024 release wave 2 (BC v25) | latest version |
| Microsoft Dynamics 365 Business Central 2025 release wave 1 (BC v26) | latest version |

<small style> <div class="footnotes">  <hr />  <ol>   <li id="footnote-1">The RTM version of Microsoft Dynamics 365 Business Central 2019 release wave 2 (BC v15) isn't supported for Continia Payment Management because Payment Management uses internal events, which are only supported in the BC-v15 CU1 and later cumulative updates.</li>  </ol> </div> </small>
