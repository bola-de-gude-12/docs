# Accessing the Source Code

Continia recognizes our partners' need for accessing our apps' source code for testing, integration, or problem-solving purposes. To access the source code, follow these steps:

1. Log in to [Continia PartnerZone](https://partnerzone.continia.com/), select **Downloads**, and download the relevant product package.
1. In the product package folder, navigate to **Apps** > **Source**.
1. To view and extract the source code zip files, open the relevant PM365 folder that matches your Microsoft Dynamics 365 Business Central (BC) version, including the correct cumulative update.


The **PM365** folder contains two different source code zip files:

* One for all base functionality, named "PM [version number] - [BC version number] (BC commercial name) Source.zip" (for example **PM 4.0.0.0 - 19 (BC 2021 Wave 2) Source.zip**)
* One for the DK version, named "PM DK [version number] - [BC version number] (BC commercial name) Source.zip" (for example **PM DK 4.0.0.0 - 19 (BC 2020 Wave 2) Source.zip**)

The base app is used for both DK and non-DK deployments, but note that for DK installations you'll need both the base app and the DK app.
