# Continia Expense Mobile App notifications

Automatic notifications for the Continia Expense Mobile App ensure expense users are always up to date with all important changes to document statuses.

Enable this feature from the start by accepting the push notifications permission prompt when installing the Continia Expense Mobile App. You can always disable notifications via the settings in the app.

Notifications of status changes are for all expense documents: expenses, mileage expenses, per diem expenses, and expense reports – with the amount displayed in local currency.

When a status changes for one or more of your expense documents while you're using the app, you'll get an in-app notification prompting you to synchronize with Continia Online so you can view all of the latest changes.

\


The following table provides an overview of when notifications are sent to expense users.

| Expense users will get a notification when:                       |
| ----------------------------------------------------------------- |
| An expense / mileage / per diem / expense report is **created**   |
| An expense / mileage / per diem / expense report is **updated**   |
| An expense / mileage / per diem / expense report is **approved**  |
| An expense / mileage / per diem / expense report is **completed** |
| An expense report preapproval is **approved**                     |
| An expense report preapproval is **rejected**                     |
