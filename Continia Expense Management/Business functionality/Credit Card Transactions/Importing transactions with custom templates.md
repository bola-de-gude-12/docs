# Importing transactions with custom templates

Before you can import credit card transactions from your bank into Expense Management, you must set up a template. If you haven’t already done this, you can set up one using [Setting up transaction templates](@EM-42).

**Importing transactions**

To import transactions:

1. Search for {{search}} **Bank Transactions**.
2. On the action bar, click **Transaction Import Journal** and select the transaction template you want to use.
3. On the action bar, click **Import From File**, and select the file you want to import from.
4. Check all the imported lines to verify that all required data has been imported correctly, and delete any unwanted lines.
5. When you’re satisfied with the content of the journal, click **Approve and Transfer**. The content of the journal will now be transferred to the Bank Transaction Inbox, and if validation is successful, credit card transactions will be created in Expense Management.
