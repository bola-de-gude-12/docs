# Viewing Multiple Comments in Expense Reports

Each document uploaded to an expense report has room for comments to be added by the user and the admin. To make your Expense Management workflow more efficient, you can view all of the comments from these multiple documents in one place.

## Viewing comments in expense reports
You can view the comments from each of the documents in a report in the same place on the Expense Report page.

**To view all of the comments of an Expense Report:**

1. On the **Expense Report** page, go to the action bar.
2. Select the **Comment** tab. Here you can view all of the comments for the expense report.
3. For a deeper dive into this functionality, see the Improved visibility of comments in Expense Reports video.
