# Enhanced bulk user import in the Continia User Setup

| Feature | General availability | Public preview |
| --- | :-: | :-: |
| Import multiple users at the same time | {{checkmark}} Oct 2025 | {{checkmark}} Sept 2025 |

## Business value
This feature streamlines the process of importing large numbers of users into the Continia User Setup. By removing the confirmation dialog that previously appeared for each user creation, the bulk import process is now faster and more efficient. The result is a huge time saver for administrators and a reduction in manual interaction during setup.

The improved workflow minimizes interruptions, enhances user management scalability, and ensures a smoother onboarding experience for organizations handling large user volumes.

## Feature details

The bulk user import functionality in the Continia User Setup has been optimized to improve efficiency and usability. This brings:

- A simplified import process - The confirmation dialog that used to appear for each user during bulk import has been removed, allowing the system to process large user lists seamlessly without requiring individual manual confirmations.
- Increased efficiency - By eliminating the need for repeated user interaction, the import process is significantly faster, making it ideal for organizations onboarding large numbers of users at the same time.
