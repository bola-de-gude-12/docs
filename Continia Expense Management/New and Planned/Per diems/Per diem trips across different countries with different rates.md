# Per Diem Trips Across Different Countries with Different Rates

| Feature | General availability on-premises | General availability online | Public preview |
| --- | :-: | :-: | :-: |
| Per Diem Trips Across Different Countries with Different Rates | {{checkmark}} Sep 1, 2021 | {{checkmark}} Oct 1, 2021 | - |

## Business value
Managing per diem trips through countries with different rates has become much easier with the addition of this feature because all trip information can be added to one per diem instead of having to make a per diem for each country.

## Feature details
With support for per diem trips through different countries, expense users will be able to create one per diem trip covering trips through multiple countries. In the Expense App and the Expense Portal, expense users will have the option of selecting via points whenever they create a per diem.
