# Enhanced Microsoft 365 Sign-In Management

| Feature | Public preview | General availability |
| --- | :-: | :-: |
| Enhanced Microsoft 365 sign-in management | {{checkmark}} Sep 19 2024 | {{checkmark}} Oct 2024 |

## Business value

Building on our existing support for Microsoft 365 sign-in, administrators in Business Central can now view sign-in types and mandate the use of Microsoft 365 for signing into the Continia Expense App and Portal.

## Feature details
**Sign-in type visibility:** Administrators can now view detailed information about the types of sign-in being used by users, including Microsoft 365 and other authentication methods.
**Mandatory Microsoft 365 sign-in:** Administrators can enforce the use of Microsoft 365 credentials for accessing the Continia Expense App and the Continia Expense Portal, ensuring a unified and secure sign-in experience.
**Welcome email notification:** Users will now receive a welcome email that clearly instructs them to use their Microsoft 365 credentials for sign-in.

For more information, refer to the **Expense Management Login Type** field description in the [Expense User Setup for Expense Management](@EM-36) article.
