# Company policies

| Feature | General availability on-premises | General availability online | Public preview |
| --- | :-: | :-: | :-: |
| Company policies | {{checkmark}} Mar 1, 2021 | {{checkmark}} Apr 1, 2021 |-|

## Business value
"Company policies " in this context means to define thresholds – specific amounts – which automatically trigger various actions when reached. In doing so, they contribute to the automation of a number of Expense Management processes, thereby simplifying things for admins and expense users alike.

## Feature details
The feature will make it possible to set limits for groups and individual users based on company polices. When limits are set, Continia Expense Management will be able to automatically:
* Approve expenses below the limit.
* Refund the document up to the limit.
* Add a warning comment when above the limit.
