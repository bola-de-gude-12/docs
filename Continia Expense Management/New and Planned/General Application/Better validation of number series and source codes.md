# Better Validation of Number Series and Source Codes

| Feature | Public preview | General availability |
| --- | :-: | :-: |
| Better validation of number series and source codes | {{checkmark}} Mar 2023 | {{checkmark}} Apr 2023 |

## Business value
More accurate validation of number series and source codes when installing Expense Management.

## Feature details
Number series and source codes will be created after the assisted setup guide has imported the Expense Management setup data. This change ensures that the Expense Management setup only contains valid codes for source codes and number series, compared with previous versions where mismatches might sometimes occur.
