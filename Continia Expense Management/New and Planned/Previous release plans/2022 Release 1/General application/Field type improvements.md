# Field Type Improvements

| Feature | General availability on-premises | General availability online | Public preview |
| --- | :-: | :-: | :-: |
| Field Type Improvements | {{checkmark}} Mar 2022 | {{checkmark}} Apr 2022 | - |

## Business value
A number of improvements concerning the whole area of field types will make it easier and more user friendly to work with field types, which will save companies time and effort in areas as diverse as custom setups and training in Expense Management. 

## Feature details
The user will primarily work with field types on the Configured Fields page, where the setup will be continuously validated, making the user journey more intuitive.
