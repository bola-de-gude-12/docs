# Description Field Extension

| Feature | Public preview | General availability |
| --- | :-: | :-: |

## Business value
More flexibility when filling out description fields in Expense Management.

## Feature details
Most of the description fields in Expense Management will be extended from 50 to 100 characters throughout the application to keep consistent with Business Central.
