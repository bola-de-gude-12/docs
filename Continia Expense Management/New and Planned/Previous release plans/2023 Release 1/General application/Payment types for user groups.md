# Payment Types for User Groups

| Feature | Public preview | General availability |
| --- | :-: | :-: |
| Payment types for user groups | {{checkmark}} Mar 2023 | {{checkmark}} Apr 2023 |

## Business value
This feature makes it faster and easier to set up and use payment types that apply to many people, thereby also expanding the application area for user groups.  

## Feature details
With this improved feature in Expense Management, payment types can be set up for user groups.

You can find information on how to set up and assign payment types in the main article [Setting up Payment Types](@EM-81)
