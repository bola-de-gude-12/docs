# Payment Type UI Improvements

| Feature | Public preview | General availability |
| --- | :-: | :-: |
| Payment type UI improvements | {{checkmark}} Mar 2023 | {{checkmark}} Apr 2023 |

## Business value
The improvements in the payment type UI means that it will be easier and faster to create payment types.

## Feature details
Fields that are dependent on other fields higher up in the hierarchy won't be displayed if said fields higher up in the hierarchy are not enabled.

You can find information on how to set up and assign payment types in the main article [Setting up Payment Types](@EM-81)
