# Support for Automatic Allocation Using Distribution Codes

| Feature | Public preview | General availability |
| --- | :-: | :-: |
| Support for automatic allocation using distribution codes | - | {{checkmark}} Dec 2023 |

## Business value
Quick and easy allocation of expenses. 

## Feature details
Using distribution codes, allocations will be automatic based on percentages specified in the setup. This feature will be available on all expense types.



You can find information on how to set up automatic allocation using distribution codes in the article [Automatically Split and Allocate Using Amount Distribution](@EM-216)
