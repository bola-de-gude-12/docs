# Support for GST/GST-Free Expenses in Australia

| Feature | Public preview | General availability |
| --- | :-: | :-: |
| Support for GST/GST-free expenses in Australia | - | {{checkmark}} Dec 2023 |

## Business value
Expense Management will automatically create allocation lines based on GST calculations, saving time for the bookkeeper.

## Feature details
With this new feature in Expense Management, it will be easy and quick for admins to process GST/GST-free expenses in Australia. Expenses will be automatically allocated based on the GST posting setup tax percentage. The AI Receipt Scanner in the Expense Mobile App will also detect GST amounts on receipts, making it easy for expense users to report it. For more details, see [Splitting expenses for GST-exempt parts (AU)](@EM-215).
