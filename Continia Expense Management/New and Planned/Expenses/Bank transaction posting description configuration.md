# Bank transaction posting description configuration

| Feature | General availability | Public preview |
| --- | :-: | :-: |
| Bank transaction posting description configuration | {{checkmark}} Oct 2025 |{{checkmark}} Sept 2025|

## Business value
You can now customize how bank transaction postings appear in the general ledger. Instead of generic descriptions, you can define a template that includes key transaction details — resulting in faster reconciliation and improved clarity for audits. 

## Feature details
You can set up a custom posting description template directly from the Expense Management Setup page, similar to other document posting descriptions (such as Expense or Mileage). The template uses placeholders that are dynamically replaced with actual transaction data at the time of posting. For example, data such as employee name, amount, or business name.
