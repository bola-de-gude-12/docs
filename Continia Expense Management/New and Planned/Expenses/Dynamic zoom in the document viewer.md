# Dynamic Zoom in the Document Viewer

| Feature | Public preview | General availability |
| --- | :-: | :-: |
| Dynamic zoom in the document viewer | - | Oct 2023 |

## Business value

With this feature, you'll no longer have to zoom and adjust document images in the document viewer yourself for each individual document that's submitted. Continia Expense Management will automatically do it for you, thereby saving you from a lot of hassle and ensuring that images are always displayed in full, even if you resize the document viewer.

## Feature details

For each expense submitted by an expense user via the Continia Expense App or the Continia Expense Portal, the Expense Management document viewer will automatically zoom and adjust the document image to ensure the best fit. This is particularly helpful if you generally receive images of varying sizes, as this typically requires a lot of manual zooming and rearranging in order for the images to look good on the screen.

If you change the size of the document viewer, it will automatically zoom and center the image, so that it's displayed in full. You'll always be able to zoom and readjust the image manually yourself though, if necessary.
