Before you can start using Continia {param1}, you must install it in your Microsoft Dynamics 365 Business Central tenant. If you’re not completely sure about using this product, you can sign up for a free 30-day trial without any worries. This allows you to explore the features before deciding on a subscription. You’re free to cancel the trial anytime if you find that it doesn’t meet your needs.

Continia {param1} is available from Microsoft AppSource, and there are various ways to install it. For more information, see [Customizing Business Central Using Extensions](https://docs.microsoft.com/dynamics365/business-central/ui-extensions).

> [!IMPORTANT]
> To install {param1}, you must be assisted by an approved Continia partner. During the activation process, a person from your partner’s organization will be required to log in using their own personal user credentials.
> 
> Also, in order to install extensions from AppSource or add per-tenant extensions, you must have the right permissions. You must either be a member of the D365 EXTENSION MGMT user group, or you must have the D365 EXTENSION MGMT permission set.

To install {param1}, follow these steps:
1. Choose the {{search}} icon, enter **Continia {param1}**, and then choose the related link under **Get from Microsoft AppSource**.
1.	Choose **Free Trial** and complete the installation. For more information, see [Customizing Business Central Using Extensions](https://docs.microsoft.com/dynamics365/business-central/ui-extensions).
1.	Optional: To follow the installation status, choose the {{search}} icon, enter **Extension Management**, and then choose the related link.
1.	When the {param1} installation is complete, you’ll receive a notification in the Role Center that {param1} has been installed.
1.	In the notification, choose **Activate now**.
1.	Follow the steps in the **Activation Guide**.
1.	When the guide is finished, you’ll see the **Continia {param1} Assisted Setup**, which will help you set it up.
