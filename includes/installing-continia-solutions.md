# Installing Continia Solutions

This article provides guidance on installing Continia solutions, such as Continia Document Capture and Continia Sustainability in Business Central. For general information on installing apps, see [Manage AppSource apps](https://learn.microsoft.com/en-us/dynamics365/business-central/admin-manage-appsource-apps) on Microsoft Learn.

> [!IMPORTANT]
>
> Microsoft is deprecating the Extension Marketplace in Business Central 2024 Wave 2 (BC25) and replacing it with the [Microsoft AppSource Apps page](https://learn.microsoft.com/en-us/dynamics365/release-plan/2024wave1/smb/dynamics365-business-central/experience-new-embedded-appsource-marketplace-page). This change affects how you install extensions from AppSource.

## To install Continia Solutions via appsource.microsoft.com

If you’re thinking about subscribing, you can try out most Continia apps for free for a 30-day trial period. To do this, go to [Microsoft AppSource](https://appsource.microsoft.com/) and search for your Continia solution. Choose the related link, and then select **Free trial** in the upper-left corner. This will install the app, which is then ready for activation and setup using assisted setup guides.

>[!NOTE]
>Sign in to appsource.microsoft.com using the email account that you use for Business Central. 

## To install Continia Solutions from Business Central

You can access AppSource directly from Business Central. On the Extension Management page, you can view the apps currently installed and open the Microsoft AppSource Apps page to explore Business Central apps available on AppSource. If you choose the **View AppSource** action, you're taken to [AppSource.microsoft.com](https://go.microsoft.com/fwlink/?linkid=2081646). For more information, see [Manage AppSource apps](https://learn.microsoft.com/en-us/dynamics365/business-central/admin-manage-appsource-apps).

To install Continia solutions via Business Central:

1. Use the {{search}} icon, search for Extension Management, and then choose the related link.
2. On the action bar, select **AppSource Gallery**. This action replaces the former Extension Marketplace action.
3. Search for the Continia app. For example, search for Continia Sustainability, select the app and select **Install From AppSource**.
4. Select **Authorize Installation**.
5. A new browser tab will open. Select **Get it now** to begin the installation process.
6. Select the environment (Production or Sandbox) and select **Install** on the confirmation page. 
7. On the **Extension Installation** page, complete the installation by selecting **Install** again.
8. Once complete, close the installation pages and return to the **Extension Management** page to verify  that the app appears in the list of installed extensions.
