# Upgrading NAV/Business Central with Document Capture installed

If you use Continia Document Capture in an old version of Microsoft Dynamics NAV/Business Central and want to upgrade NAV/Business Central to a later version, you must also upgrade Document Capture and/or carry out related tasks – depending on what version you use.

>[!NOTE]
>The last FOB version of Document Capture is 2024 R2 (25.00). On-premises users can install the app version of Document Capture 2025 R1 (26.00).

The articles below provide guidelines as to what steps you must take in such scenarios. To access the guidelines that are relevant to you, pick the version of Business Central that you want to upgrade to:

* [Business Central April 2019 (v14, FOB-based)](@DC-102)
* [Business Central 2020 release wave 2 (v17)](@DC-97)
* [Business Central 2021 release wave 1 (v18)](@DC-98)
* [Business Central 2021 release wave 2 (v19)](@DC-99)
* [Business Central 2022 Release Wave 1 (v20)](@DC-100)
* [Business Central 2022 Release Wave 2 (v21)](@DC-101)
* [Business Central 2023 Release Wave 1 (v22)](@DC-132)
* [Business Central 2023 Release Wave 2 (v23)](@DC-160)
* [Business Central 2024 Release Wave 1 (v24)](@DC-203)
* [Business Central 2024 Release Wave 2 (v25)](@DC-235)

> [!NOTE]
> To upgrade to a different version than the ones mentioned in the guides above, use the information provided in the guides (including all links) to infer what to do. For example, to upgrade to Microsoft Dynamics NAV 2018, see [Business Central April 2019 (v14, FOB-based)](Business Central April 2019, v14.md). Although that particular upgrade isn't specified directly in this guide, the guide provides all the information you need to carry out the upgrade.
