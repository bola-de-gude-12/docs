# Supported upgrade paths for Continia Document Capture (FOB)
This article describes how to upgrade Continia Document Capture for FOB versions of Microsoft Dynamics 365 Business Central. For information on how to upgrade Document Capture for app versions of Business Central, see [Upgrading Continia Document Capture for app versions of Business Central](@DC-6).

> [!TIP]
> You can download the Document Capture installation packages and upgrade files for FOB versions on the [Continia PartnerZone](https://partnerzone.continia.com/downloads/) (only available to partners).

Document Capture (DC) is available in many different versions. If you have an older version, you can choose to update to the latest one – though it may not be possible to upgrade directly, so you may have to update your version of Microsoft Dynamics NAV/Business Central as well.

It's recommended that you upgrade if you:

* Use a version of Document Capture that's no longer supported, and you encounter an error that hasn't been corrected for that version.
* Want to use functionality that's only available in newer versions than the one you're using.

In all other scenarios, it's strongly recommended to install service packs. Additionally, always install the latest service pack to ensure that your version is updated – and that any enhancements and bug fixes are implemented.

> [!NOTE]
> There's a possible add-in conflict when using Document Capture and Continia Document Output – and by extension Continia Expense Management – and only upgrading one of these products. For more information, see [Upgrade notes concerning a possible add-in conflict](https://continia.zendesk.com/hc/da/articles/4414689784210) (only available to Continia Partners).
