# The Customers, Employees, and Vendors Categories

The Customers, Employees, and Vendors categories – **CUSTOMER**, **EMPLOYEE**, and **VENDOR** – enable Continia Document Capture to automatically attach documents to the related person or business in Microsoft Dynamics 365 Business Central. Processed documents in these categories can automatically recognize the person or business related to these documents, which are then attached accordingly upon registration.

Registered documents are fully searchable within Business Central, allowing for quick and easy retrieval – while documents added via Drag & Drop aren’t searchable as they aren’t OCR-processed.

>[!TIP]
>To use Document Search, choose the {{search}} icon, enter **Document Search**, and choose the related link. You can then enter the desired keywords, with or without additional search parameters.

>[!NOTE]
>If you don't need Document Search, you can drag and drop documents into the FactBox of a person or entity. To do so, open the related page (such as **Employees**), select the desired person or entity, and drag the document from your File Explorer or desktop into the **Documents** FactBox.

## To attach a document to a customer

1. Search ({{search}}) for and select **Document Categories**.
2. Select the **CUSTOMER** code to open the document journal for customer documents.
3. In the list of documents, select the document you want to attach to a customer.
4. Check if the right customer has been recognized. If something is wrong or missing, correct this manually.
5. On the action bar, click **Home** > **Register** to complete the registration of the document.

## To attach a document to an employee

1. Search ({{search}}) for and select **Document Categories**.
2. Select the **EMPLOYEE** code to open the document journal for employee documents.
3. In the list of documents, select the document you want to attach to an employee.
4. Check if the right employee has been recognized. If something is wrong or missing, correct this manually.
5. On the action bar, click **Home** > **Register** to complete the registration of the document.

## To attach a document to a vendor

1. Search ({{search}}) for and select **Document Categories**.
2. Select the **VENDOR** code to open the document journal for vendor documents.
3. In the list of documents, select the document you want to attach to a vendor.
4. Check if the right vendor has been recognized. If something is wrong or missing, correct this manually.
5. On the action bar, click **Home** > **Register** to complete the registration of the document.
