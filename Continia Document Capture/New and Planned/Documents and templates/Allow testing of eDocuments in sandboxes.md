# Allow testing of eDocuments in sandboxes

| Feature | Public preview | General availability online |
| --- | :-: | :-: |
| Allow testing of eDocuments in sandboxes | {{checkmark}} Mar 2025 | {{checkmark}} Apr 2025 |

## Business value

This feature will allow you to use Continia’s eDocument-related features within a sandbox, allowing you to test customizations and flows without affecting your production environment.

## Feature details

When this feature is implemented, the sending and receiving of eDocuments will be supported in test environments. Although the onboarding of a participation won’t be possible from a sandbox, you'll be able to copy a participation and mark it as a test participation from the database when the sandbox is created.

For more information, see [Setting up Continia eDocuments](@DC-179).
