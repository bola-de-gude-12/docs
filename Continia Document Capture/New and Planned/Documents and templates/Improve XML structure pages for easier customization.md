# Improve XML structure pages for easier customization

| Feature | Public preview | General availability online |
| --- | :-: | :-: |
| Improve XML structure pages for easier customization | {{checkmark}} Mar 2025 | {{checkmark}} Apr 2025 |

## Business value

This feature simplifies the customization of the XML structure of eDocuments, such as adding custom fields or changing default values. A simpler interface will allow you to more easily and quickly make the changes you need.

## Feature details

Although it’s already possible to alter the XML structure of eDocuments, the interface can be cumbersome – especially when used in small screens. This update will make it easier and more user-friendly.

For more information, see [Setting up Continia eDocuments](@DC-179).
