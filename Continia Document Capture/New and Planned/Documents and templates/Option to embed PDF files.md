# Option to embed PDF files

| Feature | Public preview | General availability online |
| --- | :-: | :-: |
| Option to embed PDF files | {{checkmark}} Sep 2025 | {{checkmark}} Oct 2025 |

## Business value

Whether you use it as a failsafe or as an additional layer of convenience, this feature will allow you to attach files such as PDFs to an outgoing eDocument.

## Feature details

You'll have the option to embed the PDF generated by standard Business Central (or any other attachments) into an eDocument, before sending it via the Continia Delivery Network.

For more information, see [Setting up Continia eDocuments](@DC-179#to-set-up-continia-edocuments).
