# Continia eDocuments Support for OIOUBL 3.0

| Feature | Public preview | General availability online |
| --- | :-: | :-: |
| Continia eDocuments support for OIOUBL 3.0 | - | N/A |

> [!NOTE]
> The release of this feature depends on the official release of OIOUBL 3.0, which the Danish Business Authority (Erhvervsstyrelsen) is currently working on.

## Business value

With [Continia eDocuments](@DC-310) support for the upcoming OIOUBL 3.0 format, you'll enjoy the benefits of the latest advancements and standards within electronic document processing.

## Feature details

The Danish Business Authority (Erhvervsstyrelsen) is currently working on releasing the new version of the OIOUBL format – OIOUBL 3.0.

Continia eDocuments will support this format as soon as the format is officially released.
