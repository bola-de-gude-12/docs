# Educational Page to Improve User Knowledge of eDocuments and Formats

| Feature | Public preview | General availability online |
| --- | :-: | :-: |
| Educational page to improve user knowledge of eDocuments and formats | - | Oct 2025 |

>[!NOTE]
>This feature will be released in Document Capture 2025 R2.

## Business value

By providing more data about the fields that are available on different Continia eDocuments pages, this feature enables you to better understand the electronic format you're working with. In doing so, it can also improve your ability to handle validation errors.

## Feature details

A new page will be introduced, educating users about the fields of the different electronic formats that are supported. The page will describe each format and list all possible fields along with descriptive details such as example values and indications of whether the fields are mandatory, if they have validation rules, etc.
