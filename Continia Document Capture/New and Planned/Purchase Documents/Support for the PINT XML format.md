# Support for the PINT XML Format

| Feature | Public preview | General availability online |
| --- | :-: | :-: |
| Support for the PINT XML format | - | - |

> [!IMPORTANT]
> As the PINT XML format hasn't been released, Continia can't support it yet. When the format is released, we'll revisit this feature and decide whether to support the format in Continia Document Capture.

## Business value
With more XML document formats supported, you can receive invoices and credit memos from more countries without having to perform any manual template modifications or similar.

## Feature details
Support for the PINT XML document format (the future international Peppol format) will be added if the format is launched in time for the next Continia release.
