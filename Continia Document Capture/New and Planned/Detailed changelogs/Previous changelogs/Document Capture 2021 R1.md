# Detailed Changelog for Continia Document Capture 2021 R1

This article lists all new features and bug fixes for each version of Continia Document Capture 2021 R1.

> \[!IMPORTANT] This version of Document Capture is no longer supported. As a consequence of this, Continia will no longer correct any errors appearing in this version, and no further service packs relating to the version will be released. For supported versions of Document Capture, see [Software Lifecycle Policy and Continia Document Capture On-Premises](../../../../Continia%20Document%20Capture/New%20and%20Planned/Detailed%20changelogs/Previous%20changelogs/@DC-13/).

> \[!TIP] As a Continia partner, you can be notified of new Document Capture versions and service packs whenever they're released. To sign up for this service, go to [this page](https://continia.zendesk.com/hc/en-us/articles/360005355079-Receive-a-notification-upon-new-Document-Capture-releases) in the Continia PartnerZone (only available to partners).
