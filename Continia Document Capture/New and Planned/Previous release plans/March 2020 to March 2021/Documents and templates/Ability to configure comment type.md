# Ability to Configure Comment Type

| Feature | General availability on-premises | General availability online | Public preview |
| --- | :-: | :-: | :-: |
| Ability to configure comment type | {{checkmark}} Oct 1, 2020 | {{checkmark}} Sep 1, 2020 | - |

## Feature details
Currently, the batch registration process skips documents that have **Warning** comments. For certain comments appearing on the document page, admin users can now configure if those particular comments should be of type **Information** or **Warning**, thereby controlling which comments are able to block the batch registration process.
