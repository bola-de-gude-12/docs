# User-Specific List of Approvers When Forwarding Approval Requests

| Feature | Public preview | General availability online |
| --- | :-: | :-: |
| User-specific list of approvers when forwarding approval requests | {{checkmark}} Mar 2023 | {{checkmark}} Apr 2023 |

## Business value

By enabling you to narrow down the number of available approvers per user, this feature will make the process of forwarding documents for approval much more manageable.

## Feature details

Currently, when you want to forward a document to an approver, you're presented with a list of every single approver available, which can make it difficult to find the one you're looking for. To ease this process, this feature will make it possible for you to specify exactly which approvers each individual user should be able to forward approval requests to. This means that users will now be presented with a limited and more useful list of approvers to choose between when forwarding documents for approval.

For more information, see [Creating User-Specific Lists of Approvers for Approval Forwarding](@DC-138).
