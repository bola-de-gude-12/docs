# Bulk Deletion of Master Template Fields

| Feature | Public preview | General availability online |
| --- | :-: | :-: |
| Bulk deletion of master template fields | {{checkmark}} Mar 2023 | {{checkmark}} Apr 2023 |

## Business value

With the introduction of this feature, you'll be able to delete master template fields from multiple related templates in a single operation. This means that you'll no longer have to go through the time-consuming process of deleting irrelevant fields manually in template after template. Continia Document Capture will do it for you in one go, thereby saving you time and hassle.

## Feature details

Whenever you delete a master template field, you'll be asked if you also want to delete the field from all related templates. If you select **Yes**, the field will then automatically be deleted from all related templates, provided that it hasn't already been used to capture values in any of these.
