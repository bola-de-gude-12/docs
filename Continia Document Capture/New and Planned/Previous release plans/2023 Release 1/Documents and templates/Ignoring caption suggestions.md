# Ignoring Caption Suggestions

| Feature | Public preview | General availability online |
| --- | :-: | :-: |
| Ignoring caption suggestions | {{checkmark}} Mar 2023 | {{checkmark}} Apr 2023 |

## Business value
Using this feature, you'll be able to let Continia Document Capture know that it should leave out certain captions whenever it suggests field captions for master templates. As a result, you'll no longer have to exclude such irrelevant captions manually yourself every time you run the suggestion routine, meaning that you'll simplify your workflow and save significant amounts of time.

## Feature details
When you ask Document Capture to suggest captions for master template fields using the **Create Suggestions** action, this feature will enable you to ignore some or all of the suggested captions. To do this, simply select **Ignore** for each caption that you want Document Capture to leave out of consideration. Any ignored captions won't be suggested anymore when you run the **Create Suggestions** routine again in the future.

For more information, see [Adding Captions to PDF Master Templates](@DC-117).
