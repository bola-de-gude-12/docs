# Improved Order Matching Using eDocuments Data

| Feature | Public preview | General availability online |
| --- | :-: | :-: |
| Improved order matching using eDocuments data | {{checkmark}} Sep 2024 | {{checkmark}} Oct 2024 |

## Business value

When Continia eDocuments data is included in the document matching process, it will be easier to match purchase orders against purchase invoices, and a higher level of automation will be achieved. The eDocuments data will also be used to check the accuracy of the purchase invoices, as each purchase invoice line must match its corresponding purchase order line completely due to the 1:1 relationship between the two.

## Feature details

Additional data from Continia eDocuments will be used in the matching of purchase orders against purchase invoices, which will significantly improve the process.
