# Ability to handle payment means in Continia eDocuments

| Feature | Public preview | General availability online |
| --- | :-: | :-: |
| Ability to handle payment means in Continia eDocuments | {{checkmark}} Mar 2025 | {{checkmark}} Apr 2025 |

## Business value

With this feature, you'll be able to easily set up and handle payment means in Continia eDocuments.

## Feature details

The feature will make it easier for you to set up different payments means without the need for extensions, adjustments or similar. This will be done using eDocument tables related to payment means.

For more information, see [Using payment means in eDocuments](@DC-450).
