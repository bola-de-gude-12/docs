# Inclusion of Purchase Reason Codes When Exporting the Document Capture Setup

| Feature | General availability on-premises | General availability online | Public preview |
| --- | :-: | :-: | :-: |
| Inclusion of purchase reason codes when exporting the Document Capture setup |{{checkmark}} Mar 2022 | {{checkmark}} Apr 2022 | - |

## Business value
This feature will enable easier and faster implementation when you set up multiple companies in Continia Document Capture or migrate from test to production environments.

## Feature details
With the introduction of this feature, Document Capture purchase reason codes will be included in the configuration file when you export a configuration using the **Document Capture Assisted Setup Guide**.
