# Teaching Tips

| Feature | General availability on-premises | General availability online | Public preview |
| --- | :-: | :-: | :-: |
| Teaching tips |{{checkmark}} Mar 2022 | {{checkmark}} Apr 2022 | - |

## Supported versions
The feature will be available for Microsoft Dynamics 365 Business Central 2021 release wave 1 (BC v18) and later.

## Business value
Teaching tips make it easier for new users to get started using Continia Document Capture.

## Feature details
In Microsoft Dynamics 365 Business Central 2021 release wave 1 (BC v18), Microsoft introduced the concept of [teaching tips](https://docs.microsoft.com/en-us/dynamics365/business-central/dev-itpro/administration/onboarding-teaching-tips-tours). Similarly, Continia added a number of useful teaching tips to the document journal in Document Capture 2021 R2 (8.00), and more will now be added to the upcoming version 9.00.

The new teaching tips will be added to the following pages:

* **Document Category**
* **Template Card**
* **Translations**
* **Standard Amount Distribution Code Card**
* **Document Card**
* **Invoice Matching**
* **Split and Merge Documents**
* **Purchase Allocation**
