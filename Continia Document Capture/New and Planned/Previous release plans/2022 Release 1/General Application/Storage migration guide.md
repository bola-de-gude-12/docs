# Storage Migration Guide

| Feature | General availability on-premises | General availability online | Public preview |
| --- | :-: | :-: | :-: |
| Storage migration guide |{{checkmark}} Mar 2022 | {{checkmark}} Apr 2022 | - |

## Business value
This newly developed assisted setup guide will make it considerably easier for you to change your storage method.

## Feature details
The guide will assist you in migrating your storage within Continia Document Capture. It comes with a number of features, including but not limited to:

* Support for migration between two of the same types of storage, such as **File** > **File** or **Azure Blob** > **Azure Blob**
* Additional validation of Azure Blob credentials as well as a connection test.
