# Adding Line Translations from the Matching Page

| Feature | General availability on-premises | General availability online | Public preview |
| --- | :-: | :-: | :-: |
| Adding line translations from the matching page | {{checkmark}} Sep 1, 2021 | {{checkmark}} Oct 1, 2021 | - |

## Feature details
On the matching page, recognized lines are displayed but not editable. We plan to enable users to set up line translations (**Translate to Type** and **Translate to No.**) directly from this page. Having both the vendor document and the purchase order/receipt line immediately available will ease the process of setting up correct translations.
