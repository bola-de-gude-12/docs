# Exporting and Downloading Multiple Attachments in Bulk

| Feature | General availability on-premises | General availability online | Public preview |
| --- | :-: | :-: | :-: |
| Exporting and downloading multiple attachments in bulk | {{checkmark}} Sep 1, 2021 | {{checkmark}} Oct 1, 2021 | - |

## Business value
This feature will allow you to export a large number of PDF documents in one go, thereby speeding up the process significantly.

Sometimes it's necessary to export the PDF versions of multiple imported documents, for example if auditors are asking for all invoices received within a certain period from a certain vendor. To do this, you must currently navigate to every single document and, from there, export each document manually. With this new feature, you'll be able to export all documents in bulk simply by running a report.

## Feature details
Using the newly developed report, you can export and download multiple documents in one go. It will be possible to limit the output of the report by applying filters such as vendor number or period. The report will add all included invoices to a ZIP file and then download this ZIP file to the client.
