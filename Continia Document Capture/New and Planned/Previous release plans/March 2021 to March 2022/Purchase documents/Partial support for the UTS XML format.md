# Partial Support for the UTS XML Format

| Feature | General availability on-premises | General availability online | Public preview |
| --- | :-: | :-: | :-: |
| Partial support for the UTS XML format | {{checkmark}} Mar 1, 2021 | {{checkmark}} Apr 1, 2021 | - |

## Business value
With more XML document formats supported, you can receive invoices and credit memos from more countries without having to perform any manual template modifications or similar.

## Feature details
Partial support will be added for the UTS XML document format. It will be possible to import UTS XML documents into the document journal, but you'll not be able to process these documents. The documents will be visible in the document journal as open documents, but they can't be processed and will therefore not be attached to their related XML invoices/credit memos. Support will be added for this in a later release. See [Full Support for the UTS XML Format](Full support for the UTS XML format.md).
