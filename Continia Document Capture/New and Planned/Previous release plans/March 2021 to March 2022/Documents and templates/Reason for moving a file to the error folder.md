# Reason for Moving a File to the Error Folder

| Feature | General availability on-premises | General availability online | Public preview |
| --- | :-: | :-: | :-: |
| Reason for moving a file to the error folder | {{checkmark}} Sep 1, 2021 | {{checkmark}} Oct 1, 2021 | - |

## Business value
This feature will provide you with insights into what exactly is wrong with erroneous files, which makes it easier for you to correct the problem.

## Feature details
Any document file that's moved to the error folder will be accompanied by an additional related file containing the error description along with an explanation of why the file couldn't be processed. This may be helpful when, for example, there are no more pages left in your ABBYY license.

Using this additional information, business users will be able to take appropriate action.
