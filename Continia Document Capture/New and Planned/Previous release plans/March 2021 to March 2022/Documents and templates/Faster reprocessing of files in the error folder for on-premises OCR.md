# Faster Reprocessing of Files in the Error Folder for On-Premises OCR

| Feature | General availability on-premises | General availability online | Public preview |
| --- | :-: | :-: | :-: |
| Faster reprocessing of files in the error folder for on-premises OCR | {{checkmark}} Sep 1, 2021 | {{checkmark}} Oct 1, 2021 | - |

## Business value
It will be easier and faster for you to reprocess files that are in the error folder. Today, you must move the files manually to the processing folder in order to reprocess them, but this improvement enables you to do it automatically simply by selecting the relevant file(s) from inside Microsoft Dynamics 365 Business Central.

## Feature details
In the document list, the new feature **Reprocess file** will be added to the action bar. When you run this feature, the selected document will be moved to the processing folder and then automatically reprocessed by the Continia Document Capture service.
