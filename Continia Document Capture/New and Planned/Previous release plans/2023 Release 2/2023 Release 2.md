# Features Planned and Released in 2023 Release 2

This article lists all Continia Document Capture features that were released from October 2023 through March 2024.

## Application enhancements

| Feature | Public preview | General availability |
| --- | :-: | :-: |
| [Preventing approvers from changing the on-hold status of documents for approval](@DC-360) | - | {{checkmark}} Oct 2023 |
| [Adding scanned images to general journals](@DC-300) | - | {{checkmark}} Oct 2023 |
| [Notification of remaining pages in OCR licenses](@DC-253) | - | {{checkmark}} Oct 2023 |
| [Adding the **Change Imported Amounts** action to general and purchase journals](@DC-254) | - | {{checkmark}} Oct 2023 |
| [Option to allow both types of approval-sharing users to receive notifications](@DC-361) | - | {{checkmark}} Oct 2023 |
| [Automatic vendor and field recognition when moving documents to other companies](@DC-301) | - | {{checkmark}} Oct 2023 |
| [Option to display original sender in **Ready to Import** when using Cloud OCR](@DC-302) | - | {{checkmark}} Oct 2023 |
| [Notifications for admins about open documents](@DC-303) | - | {{checkmark}} Oct 2023 |
| [New action – **Put on Hold and Approve**](@DC-386) | - | {{checkmark}} Oct 2023 |
| [Ability to sort by due date on the **Purchase Approval Entries** page](@DC-362) | - | {{checkmark}} Oct 2023 |
| [Dynamic columns in the document journal](@DC-304) | - | {{checkmark}} Oct 2023 |
| [Support for **WORKDATE** and **TODAY** in template field formulas](@DC-306) | - | {{checkmark}} Oct 2023 |
| [Selecting or deselecting all records when exporting or importing configurations](@DC-256) | - | {{checkmark}} Oct 2023 |

| [New Danish bookkeeping – demands for electronic invoicing](@DC-311) | - | {{checkmark}} Mar 2024 |
