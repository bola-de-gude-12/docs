# Document Search Enhancement

| Feature | Public preview | General availability online |
| --- | :-: | :-: |
| Document Search enhancement | - | {{checkmark}} Oct 2022 |

## Business value
With the introduction of this feature, the **Document Search** functionality has been improved in several ways. Featuring an additional data source and added control in terms of search methods, **Document Search** now delivers more precise results than ever.

## Feature details
The search functionality has been extended to also include values from the **Document Value** table. This table includes values that are recognized, values that are auto-translated, and values that have been manually entered by the user (such as G/L accounts, approval flow codes, and fields calculated using formulas).

Also, you can now choose between the two search methods **AND** and **OR** when searching through your documents. This allows you to better control and narrow down your search results, resulting in greater accuracy.
