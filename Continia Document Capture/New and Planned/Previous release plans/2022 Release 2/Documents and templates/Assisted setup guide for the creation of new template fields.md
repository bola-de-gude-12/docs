# Assisted Setup Guide for the Creation of New Template Fields

| Feature | Public preview | General availability online |
| --- | :-: | :-: |
| Assisted setup guide for the creation of new template fields | - | {{checkmark}} Oct 2022 |

## Business value
The introduction of an assisted setup guide will make it easier and faster for you to create new template fields. The guide should also significantly reduce the number of common mistakes made by less experienced users.

## Feature details
Using the new guide, you'll be able to create new template fields without the risk of deleting mandatory existing fields or otherwise breaking the template. Only field settings that are configured frequently will be availabe in the guide.
