# Set up If a Document Should Be Auto-Registered on the Template

| Feature | Public preview | General availability online |
| --- | :-: | :-: |
| Set up if a document should be auto-registered on the template | - | {{checkmark}} Oct 2022 |

## Business value
Today you can enable auto-registration of documents on a category level, meaning that either all documents or none are attempted to be registered automatically. With this feature, you can enable auto-registration per template to specify which templates will activate the auto-registration process, thus significantly speeding up the document handling.

## Feature details
The option "Register Documents Automatically" from the Document Category card will be added to the General section on page 6085584 - Template Card.
