# Caption Suggestions for Master Template Fields

| Feature | Public preview | General availability online |
| --- | :-: | :-: |
| Caption suggestions for master template fields | - | {{checkmark}} Oct 2022 |

## Business value
This feature will improve the recognition accuracy of future templates significantly, as captions for specific fields will now be based not only on the standard setup but also on customer-specific captions collected over time.

## Feature details
The feature basically suggests captions for master template fields based on captions already used for vendor template fields.
