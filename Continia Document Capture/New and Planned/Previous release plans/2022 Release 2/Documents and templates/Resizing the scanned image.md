# Resizing the Scanned Image (Document Viewer)

| Feature | Public preview | General availability online |
| --- | :-: | :-: |
| Resizing the scanned image (document viewer) | - | {{checkmark}} Oct 2022 |

## Business value
Using Continia Document Capture on relatively large monitors may result in unused space between the header field section and the document image. This feature allows you to easily adjust the size of the document viewer (the FactBox that displays the document image), which makes it easier for you to view the details of the scanned image and enables you to utilize the screen space more efficiently.

## Feature details
With the feature, you can resize the document viewer simply by dragging the vertical handle on the left side of the document viewer FactBox. This applies to all pages that include the document viewer.

The feature is available for Microsoft Dynamics 365 Business Central as of version 15.1 (Business Central 2019 release wave 2, Cumulative Update 01). For older versions, you can adjust the size by entering a size number instead – more details on this can be found [here](/continia-document-capture/new-and-planned/detailed-changelogs/previous-changelogs/document-capture-2022-r1#document-capture-2022-r1-service-pack-2) (see ID 30245).
