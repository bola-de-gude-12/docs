# Managing Electronic Documents in  Austria
As a customer in Austria, you can import electronic documents into Continia Document Capture either by email (as attached files) or via the Peppol eDelivery Network.

Document Capture connects you directly to the Peppol eDelivery Network, thereby eliminating the need to sign up with any third parties for integration. Once connected, you can start receiving electronic documents from vendors in Austria and other countries. To learn more about how to connect to the Peppol eDelivery Network, see [Setting up the Continia Delivery Network](/Continia Document Capture/Setting up Document Capture/Setting up general business functionality/Setting up the Continia Delivery Network.md).

All Austrian suppliers to the Austrian federal government are obliged to use e-invoices according to the Austrian ICT Consolidation Act. One way of complying with this is by using the Peppol eDelivery Network and the Peppol format. Other public sector entities (such as states, municipalities, and government-owned companies) are being integrated into the invoicing solution on a voluntary basis.

The Peppol BIS 3 format is being adopted by both businesses and public authorities in Austria, and Document Capture supports this format.

Another format that has been used in Austria is ebInterface, but this is expected to eventually be replaced by Peppol. Document Capture supports ebInterface but not via the Continia Delivery Network – only via email/file transfer.

You can also receive and import electronic documents from vendors in other countries. Most widespread formats are supported, but not all. For a list of supported formats, see [Supported Formats](Supported Formats.md).
